为 std::priority_queue 提供了下列推导指引：

1-6) 允许从底层容器类型以及从迭代器范围推导。
7-9) 允许从 std::from_range_t 标签和一个 input_range 推导。

这些重载只有在

  • InputIt 满足老式输入迭代器 (LegacyInputIterator) 、
  • Comp 不满足分配器 (Allocator) 、
  • Container 不满足分配器 (Allocator) ，
  • 对于重载 (4,5)，(C++23 起)Alloc 满足分配器 (Allocator) ，并且
  • 对于重载 (3,6)，std::uses_allocator_v<Container, Alloc> 为 true 时才会参与重载决议。

注意：库确定类型是否满足老式输入迭代器 (LegacyInputIterator) 的程度是未指定的，但最低要求是整数类型不具备输入迭代器的条件。类似地，确定类型是否满足分配器 (Allocator) 是未指定的，但最低要求是成员类型 Alloc::value_type 必须存在，且表达式 std::declval<Alloc&>().allocate(std::size_t{}) 在作为不求值操作数时必须为良构。

示例

#include <functional>
#include <iostream>
#include <queue>
#include <vector>
 
int main()
{
    const std::vector<int> v = {1, 2, 3, 4};
    std::priority_queue pq1{std::greater<int>{}, v}; // 推导 std::priority_queue<
                                                     //     int, std::vector<int>,
                                                     //     std::greater<int>>
    for (; !pq1.empty(); pq1.pop())
        std::cout << pq1.top() << ' ';
    std::cout << '\n';
 
    std::priority_queue pq2{v.begin(), v.end()}; // 推导 std::priority_queue<int>
 
    for (; !pq2.empty(); pq2.pop())
        std::cout << pq2.top() << ' ';
    std::cout << '\n';
}

输出：

1 2 3 4
4 3 2 1
