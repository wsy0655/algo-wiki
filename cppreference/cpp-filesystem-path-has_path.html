检查路径是否含有相应路径元素。

1) 检查 root_path() 是否为空。
2) 检查 root_name() 是否为空。
3) 检查 root_directory() 是否为空。
4) 检查 relative_path() 是否为空。
5) 检查 parent_path() 是否为空。
6) 检查 filename() 是否为空。
7) 检查 stem() 是否为空。
8) 检查 extension() 是否为空。

参数

（无）

返回值

若对应路径组分为非空则返回 true，否则返回 false。
