C17

< c


ISO/IEC 9899:2018 ，又称 C17/C18 （分别代表完成和出版的年份），是 C 标准的上一个版本。

C17 与 C11 相同，除了它将 __STDC_VERSION__ 预定义宏更新到 201710L ，含有数个缺陷报告，以及弃用了一些特性。
