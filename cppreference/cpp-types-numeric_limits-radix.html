std::numeric_limits<T>::radix

< cpp‎ | types‎ | numeric limits
static const int radix;
 (C++11 前)
static constexpr int radix;
 (C++11 起)

std::numeric_limits<T>::radix 的值是用于表示该类型的数值系统的底。对于所有二进制数值类型是 2，但它可以，例如对 IEEE 754 [[[enwiki:Decimal64 floating-point format|十进制浮点数类型]]或第三方binary-coded decimal二进制编码十进制整数是 10。此常量对所有特化都有意义。
