operator==(std::move_iterator<Iter>, std::move_sentinel)

< cpp‎ | iterator‎ | move iterator


template< std::sentinel_for<Iter> S >

friend constexpr bool

    operator==( const move_iterator& i, const std::move_sentinel<S>& s );
 (C++20 起)

比较 move_iterator 与 move_sentinel。

此重载只有在 i.base() == s.base() 良构并且可转换到 bool 时才会参与重载决议。

此函数模板对常规的无限定或有限定查找不可见，而只能在 std::move_iterator<Iter> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

i - 要比较的移动迭代器
s - 要比较的移动哨位

返回值

i.base() == s.base()
