命名空间别名

< cpp‎ | language


命名空间别名允许程序员定义命名空间的另一个名字。

它们常用作长的或嵌套过深的命名空间的简便使用方式。

语法

namespace 别名 = 命名空间名; (1)
namespace 别名 = ::命名空间名; (2)
namespace 别名 = 嵌套名::命名空间名; (3)

解释

新别名 别名 提供另一种访问 命名空间名 的方法。

别名 必须是先前未使用过的名称。别名 在引入它的作用域的期间内有效。

关键词

namespace

示例

#include <iostream>
 
namespace foo
{
    namespace bar
    {
         namespace baz
         {
             int qux = 42;
         }
    }
}
 
namespace fbz = foo::bar::baz;
 
int main()
{
    std::cout << fbz::qux << '\n';
}

输出：

42
