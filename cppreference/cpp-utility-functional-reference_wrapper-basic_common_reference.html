一个 reference_wrapper（以 R 代表）和一个类型 T 的公共引用类型，相应地应用 cv 和引用限定符（分别以 RQ 和 TQ 代表），等价于应用左值引用的 R 的底层类型和 TQ 的公共引用类型。

仅当 R 是仅有的 reference_wrapper，且引用左值引用的 R 的底层类型和 TQ 具有公共引用类型且 RQ 可以转换为该类型时，公共引用类型才有定义。

当且仅当 R 是 std::reference_wrapper 的特化时，仅用于阐释的常量 /*is-ref-wrapper*/<R> 为 true。

示例

#include <concepts>
#include <functional>
 
static_assert(std::same_as<std::common_reference_t<int&,
                                                   std::reference_wrapper<int>>,
                                                   int&>);
static_assert(std::same_as<std::common_reference_t<std::reference_wrapper<int>&,
                                                   int&>,
                                                   int&>);
static_assert(std::same_as<std::common_reference_t<int&,
                                                   const std::reference_wrapper<int>&>,
                                                   int&>);
int main() {}
