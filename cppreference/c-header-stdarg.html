标准库标头 <stdarg.h>

< c‎ | header


此标头提供对变参数函数的支持。

类型
va_list
 保有 va_start、va_arg、va_end 及 va_copy 所需的信息
(typedef)
宏
va_start
 令函数得以访问可变实参
(宏函数)
va_arg
 访问下一个可变函数实参
(宏函数)
va_copy
(C99)
 创造函数可变实参的副本
(宏函数)
va_end
 结束对函数可变实参的遍历
(宏函数)

概要

#define __STDC_VERSION_STDARG_H__ 202311L
 
typedef /* 未指明 */ va_list;
 
/*type*/ va_arg(va_list ap, /*type*/);
void va_copy(va_list dest, va_list src);
void va_end(va_list ap);
void va_start(va_list ap, ...);
