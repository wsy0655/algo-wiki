std::isprint(std::locale)

< cpp‎ | locale
在标头 <locale> 定义
template< class CharT >
bool isprint( CharT ch, const locale& loc );

检查给定字符按给定本地环境的 std::ctype 刻面是否分类为可打印字符（包含空格）。

参数

ch - 字符
loc - 本地环境

返回值

若字符被分类为可打印则返回 true，否则返回 false。

可能的实现

template<class CharT>
bool isprint(CharT ch, const std::locale& loc)
{
    return std::use_facet<std::ctype<CharT>>(loc).is(std::ctype_base::print, ch);
}

示例

演示以不同本地环境使用 isprint()（OS 限定）。

#include <iostream>
#include <locale>
 
int main()
{
    const wchar_t c = L'\u2122'; // 商标标记
 
    std::locale loc1("C");
    std::cout << "isprint('™', C 本地环境) 返回 "
               << std::boolalpha << std::isprint(c, loc1) << '\n';
 
    std::locale loc2("en_US.UTF-8");
    std::cout << "isprint('™', Unicode 本地环境) 返回 "
              << std::boolalpha << std::isprint(c, loc2) << '\n';
}

输出：

isprint('™', C 本地环境) 返回 false
isprint('™', Unicode 本地环境) 返回 true
