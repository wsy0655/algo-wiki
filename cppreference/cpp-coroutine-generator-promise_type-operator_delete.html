std::generator<Ref,V,Allocator>::promise_type::operator delete

< cpp‎ | coroutine‎ | generator‎ | promise type
void operator delete( void* ptr, std::size_t n ) noexcept;
 (C++23 起)

解分配 ptr 所指向的存储，使用等价于用于分配此段内存的分配器进行。

传递给此函数的 ptr 必须为对 operator new 的重载之一以等于 n 的大小实参调用所返回的值。否则其行为未定义。

参数

ptr - 从之前对 operator new 的调用获得的指针
n - 要解分配的存储的大小
