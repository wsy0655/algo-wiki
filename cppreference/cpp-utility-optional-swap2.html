std::swap(std::optional)

< cpp‎ | utility‎ | optional
在标头 <optional> 定义
template< class T >

void swap( std::optional<T>& lhs,

           std::optional<T>& rhs ) noexcept(/* 见下文 */);
 (C++17 起)
(C++20 起为 constexpr)

对 std::optional 重载 std::swap 算法。交换 lhs 与 rhs 的状态。相当于调用 lhs.swap(rhs)。

此重载只有在 std::is_move_constructible_v<T> 与 std::is_swappable_v<T> 皆为 true 时才会参与重载决议。

参数

lhs, rhs - 要交换状态的 optional 对象

返回值

（无）

异常

noexcept 说明：  
noexcept(noexcept(lhs.swap(rhs)))

注解

功能特性测试宏 值 标准 功能特性
__cpp_lib_optional 202106L (C++20)
(DR20)		 完全 constexpr

示例

#include <iostream>
#include <optional>
#include <string>
 
int main()
{
    std::optional<std::string> a{"██████"}, b{"▒▒▒▒▒▒"};
 
    auto print = [&](auto const& s)
    {
        std::cout << s << "\t"
                     "a = " << a.value_or("(null)") << "  "
                     "b = " << b.value_or("(null)") << '\n';
    };
 
    print("起初:\t");
    std::swap(a, b);
    print("swap(a, b):");
    a.reset();
    print("\n""a.reset():");
    std::swap(a, b);
    print("swap(a, b):");
}

输出：

起初:            a = ██████  b = ▒▒▒▒▒▒
swap(a, b):     a = ▒▒▒▒▒▒  b = ██████
 
a.reset():      a = (null)  b = ██████
swap(a, b):     a = ██████  b = (null)
