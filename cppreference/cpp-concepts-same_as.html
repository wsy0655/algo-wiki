std::same_as

< cpp‎ | concepts
在标头 <concepts> 定义
template < class T, class U >
concept same_as = /* 见下文 */;
 (C++20 起)

概念 same_as<T, U> 当且仅当 T 与 U 代表同一类型才得到满足。

std::same_as<T, U> 蕴含 std::same_as<U, T>，反之亦然。

可能的实现

namespace detail
{
    template< class T, class U >
    concept SameHelper = std::is_same_v<T, U>;
}
 
template< class T, class U >
concept same_as = detail::SameHelper<T, U> && detail::SameHelper<U, T>;
 
#include <concepts>
#include <iostream>
 
template<typename T, typename ... U>
concept either = (std::same_as<T, U> || ...);
 
template<typename T>
concept is_printable = std::integral<T> || std::floating_point<T> ||
    either<std::remove_cvref_t<std::remove_pointer_t<std::decay_t<T>>>, char, wchar_t>;
 
void println(is_printable auto const ... arguments)
{
    (std::wcout << ... << arguments) << '\n';
}
 
int main()
{
    println("例: ", 3.14, " : ", 42, " : [", 'a', L'-', L"Z]");
}

输出：

例: 3.14 : 42 : [a-Z]
