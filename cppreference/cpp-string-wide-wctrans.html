std::wctrans

< cpp‎ | string‎ | wide
在标头 <cwctype> 定义
std::wctrans_t wctrans( const char* str );

构造 std::wctrans_t 类型的值，描述宽字符映射的 LC_CTYPE 类别。它可以是标准映射，或如 "tojhira" 或 "tojkata" 的本地环境限定的映射之一。

参数

str - 保有所欲映射的名称的 C 字符串。

下列 str 的值在所有 C 本地环境中受到支持：

str 的值 效果
"toupper" 标识 towupper 所用的映射
"tolower" 标识 towlower 所用的映射


返回值

std::wctrans_t 对象，适合由 std::towctrans 用来根据当前 C 本地环境的具名映射映射宽字符，或若 str 不指名当前 C 本地环境所支持的映射则为零。

参阅

towctrans
