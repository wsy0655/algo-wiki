C++ 标准库扩展，版本 2

< cpp‎ | experimental


C++ 库基础扩展的版本 2 (Version 2 of the C++ Extensions for Library Fundamentals)，ISO/IEC TS 19568:2017 在定义于版本 1 中的组件外，还为 C++ 标准库定义了下列组件：

C++ 检测手法

在标头 <experimental/type_traits> 定义
is_detected detected_t detected_or
 检测 模板标识 是否良构的别名模板
(别名模板)
nonesuch
 detected_t 在失败时返回的类类型
(类)

传播 const 的指针式对象包装器

在标头 <experimental/propagate_const> 定义
propagate_const
 传播 const 的指针式对象包装器
(类模板)

无所有权指针

在标头 <experimental/memory> 定义
observer_ptr
 表示无所有权指针 的类
(类模板)

std::array 创建

在标头 <experimental/array> 定义
make_array
(库基础 TS v2)
 创建 std::array 对象，从实参推导出其大小和可选的元素类型
(函数模板)

ostream_joiner

在标头 <experimental/iterator> 定义
ostream_joiner
 写入相继元素到输出流，并以分隔符分隔相邻元素的输出迭代器
(类模板)

简单的随机数生成设施

为每个线程维护一个 std::default_random_engine 类型的逐线程引擎，初始化到不可预测状态，而它为下列函数所用。

在标头 <experimental/random> 定义
randint
 生成指定范围中的随机整数
(函数模板)
reseed
 重播种单个线程的随机引擎
(函数)
在标头 <experimental/algorithm> 定义
sample
 从序列随机选择 n 个元素
(函数模板)
shuffle
 随机地重排范围中的元素
(函数模板)

已并入 C++17

库基础 v2 的下列组件已被采纳到 C++17 标准。

在标头 <experimental/type_traits> 定义

 void 的变长别名模板
(别名模板)
在标头 <experimental/type_traits> 定义
conjunction
 变长逻辑与元函数
(类模板)
disjunction
 变长逻辑或元函数
(类模板)
negation
 逻辑非元函数
(类模板)
在标头 <experimental/functional> 定义
not_fn
 创建函数对象，它返回其所保有函数对象的结果的逻辑补
(函数模板)
在标头 <experimental/numeric> 定义
gcd
 返回两个整数最大公约数的 constexpr 函数模板
(函数模板)
lcm
 返回两个整数最小公倍数的 constexpr 函数模板
(函数模板)
