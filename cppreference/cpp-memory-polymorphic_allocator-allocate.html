std::pmr::polymorphic_allocator<T>::allocate

< cpp‎ | memory‎ | polymorphic allocator
T* allocate( std::size_t n );
 (C++17 起)

用底层的内存资源分配 n 个 T 类型对象的存储。等价于 return static_cast<T*>(resource()->allocate(n * sizeof(T), alignof(T)));。

参数

n - 要分配存储的对象数

返回值

指向被分配存储的指针。

异常

若 n > std::numeric_limits<std::size_t>::max() / sizeof(T) 则抛出 std::bad_array_new_length；亦可抛出调用 resource()->allocate 所抛的任何异常。
