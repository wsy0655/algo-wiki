std::forward_list<T,Allocator>::front

< cpp‎ | container‎ | forward list

reference front();
 (1) (C++11 起)
const_reference front() const;
 (2) (C++11 起)

返回到容器首元素的引用。

如果 empty() 是 true，那么行为未定义。

 (C++26 前)

如果 empty() 是 true，那么：

  • 如果实现是硬化实现，那么就会发生契约违背。并且契约违背处理函数在“观察”求值语义下返回时行为未定义。
  • 如果实现不是硬化实现，那么行为未定义。
 (C++26 起)

返回值

到首元素的引用。

复杂度

常数。

注解

对于容器 c，表达式 c.front() 等价于 *c.begin()。

示例

下列代码用 front 显示 std::forward_list<char> 的首元素：

#include <cassert>
#include <forward_list>
 
int main()
{
    std::forward_list<char> letters{'a', 'b', 'c', 'd'};
    assert(letters.front() == 'a');
}
