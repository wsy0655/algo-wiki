1) 代表有逆序的底层 view 的视图的范围适配器。
2) 范围适配器对象 (RangeAdaptorObject) 。表达式 views::reverse(e) 表达式等价于下列表达式之一，但 e 仅求值一次：
  • e.base()，若 e 的类型是（可有 cv 限定的）reverse_view 的特化；
  • 否则，若 e 的类型是（可有 cv 限定的）某迭代器类型 I 和 ranges::subrange_kind 类型的值 K 的 ranges::subrange<std::reverse_iterator<I>, std::reverse_iterator<I>, K>：
  • ranges::subrange<I, I, K>(e.end().base(), e.begin().base(), e.size())，若 K 为 ranges::subrange_kind::sized；
  • 否则为 ranges::subrange<I, I, K>(e.end().base(), e.begin().base())；
  • 否则为 ranges::reverse_view{e}。
换言之，views::reverse 尽可能解包逆转的视图。

reverse_view 始终实现 bidirectional_range 与 common_range，而若底层视图类型 V 实现 borrowed_range、sized_range 或 random_access_range 则它也实现对应的概念。

示例

#include <iostream>
#include <ranges>
 
int main()
{
    static constexpr auto il = {3, 1, 4, 1, 5, 9};
 
    std::ranges::reverse_view rv{il};
    for (int i : rv)
        std::cout << i << ' ';
    std::cout << '\n';
 
    for (int i : il | std::views::reverse)
        std::cout << i << ' ';
    std::cout << '\n';
 
    // operator[] 继承自 std::view_interface
    for (auto i{0U}; i != rv.size(); ++i)
        std::cout << rv[i] << ' ';
    std::cout << '\n';
}

输出：

9 5 1 4 1 3
9 5 1 4 1 3
9 5 1 4 1 3
