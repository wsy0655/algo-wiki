std::experimental::filesystem::path::is_absolute,is_relative

< cpp‎ | experimental‎ | fs‎ | path
bool is_absolute() const;
 (1) (文件系统 TS)
bool is_relative() const;
 (2) (文件系统 TS)

检测路径是绝对路径还是相对路径。绝对路径是其 root_path() 元素唯一标定一个文件系统位置的路径。如果是绝对路径，第一个版本返回 true 否则返回 false；第二个版本与之相反。

参数

（无）

返回值

1) 如果为绝对路径则为 true，否则为 false。
2) 如果为绝对路径则为 false，否则为 true。

异常

可能会抛出由实现定义的异常。
