计算类型 I 关联的值类型，若它存在。用户可对程序定义类型特化 value_type。

1) 主模板是空结构体。
2) 对指针的特化。若 T 是对象类型，则提供等于 std::remove_cv_t<T> 的成员类型 type。否则，无成员 type。
3) 对数组类型的特化。
4) 对 const 限定类型的特化。
5) 对定义公开且可访问成员类型 value_type 的类型的特化。若 T::value_type 是对象类型，则提供等于 T::value_type 的成员类型 type。否则无成员 type。
6) 对定义公开且可访问成员类型 element_type 的类型（例如 std::shared_ptr）的特化。若 T::element_type 是对象类型，则提供等于 std::remove_cv_t<typename T::element_type> 的成员类型 type。否则无成员类型 type。

辅助别名模版

template< class T >
using value_type_t = typename ranges::value_type<T>::type;
 (范围 TS)

注解

若类型一同含有 value_type 成员和 element_type 成员，则特化 (5) 和 (6) 有歧义。

value_type 被有意用于如迭代器的 Readable。它并非被有意用于范围。
