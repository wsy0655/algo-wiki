std::experimental::pmr::set_default_resource

< cpp‎ | experimental
在标头 <experimental/memory_resource> 定义
memory_resource* set_default_resource( memory_resource* r ) noexcept;
 (库基础 TS)

若 r 非空，则设置默认内存资源指针为 r；否则，设置默认内存资源指针为 new_delete_resource()。

默认内存资源指针由某些设施在未提供明确的内存资源时使用。初始默认内存资源指针为 new_delete_resource() 的返回值。

此函数线程安全。每次对 set_default_resource 的调用都同步于（见 std::memory_order）后继的 set_default_resource 和 get_default_resource 调用。

返回值

返回默认内存资源指针之前的值。

参阅

get_default_resource
 获取缺省 memory_resource
(函数)
new_delete_resource
 返回一个静态的程序范围 memory_resource，它使用全局 operator new 与 operator delete 分配和解分配内存
(函数)
