putwchar

< c‎ | io
在标头 <wchar.h> 定义
wint_t putwchar( wchar_t ch );
 (C95 起)

将宽字符 ch 写入 stdout。

参数

ch - 要被写入的宽字符

返回值

成功时为 ch，失败时为 WEOF。

示例

#include <locale.h>
#include <stdio.h>
#include <stdlib.h>
#include <wchar.h>
 
int main()
{
    setlocale(LC_ALL, "en_US.utf8");
 
    const wchar_t data[] =
    {
        L'\u2200', // Unicode 名: "FOR ALL"
        L'∀',
        L'\n',
    };
 
    for (size_t t = 0; t != (sizeof data / sizeof(wchar_t)); ++t)
    {
        if (putwchar(data[t]) == WEOF)
        {
            puts("I/O error in putwchar");
            return EXIT_FAILURE;
        }
    }
 
    return EXIT_SUCCESS;
}

可能的输出：

∀∀
