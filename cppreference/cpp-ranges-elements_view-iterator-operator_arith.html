增或减迭代器。

令 current_ 为底层迭代器。

1) 等价于 ++current_; return *this;
2) 等价于 ++current_;
3) 等价于 auto tmp = *this; ++*this; return tmp;
4) 等价于 --current_; return *this;
5) 等价于 auto tmp = *this; --*this; return tmp;
6) 等价于 current_ += n; return *this;
7) 等价于 current_ -= n; return *this;

参数

n - 相对于当前位置的位置

返回值

1,4,6,7) *this
2) （无）
3,5) 更改前的 *this 的副本
