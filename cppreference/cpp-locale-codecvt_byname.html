std::codecvt_byname

< cpp‎ | locale
在标头 <locale> 定义
template< class InternT, class ExternT, class State >
class codecvt_byname : public std::codecvt<InternT, ExternT, State>;

std::codecvt_byname 是一个封装了在它构造时指定的本地环境的多字节/宽字符转换规则的 std::codecvt 刻面。

特化

标准库保证提供以下特化：

在标头 <locale> 定义
std::codecvt_byname<char, char, std::mbstate_t> 恒等转换
std::codecvt_byname<char16_t, char, std::mbstate_t>
(C++11 起)(C++20 弃用)		 在 UTF-16 和 UTF-8 间转换
std::codecvt_byname<char16_t, char8_t, std::mbstate_t>
(C++20 起)		 在 UTF-16 和 UTF-8 间转换
std::codecvt_byname<char32_t, char, std::mbstate_t>
(C++11 起)(C++20 弃用)		 在 UTF-32 和 UTF-8 间转换
std::codecvt_byname<char32_t, char8_t, std::mbstate_t>
(C++20 起)		 在 UTF-32 和 UTF-8 间转换
std::codecvt_byname<wchar_t, char, std::mbstate_t> 在系统原生宽和单字节窄字符集间转换

枚举常量 定义

partial 未转换所有源字符
error 遇到非法字符
noconv 无需转换，输入与输出类型相同

示例

此示例演示用来自具备 GB18030 的本地环境的 codecvt 刻面读取 GB18030 编码的文件：

#include <fstream>
#include <iostream>
#include <locale>
#include <string>
 
int main()
{
    // GB18030 窄多字节编码
    std::ofstream("text.txt") << "\x7a"              // 字母 'z'，U+007a
                                 "\x81\x30\x89\x38"  // 字母 'ß'，U+00df
                                 "\xcb\xae"          // CJK 表意字符 '水'，U+6c34
                                 "\x94\x32\xbc\x35"; // 音符 '𝄋' (segno)，U+1d10b
 
    std::wifstream fin("text.txt");
    fin.imbue(std::locale(fin.getloc(),
              new std::codecvt_byname<wchar_t, char, std::mbstate_t>("zh_CN.gb18030")));
 
    for (wchar_t c; fin.get(c);)
        std::cout << std::hex << std::showbase << static_cast<unsigned>(c) << '\n';
}

输出：

0x7a
0xdf
0x6c34
0x1d10b
