std::begin(std::initializer_list)

< cpp‎ | utility‎ | initializer list
在标头 <initializer_list> 定义
template< class E >
const E* begin( std::initializer_list<E> il ) noexcept;
 (C++11 起)
(C++14 起为 constexpr)

std::begin 对 initializer_list 的重载返回指向 il 首元素的指针。

参数

il - initializer_list

返回值

il.begin()

示例

#include <algorithm>
#include <initializer_list>
#include <iostream>
#include <iterator>
 
int main() 
{
    std::initializer_list ϕ = {'1', '.', '6', '1', '8', '0'};
 
    std::copy(std::begin(ϕ),
              std::end(ϕ),
              std::ostream_iterator<char>(std::cout, ""));
 
    std::cout << '\n';
}

输出：

1.6180
