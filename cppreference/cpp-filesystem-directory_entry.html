std::filesystem::directory_entry

< cpp‎ | filesystem
在标头 <filesystem> 定义
class directory_entry;
 (C++17 起)

表示目录条目。该对象存储一个 path 作为成员，并可能也在目录迭代过程中存储附带的文件属性（硬链接数、状态、符号链接状态、文件大小、及最后写入时间）。
