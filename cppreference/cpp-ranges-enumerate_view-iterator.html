std::ranges::enumerate_view<V>::iterator

< cpp‎ | ranges‎ | enumerate view
template< bool Const >
class /*iterator*/
 (C++23 起)
(仅用于阐述*)

enumerate_view::begin 的返回类型，并且在底层视图 V 是 common_range 时也是 enumerate_view::end 的返回类型。

对于 const 限定的重载返回类型 /*iterator*/<true>。对非 const 限定的重载返回类型 /*iterator*/<false>。
