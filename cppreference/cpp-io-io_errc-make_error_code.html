std::make_error_code(std::io_errc)

< cpp‎ | io‎ | io errc
在标头 <ios> 定义
std::error_code make_error_code( std::io_errc e ) noexcept;
 (C++11 起)

从 std::io_errc 类型的值构造一个 std::error_code 对象，如同通过 return std::error_code(static_cast<int>(e), std::iostream_category())。

此函数会被 std::error_code 的带有 std::io_errc 实参的构造函数调用。

参数

e - 错误码编号

返回值

std::error_code 类型的值，保有来自 e 的错误码编号，关联到错误类别 "iostream"。

示例

#include <iostream>
#include <system_error>
 
int main()
{
    std::error_code ec = std::make_error_code(std::io_errc::stream);
 
    // 因为有重载方法和 is_error_code_enum 特化，所以可以这样操作
    ec = std::io_errc::stream;
 
    std::cout << "来自 io_errc::stream 的错误码具有类别 "
              << ec.category().name() << '\n';
}

输出：

来自 io_errc::stream 的错误码具有类别 iostream
