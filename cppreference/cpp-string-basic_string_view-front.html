std::basic_string_view<CharT,Traits>::front

< cpp‎ | string‎ | basic string view
constexpr const_reference front() const;
 (C++17 起)

返回到视图首字符的引用。

如果 empty() 是 true，那么行为未定义。

 (C++26 前)

如果 empty() 是 true，那么：

  • 如果实现是硬化实现，那么就会发生契约违背。并且契约违背处理函数在“观察”求值语义下返回时行为未定义。
  • 如果实现不是硬化实现，那么行为未定义。
 (C++26 起)

返回值

data_[0]

复杂度

常数。

示例

#include <iostream>
#include <string_view>
 
int main()
{
    for (std::string_view str{"ABCDEF"}; !str.empty(); str.remove_prefix(1))
        std::cout << str.front() << ' ' << str << '\n';
}

输出：

A ABCDEF
B BCDEF
C CDEF
D DEF
E EF
F F
