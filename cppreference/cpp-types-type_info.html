std::type_info

< cpp‎ | types
在标头 <typeinfo> 定义
class type_info;

类 type_info 保有有关一个类型的实现特定信息，包括该类型的名称和比较两个类型的相等性或校排顺序的方法。这是 typeid 运算符所返回的类。

type_info 类既不满足可复制构造 (CopyConstructible) 也不满足可复制赋值 (CopyAssignable) 。
