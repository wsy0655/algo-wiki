此页面尝试列出所有来自标准库，在命名空间 std::literals::string_literals 中的符号。符号以下列方式书写：

  • 函数名带 ()。
  • 模板带 <>。

注解

这些运算符在命名空间 std::literals::string_literals 声明，其中 literals 和 string_literals 都是内联命名空间。可以通过以下方式取得对这些运算符的访问：

  • using namespace std::literals
  • using namespace std::string_literals
  • using namespace std::literals::string_literals

std::chrono::duration 亦定义 operator""s 以表示字面量的秒，但它是算术字面量：10.0s 和 10s 是十秒，但 "10"s 是字符串。
