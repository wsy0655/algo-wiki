char32_t

< c‎ | string‎ | multibyte
在标头 <uchar.h> 定义
typedef uint_least32_t char32_t;
 (C11 起)

char32_t 是用于 32 位宽字符的无符号整数类型，与 uint_least32_t 为同一类型。

注意

任何给定平台上，经由 uint_least32_t 的定义，类型 char32_t 的宽度可能大于 32 位，但存储于 char32_t 类型对象中的实际值将始终拥有 32 位宽度。

示例

#include <stdio.h>
#include <uchar.h>
 
int main(void)
{
    const char32_t wc[] = U"zß水🍌"; // 即 "z\u00df\u6c34\U0001f34c"
    const size_t wc_sz = sizeof wc / sizeof *wc;
    printf("%zu 个 UTF-32 码元：[ ", wc_sz);
    for (size_t n = 0; n < wc_sz; ++n)
        printf("%#x ", wc[n]);
    printf("]\n");
}

可能的输出：

5 个 UTF-32 码元：[ 0x7a 0xdf 0x6c34 0x1f34c 0 ]
