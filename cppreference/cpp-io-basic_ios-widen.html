std::basic_ios<CharT,Traits>::widen

< cpp‎ | io‎ | basic ios
char_type widen( char c ) const;

转换字符 c 到其在当前本地环境的等价物。若需要，则将结果从 char 转换为用于流内的字符类型。

相当于调用 std::use_facet< std::ctype<char_type> >(getloc()).widen(c)。

参数

c - 要转换的字符

返回值

转换为 char_type 的字符

参阅

narrow
 窄化字符
(公开成员函数)
do_widen
[虚]
 将一个或多个字符从 char 转换到 CharT
(std::ctype<CharT> 的虚受保护成员函数)
btowc
 若可能，加宽单字节窄字符为宽字符
(函数)
