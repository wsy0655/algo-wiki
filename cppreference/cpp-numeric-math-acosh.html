参数

num - 浮点数或整数

返回值

如果没有发生错误，那么返回 num 在区间 [0, +∞] 上的反双曲余弦（cosh-1
(num) 或 arcosh(num)）。

如果发生定义域错误，那么返回实现定义值（在受支持平台上是 NaN）。

错误处理

报告 math_errhandling 中指定的错误。

如果实参小于 1，那么发生定义域错误。

如果实现支持 IEEE 浮点数算术（IEC 60559），那么

  • 如果实参小于 1，那么引发 FE_INVALID 并返回 NaN
  • 如果实参是 1，那么返回 +0
  • 如果实参是 +∞，那么返回 +∞
  • 如果实参是 NaN，那么返回 NaN

注解

虽然（C++ 对此函数引用的）C 标准命名此函数为“弧双曲余弦”，双曲函数的反函数却是面积函数。其实参是双曲扇形的面积，而非弧长。正确的名称是“反双曲余弦”（POSIX 所用）或“面积双曲余弦”。

额外重载不需要以 (A) 的形式提供。它们只需要能够对它们的整数类型实参 num 确保 std::acosh(num) 和 std::acosh(static_cast<double>(num)) 的效果相同。

示例

#include <cerrno>
#include <cfenv>
#include <cfloat>
#include <cmath>
#include <cstring>
#include <iostream>
 
// #pragma STDC FENV_ACCESS ON
 
int main()
{
    std::cout << "acosh(1) = " << std::acosh(1) << '\n'
              << "acosh(10) = " << std::acosh(10) << '\n'
              << "acosh(DBL_MAX) = " << std::acosh(DBL_MAX) << '\n'
              << "acosh(Inf) = " <<  std::acosh(INFINITY) << '\n';
 
    // 错误处理
    errno = 0;
    std::feclearexcept(FE_ALL_EXCEPT);
 
    std::cout << "acosh(0.5) = " << std::acosh(0.5) << '\n';
 
    if (errno == EDOM)
        std::cout << "    errno == EDOM: " << std::strerror(errno) << '\n';
    if (std::fetestexcept(FE_INVALID))
        std::cout << "    发生 FE_INVALID\n";
}
