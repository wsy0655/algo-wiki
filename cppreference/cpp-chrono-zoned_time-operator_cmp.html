std::chrono::operator==(std::chrono::zoned_time)

< cpp‎ | chrono‎ | zoned time
在标头 <chrono> 定义
template< class Duration1, class Duration2, class TimeZonePtr >

bool operator==( const std::chrono::zoned_time<Duration1, TimeZonePtr>& x,

                 const std::chrono::zoned_time<Duration2, TimeZonePtr>& y );
 (C++20 起)

比较两个 zoned_time 值 x 和 y。若两个 zoned_time 对象的时间点和时区指针按照 operator== 比较相等，则它们比较相等。

!= 运算符从 operator== 运算符合成。

返回值

x.get_time_zone() == y.get_time_zone() && x.get_sys_time() == y.get_sys_time()，但比较在 x 和 y 的非静态数据成员上直接进行，而不进行复制。
