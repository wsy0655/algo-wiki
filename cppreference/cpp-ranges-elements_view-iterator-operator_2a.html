std::ranges::elements_view<V,F>::iterator<Const>::operator*

< cpp‎ | ranges‎ | elements view‎ | iterator
constexpr decltype(auto) operator*() const;
 (C++20 起)

返回底层迭代器所指向的 V 中的元素。

相当于返回 /*get-element*/(this->base())，其中对于表达式 e ， /*get-element*/(e) 为

  • std::get<N>(*e) ，若 ranges::range_reference_t<Base> 为引用类型，
  • 否则为 static_cast<E>(std::get<N>(*e)) ，其中 E 为 std::remove_cv_t<std::tuple_element_t<N, ranges::range_reference_t<Base>>> 。

参数

（无）

返回值

当前元素。

注解

不提供 operator->。

示例

#include <iostream>
#include <ranges>
#include <string_view>
#include <tuple>
 
int main()
{
    using T = std::tuple<int, char, std::string_view>;
 
    const auto il = {
        T{1, 'A', "α"},
        T{2, 'B', "β"},
        T{3, 'C', "γ"},
    };
 
    const auto view {il | std::views::elements<2>};
 
    const auto iter {view.begin() + 1};
 
    std::cout << *iter << '\n';
}

输出：

β
