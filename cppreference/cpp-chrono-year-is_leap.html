std::chrono::year::is_leap

< cpp‎ | chrono‎ | year
constexpr bool is_leap() const noexcept;
 (C++20 起)

确定 *this 是否表示外推格里高利历中的闰年。

若存储的年份值满足下列条件则 *this 表示闰年

  • 被 4 整除且不被 100 整除；或
  • 被 400 整除。

返回值

若 *this 表示闰年则为 true，否则为 false。

示例

#include <chrono>
#include <iostream>
 
int main()
{
    using namespace std::chrono_literals;
    for (const std::chrono::year y : {2020y, 2021y, 2000y, 3000y})
    {
        if (const int iy{static_cast<int>(y)}; y.is_leap())
            std::cout << iy << " 是闰年，因为它可以被 "
                      << (iy % 400 == 0 ? "400 整除\n" : "4 但不能被 100 整除\n");
        else
            std::cout << iy << " 不是闰年\n";
    }
}

输出：

2020 是闰年，因为它可以被 4 但不能被 100 整除
2021 不是闰年
2000 是闰年，因为它可以被 400 整除
3000 不是闰年
