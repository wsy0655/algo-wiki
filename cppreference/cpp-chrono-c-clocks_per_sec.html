CLOCKS_PER_SEC

< cpp‎ | chrono‎ | c


在标头 <ctime> 定义
#define CLOCKS_PER_SEC /* 由实现定义 */

展开成 std::clock_t 类型的表达式，其值等于每秒 std::clock() 所返回的时钟计次数（不必是编译时常量）。

注解

POSIX 定义 CLOCKS_PER_SEC 为 1'000'000，无关乎 std::clock() 的实际精度。

示例

#include <ctime>
#include <iostream>
#include <locale>
 
int main()
{
    const std::clock_t cps{CLOCKS_PER_SEC};
    std::cout.imbue(std::locale("en_US.utf8"));
    std::cout << cps << '\n';
}

可能的输出：

1,000,000
