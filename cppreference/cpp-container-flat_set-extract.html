std::flat_set<Key,Compare,KeyContainer>::extract

< cpp‎ | container‎ | flat set

container_type extract() &&;
 (C++23 起)

提取所适配的容器 c。等价于 return std::move(c);。

此操作后 *this 为空，即使抛出了异常也是如此。

返回值

std::move(c)。

复杂度

常数。

示例

#include <cassert>
#include <flat_set>
#include <print>
#include <type_traits>
#include <utility>
#include <vector>
 
int main()
{
    std::flat_set<int> set{1, 2, 3};
    const auto size{set.size()};
 
    auto c{std::move(set).extract()};
    assert(c.size() == size);
    assert(set.empty());
    assert(set.keys().empty());
    assert(set.values().empty());
 
    // 默认键容器是 std::vector：
    static_assert(std::is_same_v<decltype(c), std::vector<int>>);
 
    std::println("{}", c);
}

输出：

[1, 2, 3]
