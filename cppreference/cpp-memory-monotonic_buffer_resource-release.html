std::pmr::monotonic_buffer_resource::release

< cpp‎ | memory‎ | monotonic buffer resource
void release();
 (C++17 起)

通过上游内存资源的 deallocate 函数，按需要释放所有分配的内存。重设当前缓冲区及下个缓冲区大小为其构造时的初始值。

内存被释放回上游资源，即使未对某些被分配块调用 deallocate。
