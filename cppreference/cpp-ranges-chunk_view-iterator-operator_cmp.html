比较底层迭代器（或底层迭代器与默认哨位 (2)）。

令 current_ 和 end_ 为底层数据成员。

等价于：

1) return x.current_ == y.current_;。
2) return x.current_ == x.end_;。
3) return x.current_ < y.current_;。
4) return y < x;。
5) return !(y < x);。
6) return !(x < y);。
7) return x.current_ <=> y.current_;。

这些函数对常规的无限定或有限定查找不可见，而只能在 std::ranges::chunk_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

x, y - 要比较的迭代器

返回值

比较的结果。
