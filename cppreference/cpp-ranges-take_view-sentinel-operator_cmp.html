比较 take_view::/*sentinel*/ 与 std::counted_iterator（常通过调用 take_view::begin 获得）。

若 y 指向第 N 个元素后一位置，或抵达了底层视图末尾则为 true，其中 N 为传递给 take_view 的构造函数的数。

仅用于阐释的别名模板 /*add-const-if*/ 定义为
template<bool C, class T>
using /*add-const-if*/ = std::conditional_t<C, const T, T>;。

这些函数对常规的无限定或有限定查找不可见，而只能在 std::ranges::take_view::sentinel<Const> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

y - 要比较的 std::counted_iterator
x - 要比较的哨位

返回值

y.count() == 0 || y.base() == x.end_，其中 end_ 代表底层哨位。
