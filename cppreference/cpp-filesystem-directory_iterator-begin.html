std::filesystem::begin(directory_iterator), std::filesystem::end(directory_iterator)

< cpp‎ | filesystem‎ | directory iterator
在标头 <filesystem> 定义
directory_iterator begin( directory_iterator iter ) noexcept;
 (1) (C++17 起)
directory_iterator end( directory_iterator ) noexcept;
 (2) (C++17 起)
1) 返回未更改的 iter。
2) 返回默认构造的 directory_iterator，它用作尾迭代器。忽略实参。

这些非模板函数允许将 directory_iterator 用于基于范围的 for 循环并使 directory_iterator 为 range 类型(C++20 起)。

参数

iter - directory_iterator

返回值

1) 不更改的 iter。
2) 尾迭代器（默认构造的 directory_iterator）。

示例

#include <filesystem>
#include <fstream>
#include <iostream>
namespace fs = std::filesystem;
 
int main()
{
    fs::create_directories("sandbox/a/b");
    std::ofstream("sandbox/file1.txt");
    std::ofstream("sandbox/file2.txt");
    for (auto& p : fs::directory_iterator("sandbox"))
        std::cout << p << '\n';
    fs::remove_all("sandbox");
}

可能的输出：

"sandbox/a"
"sandbox/file1.txt"
"sandbox/file2.txt"
