 (B)
1) 计算复数 z 的辐角（以弧度表示）。
A,B) 为所有整数和浮点类型提供额外重载，它们被处理为拥有零虚部的复数。
 (C++11 起)

参数

z - 复数值
f - 浮点值
i - 整数值

返回值

1) 如果没有发生错误，那么返回 z 在区间 (−π; π) 内的辐角。
A) 如果 f 为正数或 +0 则为零，如果 f 为负数或 -0 则为 π，否则为 NaN。
B) 如果 i 非负则为零，如果它是负数则为 π。

注解

额外重载不需要严格以 (A,B) 的形式提供。它们只需要能够对它们的实参 num 满足以下要求即可：

  • 如果 num 具有标准(C++23 前)浮点类型 T，那么 std::arg(num) 和 std::arg(std::complex<T>(num)) 的效果相同。
  • 否则，如果 num 具有整数类型，那么 std::arg(num) 和 std::arg(std::complex<double>(num)) 的效果相同。

示例

#include <complex>
#include <iostream>
 
int main() 
{
    std::complex<double> z1(1, 0);
    std::complex<double> z2(0, 0);
    std::complex<double> z3(0, 1);
    std::complex<double> z4(-1, 0);
    std::complex<double> z5(-1, -0.0);
    double f = 1.;
    int i = -1;
 
    std::cout << "phase angle of " << z1 << " is " << std::arg(z1) << '\n'
              << "phase angle of " << z2 << " is " << std::arg(z2) << '\n'
              << "phase angle of " << z3 << " is " << std::arg(z3) << '\n'
              << "phase angle of " << z4 << " is " << std::arg(z4) << '\n'
              << "phase angle of " << z5 << " is " << std::arg(z5) << " "
                 "(the other side of the cut)\n"
              << "phase angle of " << f << " is " << std::arg(f) << '\n'
              << "phase angle of " << i << " is " << std::arg(i) << '\n';
 
}

输出：

(1,0) 的幅角是 0
(0,0) 的幅角是 0
(0,1) 的幅角是 1.5708
(-1,0) 的幅角是 3.14159
(-1,-0) 的幅角是 -3.14159（切面的另一侧）
1 的幅角是 0
-1 的幅角是 3.14159
