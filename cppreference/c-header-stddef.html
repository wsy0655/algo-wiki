标准库标头 <stddef.h>

< c‎ | header


此标头是类型支持库的一部分，特别是，它提供了额外的基本类型和便利宏。

类型
ptrdiff_t
 两个指针相减返回的有符号整数类型
(typedef)
nullptr_t
(C23)
 预定义空指针常量 nullptr 的类型
(typedef)
max_align_t
(C11)
 对齐要求不小于任何其他标量类型的类型
(typedef)
size_t
 sizeof 运算符返回的无符号整数类型
(typedef)

常量
NULL
 实现定义的空指针常量
(宏常量)

宏
offsetof
 从结构体类型的起始到指定成员的字节偏移
(宏函数)

概要

#define __STDC_VERSION_STDDEF_H__ 202311L
 
typedef /* 见描述 */ ptrdiff_t;
typedef /* 见描述 */ nullptr_t;
typedef /* 见描述 */ max_align_t;
typedef /* 见描述 */ wchar_t;
typedef /* 见描述 */ size_t;
 
#define NULL /* 见描述 */
#define unreachable() /* 见描述 */
#define offsetof(P, D) /* 见描述 */

仅当实现定义了 __STDC_LIB_EXT1__ 而且用户代码在对 stddef.h 的所有包含之前定义了 __STDC_WANT_LIB_EXT1__：

#if defined(__STDC_WANT_LIB_EXT1__)
typedef /* 见描述 */ rsize_t;
#endif
