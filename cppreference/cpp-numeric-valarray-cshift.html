std::valarray<T>::cshift

< cpp‎ | numeric‎ | valarray
valarray<T> cshift( int count ) const;

返回大小相同的新 valarray，它的每个元素的位置被循环迁移 count 个元素。

非负 count 值会将元素向左循环迁移 count 个位置，负 count 值会将元素向右循环迁移 -count 个位置。

参数

count - 元素迁移的位置数

返回值

作为结果的 valarray，拥有循环迁移后的元素。

注解

函数可以实现为拥有不同于 std::valarray 的返回类型。此时替换它的类型拥有下列属性：

  • 提供 std::valarray 的所有 const 成员函数。
  • 能从替换类型构造 std::valarray、std::slice_array、std::gslice_array、std::mask_array 和 std::indirect_array。
  • 所有接受一个 const std::valarray& 类型参数的函数，除了 begin() 和 end()(C++11 起)也应该接受替换类型。
  • 所有接受两个 const std::valarray& 类型参数的函数都应该接受 const std::valarray& 和替换类型的每种组合。
  • 返回类型添加不多于两层嵌套在最深层嵌套的参数类型上的模板。

示例

#include <iostream>
#include <valarray>
 
int main()
{
    std::valarray<int> v{1, 2, 3, 4, 5, 6, 7, 8};
 
    for (auto const& val : v)
        std::cout << val << " ";
    std::cout << "\n";
 
    std::valarray<int> v2 = v.cshift(2);
 
    for (auto const& val : v2)
        std::cout << val << " ";
    std::cout << "\n";
}

输出：

1 2 3 4 5 6 7 8 
3 4 5 6 7 8 1 2
