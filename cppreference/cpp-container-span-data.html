std::span<T,Extent>::data

< cpp‎ | container‎ | span
constexpr pointer data() const noexcept;
 (C++20 起)

返回指向底层序列起始的指针。

返回值

data_

示例

#include <iostream>
#include <span>
 
int main()
{
    constexpr char str[] = "ABCDEF\n";
 
    const std::span sp{str};
 
    for (auto n{sp.size()}; n != 2; --n)
        std::cout << sp.last(n).data();
}

输出：

ABCDEF
BCDEF
CDEF
DEF
EF
F
