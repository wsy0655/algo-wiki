std::erase_if (std::flat_multimap)

< cpp‎ | container‎ | flat multimap

在标头 <flat_map> 定义
template< class Key, class T, class Compare, class KeyContainer, class MappedContainer,

          class Pred >
std::flat_multimap<Key, T, Compare, KeyContainer, MappedContainer>::size_type
    erase_if( std::flat_multimap<Key, T, Compare, KeyContainer, MappedContainer>& c,

              Pred pred );
 (C++23 起)

从 c 中擦除所有满足谓词 pred 的元素。

当表达式 bool(pred(std::pair<const Key&, const T&>(e))) 为 true 时满足谓词 pred，其中 e 为 c 中的某个元素。

Key 和 T 必须可移动赋值 (MoveAssignable) ，否则，其行为未定义。

参数

c - 要从中擦除的容器适配器
pred - 若应该擦除元素则对它返回 true 的谓词

返回值

擦除的元素数。

复杂度

恰好 c.size() 次运用谓词 pred。

异常

如果 erase_if 抛出异常，c 保留处于有效但未指明的（可能为空）状态。

注解

此算法稳定，即不改变未被删除的元素之间的顺序。

示例

#include <iostream>
#include <flat_map>
 
void println(auto rem, auto const& container)
{
    std::cout << rem << '{';
    for (char sep[]{0, ' ', 0}; const auto& [key, value] : container)
        std::cout << sep << '{' << key << ", " << value << '}', *sep = ',';
    std::cout << "}\n";
}
 
int main()
{
    std::flat_multimap<int, char> data
    {
        {1, 'a'}, {2, 'b'}, {3, 'c'}, {4, 'd'},
        {5, 'e'}, {4, 'f'}, {5, 'g'}, {5, 'g'},
    };
    println("起初:\n", data);
 
    const auto count = std::erase_if(data, [](const auto& item)
    {
        auto const& [key, value] = item;
        return (key & 1) == 1;
    });
 
    println("擦除奇数键项:\n", data);
    std::cout << count << " 项被移除。\n";
}

输出：

起初:
{{1, a}, {2, b}, {3, c}, {4, d}, {4, f}, {5, e}, {5, g}, {5, g}}
Erase items with odd keys:
{{2, b}, {4, d}, {4, f}}
5 项被移除。
