std::ranges::chunk_by_view<V,Pred>::end

< cpp‎ | ranges‎ | chunk by view
constexpr auto end();
 (C++23 起)

返回表示 chunk_by_view 的末尾的迭代器或哨位。

等价于：

if constexpr (ranges::common_range<V>)
    return /*iterator*/(*this, ranges::end(base_), ranges::end(base_));
else
    return std::default_sentinel;

参数

（无）

返回值

指向最后一个元素之后的元素的迭代器，或者比较等于尾迭代器的哨位。

注解

当且仅当底层视图是 common_range 时，end() 返回迭代器：只要 V 实现 common_range，那么 chunk_by_view<V,Pred> 也实现它。
