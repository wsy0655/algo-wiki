enum class launch : /* 未指定 */ {

 (C++11 起)

std::launch 是一个位掩码类型 (BitmaskType) 。它指定 std::async 所执行的任务的启动策略。

常量

标准库定义下列代表单独位的常量：

枚举项 含义
async 在不同线程上执行任务，可能要首先创建并启动它
deferred 调用方线程上首次请求其结果时执行任务（惰性求值）

另外，实现也可以：

  • 定义额外的位和位掩码，以指定可应用于运行策略子集的任务交互上的限制，并
  • 对 std::async 的首个（默认）重载启用这些额外位掩码。
