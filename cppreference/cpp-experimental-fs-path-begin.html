std::experimental::filesystem::path::begin, std::experimental::filesystem::path::end

< cpp‎ | experimental‎ | fs‎ | path
iterator begin() const;
 (1) (文件系统 TS)
iterator end() const;
 (2) (文件系统 TS)
1) 返回指向路径首元素的迭代器。如果路径为空，则返回的迭代器等于 end()。
2) 返回越过路径最末元素后一位的迭代器。解引用此迭代器是未定义行为。

这样一对迭代器所代表的序列由以下各项组成：

1) 根名称（如果有）。
2) 根目录（如果有）。
3) 文件名 的序列，忽略任何目录分隔符。
4) 如果路径中最后一个 file-name 之后有目录分隔符，则末尾迭代器之前的最末元素为一个虚设的 点 文件名。

参数

（无）

返回值

1) 指向路径首个元素的迭代器。
2) 越过路径末尾一位的迭代器。

异常

可能会抛出由实现定义的异常。

示例

#include <experimental/filesystem>
#include <iostream>
namespace fs = std::experimental::filesystem;
 
int main()
{
    fs::path p = "C:\\users\\abcdef\\AppData\\Local\\Temp\\";
    std::cout << "通过迭代器检查路径 " << p << " 得到\n";
    for (auto& e : p)
        std::cout << e << '\n';
}
