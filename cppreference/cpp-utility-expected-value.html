如果 *this 包含预期值，那么返回到该预期值的引用。对于 void 部分特化不会返回值。

否则，抛出包含 error() 的副本的 std::bad_expected_access<std::decay_t<E>> 类型的异常。

1,2) 如果 std::is_copy_constructible_v<E> 是 false，那么程序非良构。
3,4) 如果 std::is_copy_constructible_v<E> 或 std::is_constructible_v<E, decltype(std::move(error()))> 是 false，那么程序非良构。
5) 如果 std::is_copy_constructible_v<E> 是 false，那么程序非良构。
6) 如果 std::is_move_constructible_v<E> 是 false，那么程序非良构。

返回值

1,2) val
3,4) std::move(val)

异常

1,2,5) 在 *this 包含非预期值时抛出 std::bad_expected_access(std::as_const(error()))。
3,4,6) 在 *this 包含非预期值时抛出 std::bad_expected_access(std::move(error()))。
