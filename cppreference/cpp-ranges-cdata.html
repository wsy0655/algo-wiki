返回指向 const 限定的(C++23 前)实参所代表的连续范围的常量类型的(C++23 起)首元素的指针。

令 CT

  • 若实参为左值（即 T 为左值引用类型）则为 const std::remove_reference_t<T>&，
  • 否则为 const T。

则对 ranges::cdata 的调用表达式等价于

  • as-const-pointer(ranges::data(possibly-const-range(t)))。

返回类型等价于 std::remove_reference_t<ranges::range_reference_t<CT>>*。

 (C++23 前)

若实参为作者，或者 ranges::enable_borrowed_range<std::remove_cv_t<T>> 为 true，则对 ranges::cdata 的调用表达式等价于：

  • /*as-const-pointer*/(ranges::data(/*possibly-const-range*/(t)))。

返回类型等价于 std::remove_reference_t<ranges::range_const_reference_t<T>>*。

所有其他情况下，对 ranges::cdata 的调用都非良构，在调用出现于模板实例化的立即语境中时会导致代换失败。

 (C++23 起)

定制点对象

名字 ranges::cdata 代表一个定制点对象，它是某个字面 semiregular 类类型的 const 函数对象。 细节参见定制点对象 (CustomizationPointObject) 。

示例

#include <cstring>
#include <iostream>
#include <ranges>
#include <string>
 
int main()
{
    std::string src {"hello world!\n"};
 
//  std::ranges::cdata(src)[0] = 'H'; // 错误，src.data() 被当作只读
    std::ranges::data(src)[0] = 'H'; // OK，src.data() 为非 const 存储
 
    char dst[20]; // C 风格字符串的存储
    std::strcpy(dst, std::ranges::cdata(src));
    // [data(src), data(src) + size(src)] 保证为 NTBS
 
    std::cout << dst;
}

输出：

Hello world!
