std::hash 对各种字符串类的模板特化允许用户获得字符串的散列。

这些散列值与对应的 std::basic_string_view 类的散列相等：若 S 是这些字符串类型之一，SV 是对应的字符串视图类型，而 s 是 S 类型的对象，则 std::hash<S>()(s) == std::hash<SV>()(SV(s))。

 (C++17 起)

示例

下列代码显示 string 上使用的散列函数的一种可能输出：

#include <functional>
#include <iostream>
#include <memory_resource>
#include <string>
#include <string_view>
using namespace std::literals;
 
int main()
{
    auto sv = "Stand back! I've got jimmies!"sv;
    std::string s(sv);
    std::pmr::string pmrs(sv); // 使用默认分配器
 
    std::cout << std::hash<std::string_view>{}(sv) << '\n';
    std::cout << std::hash<std::string>{}(s) << '\n';
    std::cout << std::hash<std::pmr::string>{}(pmrs) << '\n';
}
