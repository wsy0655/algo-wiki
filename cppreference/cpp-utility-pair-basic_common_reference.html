struct basic_common_reference<std::pair<T1, T2>, std::pair<U1, U2>, TQual, UQual>;
 (C++23 起)

两个 pair 的公共引用类型是两个 pair 的对应元素类型的两个公共引用类型的 pair，其中 pair 上的 cv 与引用限定符应用到其元素类型上。

仅当两对对应的元素类型都拥有公共引用类型时，公共引用类型才有定义。
