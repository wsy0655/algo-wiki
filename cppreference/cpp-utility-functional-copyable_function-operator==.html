operator==(std::copyable_function)

< cpp‎ | utility‎ | functional‎ | copyable function


friend bool operator==( const std::copyable_function& f, std::nullptr_t ) noexcept;
 (C++26 起)

将包装器 f 和 std::nullptr_t 比较以检查其是否为空。空包装器（即没有目标的包装）与之比较相等，非空函数比较则不相等。

此函数对常规的无限定或有限定查找不可见，而只能在 std::copyable_function<FunctionType> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

f - 要比较的 std::copyable_function

返回值

!f。
