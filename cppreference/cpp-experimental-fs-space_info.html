std::experimental::filesystem::space_info

< cpp‎ | experimental‎ | fs
在标头 <experimental/filesystem> 定义
struct space_info {

    uintmax_t capacity;
    uintmax_t free;
    uintmax_t available;

};
 (文件系统 TS)

表示由 space 确定的文件系统信息。

各成员含义如下：

  • capacity——文件系统的总大小，以字节计
  • free——文件系统的自由空间，以字节计
  • available——对非特权进程可用的自由空间（可能等于或小于 free）

示例

#include <experimental/filesystem>
#include <iostream>
namespace fs = std::experimental::filesystem;
 
int main()
{
    fs::space_info devi = fs::space("/dev/null");
    fs::space_info tmpi = fs::space("/tmp");
 
    std::cout << "         Capacity         Free    Available\n"
              << "/dev:   " << devi.capacity << "   "
              << devi.free << "   " << devi.available << '\n'
              << "/tmp: " << tmpi.capacity << ' '
              << tmpi.free << ' ' << tmpi.available << '\n';
}

可能的输出：

          Capacity         Free    Available
/dev:   4175114240   4175110144   4175110144
/tmp: 420651237376 411962273792 390570749952
