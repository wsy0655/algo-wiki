1) 表示 view 的范围适配器，由拉平范围的视图得到的序列组成，每个元素视图元素之间插入了分隔符。分隔符可以是单个元素，也可以是元素视图。
/*concatable*/ 的定义见 std::ranges::concat_view。
2) 范围适配器对象 (RangeAdaptorObject) 。对于任何合适的子表达式 e 和 f 来说，views::join_with(e, f) 表达式等价于 join_with_view(e, f)。

join_with_view 实现 input_range。

join_with_view 实现 forward_range，仅当：

  • ranges::range_reference_t<V> 是引用，且
  • V 和 ranges::range_reference_t<V> 都实现 forward_range。

join_with_view 实现 bidirectional_range，仅当：

  • ranges::range_reference_t<V> 是引用，且
  • V，ranges::range_reference_t<V> 和 Pattern 都实现 bidirectional_range，且
  • ranges::range_reference_t<V> 和 Pattern 都实现 common_range。

join_with_view 实现 common_range，仅当：

  • ranges::range_reference_t<V> 是引用，
  • V 和 ranges::range_reference_t<V> 都实现 forward_range 和 common_range。

示例

#include <iostream>
#include <ranges>
#include <string_view>
#include <vector>
 
int main()
{
    using namespace std::literals;
 
    std::vector v{"This"sv, "is"sv, "a"sv, "test."sv};
    auto joined = v | std::views::join_with(' ');
 
    for (auto c : joined)
        std::cout << c;
    std::cout << '\n';
}

输出：

This is a test.
