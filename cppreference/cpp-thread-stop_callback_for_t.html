std::stop_callback_for_t

< cpp‎ | thread
在标头 <stop_token> 定义
template< class T, class CallbackFn >
using stop_callback_for_t = T::template callback_type<CallbackFn>;
 (C++26 起)

别名模板 stop_callback_for_t 用户获取 T 类型的停止回调类型。

注解

对于标准停止令牌类型和任何有效的 CallbackFn，相应的 stop_callback_for_t 为以下之一：

  • std::stop_callback 针对 std::stop_token 的特化，
  • std::inplace_stop_callback 针对 std::inplace_stop_token 的特化，以及
  • 未指明的类型，不带有用于 std::never_stop_token 的停止回调注册和注销。
