std::new_handler

< cpp‎ | memory‎ | new
在标头 <new> 定义
typedef void (*new_handler)();

std::new_handler 是函数指针类型（指向不接收参数并返回 void 的指针），它为函数 std::set_new_handler 与 std::get_new_handler 所用。
