                                std::memory_order order ) noexcept;
 (3) (C++20 起)
bool atomic_flag_test_explicit( const std::atomic_flag* object,
                                std::memory_order order ) noexcept;
 (4) (C++20 起)

原子地读取 *object 的值并返回该值。

1,2) 内存同步定序是 std::memory_order_seq_cst。
3,4) 内存同步定序是 order。
如果 order 是 std::memory_order::release 和 std::memory_order::acq_rel 之一，那么行为未定义。

参数

object - 指向要读取的 atomic_flag 对象的指针
order - 内存同步定序

返回值

原子地读取的值。
