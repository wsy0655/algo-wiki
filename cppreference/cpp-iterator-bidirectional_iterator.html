std::bidirectional_iterator

< cpp‎ | iterator
在标头 <iterator> 定义
template< class I >

    concept bidirectional_iterator =
        std::forward_iterator<I> &&
        std::derived_from</*ITER_CONCEPT*/<I>, std::bidirectional_iterator_tag> &&
        requires(I i) {
            { --i } -> std::same_as<I&>;
            { i-- } -> std::same_as<I>;

        };
 (C++20 起)

概念 bidirectional_iterator 精化了 forward_iterator，添加向后移动迭代器的能力。

迭代器概念确定

此概念的定义通过仅用于阐述的别名模板 /*ITER_CONCEPT*/ 说明。

为确定 /*ITER_CONCEPT*/<I>，令 ITER_TRAITS<I> 在特化 std::iterator_traits<I> 从主模板生成时代表 I，否则代表 std::iterator_traits<I>：

  • 如果 ITER_TRAITS<I>::iterator_concept 合法并指名类型，那么 /*ITER_CONCEPT*/<I> 代表该类型。
  • 否则，如果 ITER_TRAITS<I>::iterator_category 合法并指名类型，那么 /*ITER_CONCEPT*/<I> 代表该类型。
  • 否则，如果 std::iterator_traits<I> 从主模板生成，那么 /*ITER_CONCEPT*/<I> 指代 std::random_access_iterator_tag。
  • 否则 /*ITER_CONCEPT*/<I> 不代表类型并导致替换失败。

语义要求

当且仅当存在某个 s 使得 ++s == r 时，称双向迭代器 r 可自减。

仅当 bidirectional_iterator<I> 所纳入的概念均被实现，并且给定两个 I 类型对象 a 与 b 满足以下条件时，它得到实现：

  • 若 a 可自减，则 a 处于表达式 --a 与 a-- 的定义域中。
  • 前自减生成指代操作数的左值：std::addressof(--a) == std::addressof(a)。
  • 后自减生成操作数先前的值：若 bool(a == b)，则 bool(a-- == b)。
  • 后自减和前自减在操作数上设施相同修改：若 bool(a == b)，则求值 a-- 与 --b 两者后，bool(a == b) 仍然成立。
  • 自增和自减互逆：
  • 若 a 可自增且 bool(a == b)，则 bool(--(++a) == b)。
  • 若 a 可自减且 bool(a == b)，则 bool(++(--a) == b)。

相等性保持

标准库概念的 requires 表达式中声明的表达式都要求保持相等性（除非另外说明）。

注解

不同于老式双向迭代器 (LegacyBidirectionalIterator) 的要求，bidirectional_iterator 概念不要求解引用返回左值。

示例

最小化的双向迭代器。

#include <cstddef>
#include <iterator>
 
struct SimpleBidiIterator
{
    using difference_type = std::ptrdiff_t;
    using value_type = int;
 
    int operator*() const;
 
    SimpleBidiIterator& operator++();
 
    SimpleBidiIterator operator++(int)
    {
        auto tmp = *this;
        ++*this;
        return tmp;
    }
 
    SimpleBidiIterator& operator--();
 
    SimpleBidiIterator operator--(int)
    {
        auto tmp = *this;
        --*this;
        return tmp;
    }
 
    bool operator==(const SimpleBidiIterator&) const;
};
 
static_assert(std::bidirectional_iterator<SimpleBidiIterator>);
