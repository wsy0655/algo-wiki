std::basic_istream<CharT,Traits>::swap

< cpp‎ | io‎ | basic istream
protected:
void swap( basic_istream& rhs );
 (C++11 起)

调用 basic_ios::swap(rhs) 交换基类除了 rdbuf() 之外的所有数据成员，再交换 *this 和 rhs 间的 gcount() 计数器值。此 swap 函数受保护：它被可交换输入流类 std::basic_ifstream 和 std::basic_istringstream 的 swap 函数所调用，它们知道如何正确地交换关联流缓冲。

参数

rhs - 与之交换的同一类型的不同 basic_istream 对象

示例

#include <iostream>
#include <sstream>
#include <utility>
 
int main()
{
    std::istringstream s1("hello");
    std::istringstream s2("bye");
 
    s1.swap(s2); // OK，istringstream 拥有公开的 swap()
    std::swap(s1, s2); // OK，调用 s1.swap(s2)
//  std::cin.swap(s2); // 错误：swap 是受保护成员
 
    std::cout << s1.rdbuf();
}

输出：

hello
