std::regex_traits<CharT>::regex_traits

< cpp‎ | regex‎ | regex traits
regex_traits();

默认构造 std::regex_traits 对象，包含默认构造私有 std::locale 成员或任何其他必要的内部状态。

参数

（无）

返回值

（无）

示例

#include <iostream>
#include <regex>
 
int main()
{
    std::locale::global(std::locale("en_US.utf8"));
    std::regex_traits<char> r1;
    std::regex_traits<wchar_t> r2;
    std::cout << "正则表达式的本地环境为 " << r1.getloc().name() << '\n';
}

输出：

正则表达式的本地环境为 en_US.utf8
