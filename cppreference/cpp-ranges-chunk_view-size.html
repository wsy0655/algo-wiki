std::ranges::chunk_view<V>::size

< cpp‎ | ranges‎ | chunk view
constexpr auto size() requires ranges::sized_range<V>;
 (1) (C++23 起)
constexpr auto size() const requires ranges::sized_range<const V>;
 (2) (C++23 起)

返回元素数量，即不小于底层视图 base_ 大小除以底层数据成员 n_ 的商的最小整数值，n_ 持有传给构造函数的值（默认构造时为 ​0​）。等价于： return to-unsigned-like(div-ceil(ranges::distance(base_), n_)); 。

返回值

元素数量。

示例

#include <ranges>
 
int main()
{
    constexpr static auto v = {1, 2, 3, 4, 5};
    constexpr auto w{ std::ranges::chunk_view(v, 2) };
    static_assert(w.size() == (5 / 2 + (5 % 2 ? 1 : 0)));
}
