operator==(ranges::chunk_view::inner-iterator)

friend constexpr bool operator==( const /*inner-iterator*/& x,
                                  std::default_sentinel_t );
 (C++23 起)

比较迭代器与哨位。

令 parent_ 为指向外围视图 chunk_view 的底层指针。

等价于：returns x.parent_->remainder_ == 0;

此函数对常规的无限定或有限定查找不可见，而只能在 std::ranges::chunk_view::inner-iterator 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

x - 要比较的迭代器

返回值

比较的结果。
