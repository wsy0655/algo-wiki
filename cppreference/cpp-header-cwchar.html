标准库标头 <cwchar>

< cpp‎ | header


此标头原作为 <wchar.h> 存在于 C 标准库。

此头文件是空终止宽及多字节字符串库的一部分。 它也提供了一些 C 风格输入/输出函数和源自 C 风格日期的转换。

注解

  • NULL 也在以下标头定义：
    • <cstddef>
    • <cstring>
    • <ctime>
    • <clocale>
    • <cstdio>
  • std::size_t 也在以下标头定义：
    • <cstddef>
    • <ctime>
    • <cstdio>
  • std::wint_t 也在以下标头定义：
    • <cwctype>
  • std::tm 也在以下标头定义：
    • <ctime>
