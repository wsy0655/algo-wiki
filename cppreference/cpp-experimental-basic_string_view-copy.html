std::experimental::basic_string_view<CharT,Traits>::copy

< cpp‎ | experimental‎ | basic string view
size_type copy( CharT* dest,

                size_type count,

                size_type pos = 0) const;
 (库基础 TS)

将子字符串 [pos, pos + rcount) 复制到由 dest 指向的字符串，其中 rcount 为 count 和 size() - pos 中的较小值。

参数

dest - 指向目标字符串的指针
pos - 首个字符的位置
count - 所请求的子字符串长度

返回值

所复制的字符数

异常

当 pos > size() 时抛出 std::out_of_range。

复杂度

与 rcount 呈线性。

参阅

substr
 返回子字符串
(公开成员函数)
