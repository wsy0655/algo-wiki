std::strcat

< cpp‎ | string‎ | byte
在标头 <cstring> 定义
char* strcat( char* dest, const char* src );

后附 src 所指向的空终止字节字符串的副本到 dest 所指向的空终止字节字符串的结尾。字符 src[0] 替换 dest 末尾的空终止符。产生的字节字符串是空终止的。

若目标数组对于 src 和 dest 的内容以及空终止符不够大，则行为未定义。

若字符串重叠，则行为未定义。

参数

dest - 指向要后附到的空终止字节字符串的指针
src - 指向作为复制来源的空终止字节字符串的指针

返回值

dest

注解

因为 strcat 需要在每次调用时找到 dest 的结尾，故用 strcat 将多个字符串连接成一体是低效的。

示例

#include <cstdio>
#include <cstring>
 
int main() 
{
    char str[50] = "Hello ";
    char str2[50] = "World!";
    std::strcat(str, str2);
    std::strcat(str, " Goodbye World!");
    std::puts(str);
}

输出：

Hello World! Goodbye World!
