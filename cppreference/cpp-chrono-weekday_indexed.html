std::chrono::weekday_indexed

< cpp‎ | chrono
在标头 <chrono> 定义
class weekday_indexed;
 (C++20 起)

类 weekday_indexed 把表示外推格里高利历中星期之日的 weekday，与范围 [1, 5] 中的小索引 n 结合。它表示某个月中的第一、第二、第三、第四或第五个星期之日。

weekday_indexed 是可平凡复制 (TriviallyCopyable) 的标准布局类型 (StandardLayoutType) 。

示例

#include <chrono>
#include <iostream>
 
int main()
{
    using namespace std::chrono;
 
    constexpr weekday_indexed wi = Friday[2];
 
    // 带索引的星期之日可以用于任何可以使用 chrono::day 的地方：
    constexpr year_month_weekday ymwd = 2021y / August / wi;
    static_assert(ymwd == August / wi / 2021y and
                  ymwd == wi / August / 2021y);
    std::cout << ymwd << '\n';
 
    constexpr year_month_day ymd{ymwd}; // 转换
    static_assert(ymd == 2021y / 8 / 13);
    std::cout << ymd << '\n';
}
