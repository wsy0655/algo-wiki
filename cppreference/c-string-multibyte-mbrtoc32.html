mbrtoc32

< c‎ | string‎ | multibyte
在标头 <uchar.h> 定义
size_t mbrtoc32( char32_t restrict* pc32, const char* restrict s,
                 size_t n, mbstate_t* restrict ps );
 (C11 起)

从窄多字符编码转换编码点到其变长 32 位宽字符表示（但典型地为 UTF-32）。

若 s 不是空指针，则审查以 s 所指向的字节开始的多字节字符串的至多 n 个字节，以确定完成下个多字节字符所需的字节数（包含任何迁移序列，并计入当前多字节转换状态 *ps）。若函数确定 s 中下个多字节字符完整且合法，则将它转换成对应的 32 位宽字符并存储于 *pc32（若 pc32 非空）。

若 *s 中下个多字节字符对应多 char32_t 序列（对于 UTF-32 不可能），则在首次调用此函数后更新 *ps，以满足下次到 mbrtoc32 的调用将写出附加的 char32_t，而不考虑 *s。

若 s 是空指针，则忽略 n 与 pc32 的值，而调用等价于 mbrtoc32(NULL, "", 1, ps)。

若产生的宽字符是空字符，则转换状态 *ps 表示初始迁移状态。

若定义了宏 __STDC_UTF_32__，则此函数所用的 32 位编码是 UTF-32；否则它是实现定义的。总是定义这个宏，且编码总是 UTF-32。(C23 起)任何情况下，此函数所用的多字节编码为当前活跃的 C 本地环境所指定。

参数

pc32 - 指向将写入产生的 32 位宽字符的位置的指针
s - 指向用作输入的多字节字符串的指针
n - s 中能检验的字节数上的限制
ps - 指向转译多字节字符串时所用的转换状态对象的指针

返回值

下列首个适用者：

  • 若从 s 转换的字符为空字符（并存储于 *pc32 中，若它非空）则为 ​0​。
  • 成功从 s 转换的多字节字符 [1, n] 的字节数。
  • 若现已将来自多 char32_t 字符的下个 char32_t 写入 *pc32，则为 (size_t)-3。此情况下不处理输入的任何字节。
  • 若接下来 n 个字节组成不完整，但到此仍合法的多字节字符则为 (size_t)-2。不写入内容到 *pc32。
  • 若出现编码错误则为 (size_t)-1。不写入内容到 *pc32，存储 EILSEQ 于 errno 且 *ps 的值未指定。

示例

MSVC 上需要编译器标志 /utf-8 使得 UTF_8 正常工作。

#include <assert.h>
#include <locale.h>
#include <stdio.h>
#include <uchar.h>
 
int main(void)
{
    setlocale(LC_ALL, "en_US.utf8");
    char in[] = u8"zß水🍌"; // 即 "z\u00df\u6c34\U0001F34C"
    enum { in_sz = sizeof in / sizeof *in };
 
    printf("%d 个 UTF-8 码元：[", in_size);
    for (int i = 0; i < in_size; ++i)
        printf("%s%02X", i ? " " : "", (unsigned char)in[i]);
 
    puts("]");
 
    char32_t out[in_size];
    char32_t* p_out = out;
    char* p_in = in;
    char* end = in + in_size;
    mbstate_t state = {0};
 
    size_t rc;
    while ((rc = mbrtoc32(p_out, p_in, end - p_in, &state)))
    {
        assert(rc != (size_t)-3); // UTF-32 中无代理对
        if (rc == (size_t)-1) break; // 无效输入
        if (rc == (size_t)-2) break; // 截断的输入
        p_in += rc;
        ++p_out;
    }
 
    size_t out_size = p_out+1 - out;
    printf("处理成 %zu 个 UTF-32 码元：[", out_size);
    for (size_t i = 0; i < out_size; ++i)
        printf("%s%08X", i ? " " : "", out[i]);
 
    puts("]");
}

输出：

11 个 UTF-8 码元：[7A C3 9F E6 B0 B4 F0 9F 8D 8C 00]
处理成 5 个 UTF-32 码元：[0000007A 000000DF 00006C34 0001F34C 00000000]
