std::perror

< cpp‎ | io‎ | c
在标头 <cstdio> 定义
void perror( const char *s );

打印当前存储于系统变量 errno 的错误码的文本表示到 stderr。

通过连接下列组分构成描述：

  • s 所指向的空终止字节字符串的内容后随 ": "（除非 s 为空指针或 s 所指向字符为空字符）。
  • 描述存储于 errno 的错误码的实现定义的错误消息字符串，后随 '\n'。错误消息字符串等同于 std::strerror(errno) 的结果。

参数

s - 指向拥有解释性消息的空终止字符串的指针

返回值

（无）

示例

#include <cerrno>
#include <cmath>
#include <cstdio>
 
int main()
{
    double not_a_number = std::log(-1.0);
    if (errno == EDOM)
        std::perror("log(-1) failed");
    std::printf("%f\n", not_a_number);
}

可能的输出：

log(-1) failed: Numerical argument out of domain
nan
