C 风格的文件输入/输出

< cpp‎ | io


C++ 标准库的 C 输入/输出子集实现了 C 风格流的输入/输出操作。<cstdio> 头文件提供了对通用文件操作的支持，并提供具有窄和多字节字符的输入/输出能力的函数，而 <cwchar> 头文件提供了具有宽字符输入/输出能力的函数。

C 流以 std::FILE 类型的对象所代表，只能通过 std::FILE* 类型的指针予以访问及操作。每个 C 流均与外部物理设备（文件、标准输入流、打印机、串口等）相关联。
