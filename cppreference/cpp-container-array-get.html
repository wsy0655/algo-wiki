使用元组式接口从 array 提取第 I 个元素。

I 必须是范围 [​0​, N) 中的整数值。与 at() 或 operator[] 相反，这在编译时强制。

参数

a - 要提取其内容的数组

返回值

到 a 的第 I 元素的引用。

复杂度

常数。

示例

#include <array>
#include <iostream>
 
constexpr std::array v{1, 2, 3};
static_assert(get<0>(v) == 1 && get<1>(v) == 2 && get<2>(v) == 3);
 
int main()
{
    std::array<int, 3> a;
 
    // 设置值：
    get<0>(a) = 1, get<1>(a) = 2, get<2>(a) = 3;
 
    // 获取值：
    std::cout << '(' << get<0>(a) << ',' << get<1>(a) << ',' << get<2>(a) << ")\n";
}

输出：

(1,2,3)
