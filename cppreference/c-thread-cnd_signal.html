cnd_signal

< c‎ | thread
在标头 <threads.h> 定义
int cnd_signal( cnd_t *cond );
 (C11 起)

接触当前在 cond 所指向的条件变量上等待的一个线程的阻塞。若无线程被阻塞，则不做任何事并返回 thrd_success。

参数

cond - 指向条件变量的指针

返回值

若成功则为 thrd_success，否则为 thrd_error。
