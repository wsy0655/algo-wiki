std::ios_base::event

< cpp‎ | io‎ | ios base
enum event { erase_event, imbue_event, copyfmt_event };

指定在特殊事件时传递给 register_callback() 所注册函数的事件类型。定义了下列常量：

常量 解释
erase_event 在 ~ios_base() 或 basic_ios::copyfmt()（发生成员复制前）时发出
imbue_event 在 imbue() 时发出
copyfmt_event 在 basic_ios::copyfmt()（发生成员复制后，但在复制异常设置前）时发出
