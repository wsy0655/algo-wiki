std::variant<Types...>::visit

< cpp‎ | utility‎ | variant
template< class Self, class Visitor >
constexpr decltype(auto) visit( this Self&& self, Visitor&& vis );
 (1) (C++26 起)
template< class R, class Self, class Visitor >
constexpr R visit( this Self&& self, Visitor&& vis );
 (2) (C++26 起)

向 self 保有的变体应用探访器 vis（可以变体中类型的任何组合的调用的可调用 (Callable) ）。

给定类型 V 为 decltype(std::forward_like<Self>(std::declval<variant>()))，等价的调用是：

1) return std::visit(std::forward<Visitor>(vis), (V) self);。
2) return std::visit<R>(std::forward<Visitor>(vis), (V) self);。

参数

vis - 接受变体中每种可能可选项的可调用 (Callable)
self - 要传递给探访器的变体

返回值

1) std::visit 调用的结果。
2) 当 R 为（可能有 cv 限定的）void 时无返回值；否则为调用 std::visit<R> 的结果。

异常

仅在调用 std::visit 有抛出时抛出。

注解

功能特性测试宏 值 标准 功能特性
__cpp_lib_variant 202306L (C++26) 成员 visit

示例

#include <print>
#include <string>
#include <string_view>
#include <variant>
 
struct Base {};
struct Derived : Base {};
 
// 探访器的辅助类型
template<class... Ts>
struct overloads : Ts... { using Ts::operator()...; };
 
// 要探访的变体
using var_t = std::variant<int, std::string, Derived>;
 
int main()
{
    const auto visitor = overloads
    {
        [](int i){ std::print("int = {}\n", i); },
        [](std::string_view s){ std::println("string = “{}”", s); },
        [](const Base&){ std::println("base"); }
    };
 
    const var_t var1 = 42, var2 = "abc", var3 = Derived();
 
#if (__cpp_lib_variant >= 202306L)
    var1.visit(visitor);
    var2.visit(visitor);
    var3.visit(visitor);
#else
    std::visit(visitor, var1);
    std::visit(visitor, var2);
    std::visit(visitor, var3);
#endif
}

输出：

int = 42
string = “abc”
base
