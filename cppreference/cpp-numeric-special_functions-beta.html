错误处理

可能报告 math_errhandling 中指定的错误。

额外重载不需要严格以 (A) 的形式提供。它们只需要能够对它们的第一个实参 num1 和第二个实参 num2 满足以下要求即可：

  • 如果 num1 或 num2 具有 long double 类型，那么 std::beta(num1, num2) 和 std::beta(static_cast<long double>(num1),
              static_cast<long double>(num2))     的效果相同。
  • 否则，如果 num1 和/或 num2 具有 double 或整数类型，那么 std::beta(num1, num2) 和 std::beta(static_cast<double>(num1),
              static_cast<double>(num2))     的效果相同。
  • 否则，如果 num1 或 num2 具有 float 类型，那么 std::beta(num1, num2) 和 std::beta(static_cast<float>(num1),
              static_cast<float>(num2))     的效果相同。

如果 num1 和 num2 具有算术类型，那么 std::beta(num1, num2) 和 std::beta(static_cast</* 公共浮点数类型 */>(num1),

示例

#include <cassert>
#include <cmath>
#include <iomanip>
#include <iostream>
#include <numbers>
#include <string>
 
long binom_via_beta(int n, int k)
{
    return std::lround(1 / ((n + 1) * std::beta(n - k + 1, k + 1)));
}
 
long binom_via_gamma(int n, int k)
{
    return std::lround(std::tgamma(n + 1) /
                      (std::tgamma(n - k + 1) * 
                       std::tgamma(k + 1)));
}
 
int main()
{
    std::cout << "帕斯卡三角形：\n";
    for (int n = 1; n < 10; ++n)
    {
        std::cout << std::string(20 - n * 2, ' ');
        for (int k = 1; k < n; ++k)
        {
            std::cout << std::setw(3) << binom_via_beta(n, k) << ' ';
            assert(binom_via_beta(n, k) == binom_via_gamma(n, k));
        }
        std::cout << '\n';
    }
 
    // 点检查
    const long double p = 0.123; // [0, 1] 中的随机值
    const long double q = 1 - p;
    const long double π = std::numbers::pi_v<long double>;
    std::cout << "\n\n" << std::setprecision(19)
              << "β(p,1-p)   = " << std::beta(p, q) << '\n'
              << "π/sin(π*p) = " << π / std::sin(π * p) << '\n';
}
