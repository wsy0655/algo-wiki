std::basic_stacktrace<Allocator>::empty

< cpp‎ | utility‎ | basic stacktrace
bool empty() const noexcept;
 (C++23 起)

检查栈踪迹是否无栈踪迹条目。

参数

（无）

返回值

若栈踪迹为空则为 true，否则为 false。

复杂度

常数。

示例

#include <stacktrace>
#include <iostream>
 
int main()
{
    std::cout << std::boolalpha;
    std::stacktrace bktr;
    std::cout << "起初, bktr.empty(): " << bktr.empty() << '\n';
 
    bktr = std::stacktrace::current();
    std::cout << "获取条目后, bktr.empty(): " << bktr.empty() << '\n';
}

可能的输出：

起初, bktr.empty(): true
获取条目后, bktr.empty(): false

参阅

size
 返回栈踪迹条目的数量
(公开成员函数)
