围绕成员函数指针的包装器。将要调用其成员函数的类实例作为指针传递给 operator()。

1) 包装无形参的非 const 成员函数。
2) 包装无形参的 const 成员函数。
3) 包装有单形参的非 const 成员函数。
4) 包装有单形参的 const 成员函数。
