std::queue<T,Container>::push

< cpp‎ | container‎ | queue

void push( const value_type& value );
 (1)
void push( value_type&& value );
 (2) (C++11 起)

将给定的元素 value 推到 queue 尾。

1) 等价于：c.push_back(value)。
2) 等价于：c.push_back(std::move(value))。

参数

value - 要推入的元素值

返回值

（无）

复杂度

等于 Container::push_back 的复杂度。
