std::experimental::make_optional

< cpp‎ | experimental‎ | optional
在标头 <experimental/optional> 定义
template< class T >

constexpr optional<typename std::decay<T>::type>

    make_optional( T&& value );
 (库基础 TS)

从 value 创建 optional 对象。相当于调用 optional<typename std::decay<T>::type>(std::forward<T>(value))。

参数

value - 用以构造 optional 对象的值

返回值

以 value 为所含值的 optional 对象。

异常

由 T 的构造函数抛出的任何异常。

参阅

(构造函数)
 构造 optional 对象
(公开成员函数)
