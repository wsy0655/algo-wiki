std::ranges::zip_transform_view<F,Views...>::zip_transform_view

< cpp‎ | ranges‎ | zip transform view
zip_transform_view() = default;
 (1) (C++23 起)
constexpr zip_transform_view( F fun, Views... views );
 (2) (C++23 起)
1) 默认构造函数。值初始化存储的可调用对象与所有适配的 view 对象。

以下情况下默认构造函数被弃置：

  • F 不满足 default_initializable，或
  • std::is_default_constructible_v 对 Views... 中至少一个类型为 false。
2) 从 fun 移动构造存储的可调用对象，并从 views... 中对应的视图移动构造每个适配的 view 对象。

参数

f - 用于生成 zip_transform_view 的元素的可调用对象
views - 要适配的视图对象
