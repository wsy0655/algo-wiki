std::initializer_list<T>::end

< cpp‎ | utility‎ | initializer list
const T* end() const noexcept;
 (C++11 起)
(C++14 起为 constexpr)

获得指向初始化器列表末元素后一位置的指针，即 begin() + size()。

若初始化器列表为空，则 begin() 与 end() 的值是未指定的，但一定相同。

参数

（无）

返回值

指向初始化器列表末元素后一位置的指针

复杂度

常数

示例

#include <initializer_list>
#include <numeric>
 
int main()
{
    static constexpr auto l = {3, 13, 13};
    static_assert(std::accumulate(l.begin(), l.end(), 13) == 42);
}
