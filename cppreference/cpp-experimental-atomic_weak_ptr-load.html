std::experimental::atomic_weak_ptr<T>::load

< cpp‎ | experimental‎ | atomic weak ptr
weak_ptr<T> load( std::memory_order order = std::memory_order_seq_cst ) const noexcept;

原子地加载并返回 atomic_weak_ptr 的当前值。根据 order 的值影响内存。

order 必须为 std::memory_order_relaxed、std::memory_order_consume、std::memory_order_acquire 或 std::memory_order_seq_cst 之一。否则其行为未定义。

参数

order - 强制的内存定序约束

说明

所有关联的 use_count 自增均保证作为原子操作的一部分实施。

返回值

atomic_weak_ptr 的当前值。
