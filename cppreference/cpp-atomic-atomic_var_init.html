ATOMIC_VAR_INIT

< cpp‎ | atomic


在标头 <atomic> 定义
#define ATOMIC_VAR_INIT(value) /* 由实现定义 */
 (C++11 起)
(C++20 弃用)

展开成能用于初始化可从 value 初始化的 std::atomic 对象的表达式。若原子对象拥有静态存储期，则此初始化为常量初始化。

注解

在另一线程在初始化期间访问，即使通过原子操作，也是数据竞争（若地址立即传递给带 std::memory_order_relaxed 操作的另一线程则可以发生）。

此宏主要为与 C 的兼容性提供；它表现与 std::atomic 的构造函数相同。

示例

#include <atomic>
#include <iostream>
 
int main()
{
    std::atomic<int> a = ATOMIC_VAR_INIT(1);
    // std::atomic<int> a(1);   // 仅 C++ 的替用品
    std::cout << "Initialized std::atomic<int> as: " << a << '\n';
}

输出：

Initialized std::atomic<int> as: 1
