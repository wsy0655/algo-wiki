std::queue<T,Container>::emplace

< cpp‎ | container‎ | queue

template< class... Args >
void emplace( Args&&... args );
 (C++11 起)
(C++17 前)
template< class... Args >
decltype(auto) emplace( Args&&... args );
 (C++17 起)

推入新元素到队列结尾。原位构造元素，即不进行移动或复制操作。以与提供给函数严格相同的实参调用元素的构造函数。

相当于调用 c.emplace_back(std::forward<Args>(args)...); 。

参数

args - 转发给元素构造函数的实参

返回值

（无）

 (C++17 前)

上述对 Container::emplace_back 的调用返回的值或引用，若它存在。

 (C++17 起)

复杂度

与 Container::emplace_back 的复杂度相同。

示例

#include <iostream>
#include <queue>
 
struct S
{
    int id;
 
    S(int i, double d, std::string s) : id{i}
    {
        std::cout << "S::S(" << i << ", " << d << ", \"" << s << "\");\n";
    }
};
 
int main()
{
    std::queue<S> queue;
    const S& s = queue.emplace(42, 3.14, "C++"); // 对于返回值 C++17 有此要求
    std::cout << "id = " << s.id << '\n';
}

输出：

S::S(42, 3.14, "C++")
id = 42
