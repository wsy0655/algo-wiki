对星期之日值加或减 1，将结果对 7 取余，规约成范围 [​0​, 6] 中的整数。

1,2) 实施 *this += std::chrono::days{1};。
3,4) 实施 *this -= std::chrono::days{1};。

参数

（无）

返回值

1,3) 到修改后的此 weekday 的引用。
2,4) 修改前所作的 weekday 的副本。

注意

调用这些函数之一后，ok()() 始终为 true。

示例

#include <cassert>
#include <chrono>
#include <iostream>
 
int main()
{
    std::cout << std::boolalpha;
 
    std::chrono::weekday wd{0}; // 星期日为 0 或 7
 
    --wd;
    std::cout << (wd == std::chrono::Saturday) << ' ';
 
    ++wd;
    std::cout << (wd == std::chrono::Sunday) << '\n';
 
    wd = std::chrono::weekday{13};
    assert(!wd.ok());
    wd++;
    assert(wd.ok());
}

输出：
