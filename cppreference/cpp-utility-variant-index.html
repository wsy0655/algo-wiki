std::variant<Types...>::index

< cpp‎ | utility‎ | variant
constexpr std::size_t index() const noexcept;
 (C++17 起)

返回变体当前所保有的可选项的零基索引。

若变体为 valueless_by_exception（因异常无值），则返回 variant_npos。

示例

#include <iostream>
#include <string>
#include <variant>
 
int main()
{
    std::variant<int, std::string> v = "abc";
    std::cout << "v.index = " << v.index() << '\n';
    v = {}; 
    std::cout << "v.index = " << v.index() << '\n';
}

输出：

v.index = 1
v.index = 0
