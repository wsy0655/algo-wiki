构造新的分布对象。

2) 以 k 和 p 为分布参数。 @32 以 params 为分布参数。

参数

k - k 分布参数（试验成功次数）
p - p 分布参数（试验生成 true 的概率）
params - 分布参数集

注解

要求 0 < p ≤ 1 且 0 < k。

若 p == 1，则调用不接受 param_type 对象的 operator() 的重载将导致未定义行为。

默认构造的 std::negative_binomial_distribution 等价于默认构造的 std::geometric_distribution。
