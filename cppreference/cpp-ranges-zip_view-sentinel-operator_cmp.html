 (C++23 起)

比较 x 的底层迭代器元组和 y 的底层哨位元组。

此函数对常规的无限定或有限定查找不可见，而只能在 zip_view::sentinel<Const> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

x - 要比较的迭代器
y - 要比较的哨位

返回值

令 x.current_ 代表底层迭代器元组，并令 y.end_ 代表底层哨位元组。

返回

  • true，如果从等价于 std::get<i>(x.current_) 的表达式所获取的至少一个底层迭代器，（使用适当的 operator==）求值等于从等价于 std::get<i>(y.end_) 的表达式所获取的底层哨位，其中索引 i 属于范围 0 <= i < sizeof...(Views)，
  • 否则为 false。
