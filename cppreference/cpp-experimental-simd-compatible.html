std::experimental::simd_abi::compatible

< cpp‎ | experimental‎ | simd
在标头 <experimental/simd> 定义
template < class T >
using compatible = /* 由实现定义 */;
 (并行 TS v2)

compatible<T> 是 ABI 标签的实现定义别名。

注解

意图是用 ABI 标签产生对于元素类型 T 最高效的数据并行执行，并确保目标架构上的翻译单元间的 ABI 兼容性。
