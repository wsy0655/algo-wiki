BidirIt prev( BidirIt it, typename std::iterator_traits<BidirIt>::difference_type n = 1 );
 (C++17 起)

返回迭代器 it 的第 n 个前驱（或当 n 是负数时为第 n 个后继）。

参数

it - 迭代器
n - it 要被减少的元素个数
类型要求
-
BidirIt 必须满足老式双向迭代器 (LegacyBidirectionalIterator) 。

返回值

BidirIt 类型的迭代器，持有迭代器 it 的第 n 个前驱（或当 n 是负数时为第 n 个后继）。

复杂度

线性。

然而，若 BidirIt 还满足老式随机访问迭代器 (LegacyRandomAccessIterator) ，则复杂度为常数。

可能的实现

template<class BidirIt>
constexpr // since C++17
BidirIt prev(BidirIt it, typename std::iterator_traits<BidirIt>::difference_type n = 1)
{
    std::advance(it, -n);
    return it;
}

注解

尽管表达式 --c.end() 通常能编译，然而不保证会这么做：c.end() 是右值表达式，并无指定了“保证可进行右值的自减”的迭代器要求。尤其是当迭代器实现为指针或其 operator-- 带有左值引用限定时，--c.end() 不能编译，而 std::prev(c.end()) 可以。

示例

#include <iostream>
#include <iterator>
#include <vector>
 
int main()
{
    std::vector<int> v{3, 1, 4};
 
    auto it = v.end();
    auto pv = std::prev(it, 2);
    std::cout << *pv << '\n';
 
    it = v.begin();
    pv = std::prev(it, -2);
    std::cout << *pv << '\n';
}

输出：

1
4
