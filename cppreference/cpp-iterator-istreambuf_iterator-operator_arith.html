std::istreambuf_iterator<CharT,Traits>::operator++, operator++(int)

< cpp‎ | iterator‎ | istreambuf iterator
istreambuf_iterator& operator++();
 (1)
/* proxy */ operator++( int );
 (2)

通过调用 sbuf_->sbumpc() 推进迭代器，其中 sbuf_ 是存储的指向流缓冲的指针。

若迭代器是流尾迭代器则行为未定义。

参数

（无）

返回值

1) *this
2) 保有经由 operator*() 获得的当前字符和 sbuf_ 指针的 proxy 对象。以 operator* 解引用 proxy 对象将产生存储的字符。
名称 proxy 仅用于阐释。

异常

可能会抛出由实现定义的异常。
