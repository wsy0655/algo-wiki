std::experimental::filesystem::path::operator=

< cpp‎ | experimental‎ | fs‎ | path
path& operator=( const path& p );
 (1) (文件系统 TS)
path& operator=( path&& p );
 (2) (文件系统 TS)
template< class Source >
path& operator=( const Source& source );
 (3) (文件系统 TS)
1) 替换 *this 的内容为 p 的内容的副本。
2) 替换 *this 的内容为 p，可能使用移动语义：p 遗留为有效单未指明的状态。
3) 替换 *this 的内容为如同以 path 构造函数的重载 (4) 从 source 构造的新路径。等价于 assign(source)。

参数

p - 要赋值的路径
source - std::basic_string，指向空终止字符/宽字符串的指针，或指向空终止字符/宽字符序列的输入迭代器。字符类型必须为 char、char16_t、char32_t、wchar_t 之一

返回值

*this

异常

1) （无）
2)
noexcept 规定：  
noexcept
  
3) （无）

示例

#include <experimental/filesystem>
namespace fs = std::experimental::filesystem;
 
int main()
{
    fs::path p = "C:/users/abcdef/AppData/Local";
    p = p / "Temp"; // 移动赋值
    const wchar_t* wstr = L"D:/猫.txt";
    p = wstr; // 从一个源赋值
}
