 (3) (C++11 起)
template< class T >

T atomic_fetch_xor_explicit( volatile std::atomic<T>* obj,
                             typename std::atomic<T>::value_type arg,

                             std::memory_order order) noexcept;
 (4) (C++11 起)

以 obj 的旧值和 arg 的逐位异或结果原子地替换 obj 指向的值。返回 obj 先前保有的值。

如同执行下列内容一般进行运算：

1,2) obj->fetch_xor(arg)
3,4) obj->fetch_xor(arg, order)

如果 std::atomic<T> 没有 fetch_xor 成员（此成员仅对 bool 以外的整数类型提供），那么程序非良构。

参数

obj - 指向要修改的原子对象的指针
arg - 要和存储到原子对象的值逐位异或的值
order - 内存同步定序

返回值

*obj 的修改顺序中，紧接此函数生效之前的值。
