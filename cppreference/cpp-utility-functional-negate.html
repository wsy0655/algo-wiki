std::negate

< cpp‎ | utility‎ | functional
在标头 <functional> 定义
template< class T >
struct negate;
 (C++14 前)
template< class T = void >
struct negate;
 (C++14 起)

进行取负的函数对象。相当于在 T 类型实例上调用 operator-。

特化

标准库提供 std::negate 在不指定 T 时的特化，使形参类型和返回类型留待推导。

negate<void>
(C++14)
 实现 -x 的函数对象，推导形参类型和返回类型
(类模板特化)
(C++14 起)

成员类型

类型 定义
result_type (C++17 弃用)(C++20 移除) T
argument_type (C++17 弃用)(C++20 移除) T

这些成员类型是通过公开继承 std::unary_function<T, T> 获得的。

 (C++11 前)

成员函数

operator()
 返回实参的相反数
(公开成员函数)

std::negate::operator()

T operator()( const T& arg ) const;
 (C++14 起为 constexpr)

返回 arg 的相反数。

参数

arg - 要计算相反数的值

返回值

-arg 的结果。

异常

可能会抛出由实现定义的异常。

可能的实现

constexpr T operator()(const T& arg) const 
{
    return -arg;
}
