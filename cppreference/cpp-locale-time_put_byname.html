std::time_put_byname

< cpp‎ | locale
在标头 <locale> 定义
template<

    class CharT,
    class OutputIt = std::ostreambuf_iterator<CharT>

> class time_put_byname : public std::time_put<CharT, OutputIt>

std::time_put_byname 是 std::time_put 刻面，封装在它构造时指定的本地环境的日期和时间格式化规则。

特化

标准库保证提供满足以下类型要求的所有特化：

  • CharT 是 char 或 wchar_t
  • OutputIt 需要满足老式输出迭代器 (LegacyOutputIterator)

 (C++11 起)

为名为 name 的本地环境构造新的 std::time_put_byname 刻面。

refs 用于资源管理：在销毁最后一个保有刻面的 std::locale 对象时，如果 refs == 0，那么实现会销毁刻面对象。否则不销毁对象。

示例

此示例使用被各种 std::time_put_byname 平面替换了 time_put 刻面的 "C" 本地环境打印当前时间：

#include <codecvt>
#include <ctime>
#include <iomanip>
#include <iostream>
 
int main()
{
    std::time_t t = std::time(nullptr);
    std::wbuffer_convert<std::codecvt_utf8<wchar_t>> conv(std::cout.rdbuf());
    std::wostream out(&conv);
 
    out.imbue(std::locale(out.getloc(),
                          new std::time_put_byname<wchar_t>("ja_JP.utf8")));
    out << std::put_time(std::localtime(&t), L"%A %c") << '\n';
 
    out.imbue(std::locale(out.getloc(),
                          new std::time_put_byname<wchar_t>("ru_RU.utf8")));
    out << std::put_time(std::localtime(&t), L"%A %c") << '\n'; 
 
    out.imbue(std::locale(out.getloc(),
                          new std::time_put_byname<wchar_t>("sv_SE.utf8")));
    out << std::put_time(std::localtime(&t), L"%A %c") << '\n'; 
}

可能的输出：

木曜日 2023年10月05日 19時44分51秒
Четверг Чт 05 окт 2023 19:44:51
torsdag tor  5 okt 2023 19:44:51
