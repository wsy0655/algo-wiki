std::copyable

< cpp‎ | concepts
在标头 <concepts> 定义
template< class T >

concept copyable =
    std::copy_constructible<T> &&
    std::movable<T> &&
    std::assignable_from<T&, T&> &&
    std::assignable_from<T&, const T&> &&

    std::assignable_from<T&, const T>;
 (C++20 起)

概念 copyable<T> 指定 T 是亦能复制的 movable 对象类型（即它支持复制构造和复制赋值）。
