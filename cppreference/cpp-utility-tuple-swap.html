std::tuple<Types...>::swap

< cpp‎ | utility‎ | tuple
在标头 <tuple> 定义
void swap( tuple& other ) noexcept(/* 见下文 */);
 (1) (C++11 起)
(C++20 起为 constexpr)
constexpr void swap( const tuple& other ) noexcept(/* 见下文 */) const;
 (2) (C++23 起)

对 *this 中的每个元素与 other 中的对应元素调用 swap（可以是 std::swap，或通过 ADL 找到的）。

若任何选择的 swap 函数调用非良构，或不交换两个 tuple 的元素，则行为未定义。

 (C++23 前)

若任何选择的 swap 函数调用不交换两个 tuple 的元素，则行为未定义。

1) 若 (std::is_swappable_v<Types> && ...) 非 true 则程序非良构。
2) 若 (std::is_swappable_v<const Types> && ...) 非 true 则程序非良构。
 (C++23 起)

参数

other - 要交换值的元组

返回值

（无）

异常

noexcept 说明：  
noexcept(

    noexcept(swap(std::declval<T0&>>(), std::declval<T0&>())) &&
    noexcept(swap(std::declval<T1&>>(), std::declval<T1&>())) &&
    noexcept(swap(std::declval<T2&>>(), std::declval<T2&>())) &&
    ...

)

上述表达式中，以 C++17 std::is_nothrow_swappable 特征所用的相同方式查找标识符 swap。

 (C++17 前)
1)
noexcept 说明：  
noexcept((std::is_nothrow_swappable_v<Types> && ...))
2)
noexcept 说明：  
noexcept((std::is_nothrow_swappable_v<const Types> && ...))
 (C++17 起)

示例

#include <iostream>
#include <string>
#include <tuple>
 
int main()
{
    std::tuple<int, std::string, float>
                      p1{42, "ABCD", 2.71},
                      p2;
    p2 = std::make_tuple(10, "1234", 3.14);
 
    auto print_p1_p2 = [&](auto rem) {
        std::cout << rem
                  << "p1 = {" << std::get<0>(p1)
                  << ", "     << std::get<1>(p1)
                  << ", "     << std::get<2>(p1) << "}, "
                  << "p2 = {" << std::get<0>(p2)
                  << ", "     << std::get<1>(p2)
                  << ", "     << std::get<2>(p2) << "}\n";
    };
 
    print_p1_p2("Before p1.swap(p2): ");
    p1.swap(p2);
    print_p1_p2("After  p1.swap(p2): ");
    swap(p1, p2);
    print_p1_p2("After swap(p1, p2): ");
}

输出：

Before p1.swap(p2): p1 = {42, ABCD, 2.71}, p2 = {10, 1234, 3.14}
After  p1.swap(p2): p1 = {10, 1234, 3.14}, p2 = {42, ABCD, 2.71}
After swap(p1, p2): p1 = {42, ABCD, 2.71}, p2 = {10, 1234, 3.14}
