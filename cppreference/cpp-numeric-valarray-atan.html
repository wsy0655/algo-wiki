std::atan(std::valarray)

< cpp‎ | numeric‎ | valarray
在标头 <valarray> 定义
template< class T >
valarray<T> atan( const valarray<T>& va );

对于 va 中的每个元素，计算元素值的弧（反）正切。

参数

va - 要应用操作到的值数组

返回值

含有 va 中各值的反正切的值数组。

注解

用无限定函数 (atan) 进行计算。若该函数不可用，则会由于实参依赖查找而使用 std::atan。

函数可以实现为拥有不同于 std::valarray 的返回类型。此时替换它的类型拥有下列属性：

  • 提供 std::valarray 的所有 const 成员函数。
  • 能从替换类型构造 std::valarray、std::slice_array、std::gslice_array、std::mask_array 和 std::indirect_array。
  • 所有接受一个 const std::valarray& 类型参数的函数，除了 begin() 和 end()(C++11 起)也应该接受替换类型。
  • 所有接受两个 const std::valarray& 类型参数的函数都应该接受 const std::valarray& 和替换类型的每种组合。
  • 返回类型添加不多于两层嵌套在最深层嵌套的参数类型上的模板。

可能的实现

template<class T>
valarray<T> atan(const valarray<T>& va)
{
    valarray<T> other = va;
    for (T& i : other)
        i = atan(i);
 
    return other; // 可以返回代理对象
}

示例

#include <algorithm>
#include <cmath>
#include <iostream>
#include <valarray>
 
auto show = [](char const* title, const std::valarray<float>& va)
{
    std::cout << title << " :";
    std::for_each(std::begin(va), std::end(va), 
        [](const float x) { std::cout << "  " << std::fixed << x; });
    std::cout << '\n';
};
 
int main()
{
    const std::valarray<float> x = {.1f, .3f, .6f, .9f};
    const std::valarray<float> f = std::atan(x);
    const std::valarray<float> g = std::tan(f);
 
    show("x          ", x);
    show("f = atan(x)", f);
    show("g = tan(f) ", g);
}

输出：

x           :  0.100000  0.300000  0.600000  0.900000
f = atan(x) :  0.099669  0.291457  0.540420  0.732815
g = tan(f)  :  0.100000  0.300000  0.600000  0.900000
