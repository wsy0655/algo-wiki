C++ 关键词：and

< cpp‎ | keyword


用法

  • 代用运算符：用作 && 的替代写法

示例

int main()
{
    static_assert((false and false) == false);
    static_assert((false and true)  == false);
    static_assert((true  and false) == false);
    static_assert((true  and true)  == true);
}
