返回通用路径格式的内部路径名，转换为指定的字符串类型。如果进行转换，则按 todo 进行。以 / 字符为目录分隔符。

1) 所有内存分配均由 a 进行。
2) u8string() 的情况中编码总是 UTF-8。

参数

a - 用以构造字符串的分配器
-
CharT 必须为编码字符类型（char、wchar_t、char16_t 和 char32_t）之一

返回值

通用路径格式的内部路径名，转换为指定的字符串类型。

异常

可能会抛出由实现定义的异常。
