std::list<T,Allocator>::size

< cpp‎ | container‎ | list
size_type size() const;
 (C++11 前)
size_type size() const noexcept;
 (C++11 起)

返回容器中的元素数，即 std::distance(begin(), end())。

参数

（无）

返回值

容器中的元素数量。

复杂度

常数或线性。 (C++11 前)
常数。 (C++11 起)

示例

以下代码用 size 来显示 std::list 中的元素数量：

#include <iostream>
#include <list>
 
int main()
{ 
    std::list<int> nums{1, 3, 5, 7};
 
    std::cout << "nums 中有 " << nums.size() << " 个元素。\n";
}

输出：

nums 中有 4 个元素。
