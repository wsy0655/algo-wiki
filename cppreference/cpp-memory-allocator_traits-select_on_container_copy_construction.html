std::allocator_traits<Alloc>::select_on_container_copy_construction

< cpp‎ | memory‎ | allocator traits
在标头 <memory> 定义
static Alloc select_on_container_copy_construction( const Alloc& a );
 (C++11 起)
(C++20 起为 constexpr)

若可能，则通过调用 a.select_on_container_copy_construction() 获得分配器 a 的复制构造的版本。若上述不可行（例如 Alloc 无成员函数 select_on_container_copy_construction()），则返回未修改的 a。

此函数为所有标准库容器的复制构造函数所调用。它允许构造函数参数所用的分配器为正在复制的容器所具备，并在需要的情况下修改状态。

参数

a - 标准容器用以传递给容器复制构造函数的分配器

返回值

复制构造的标准容器所用的分配器。

参阅

select_on_container_copy_construction
 复制 scoped_allocator_adaptor 与其所有分配器的状态
(std::scoped_allocator_adaptor<OuterAlloc,InnerAlloc...> 的公开成员函数)
