std::thread::id

< cpp‎ | thread‎ | thread
在标头 <thread> 定义
class thread::id;
 (C++11 起)

类 thread::id 是轻量的可平凡复制类，它的作用是 std::thread 及 std::jthread (C++20 起)对象的唯一标识符。

此类的实例也可以持有不表示任何线程的特殊值。一旦线程结束，那么 std::thread::id 的值可能被另一线程复用。

此类为用作包括有序和无序的关联容器的键而设计。
