std::layout_stride::mapping<Extents>::required_span_size

< cpp‎ | container‎ | mdspan‎ | layout stride‎ | mapping
constexpr index_type required_span_size() const noexcept;
 (C++23 起)

返回映射所需大小。

令 s：

  • 等于 1，如果 extents().rank() == 0 为 true，
  • 否则等于 ​0​，如果多维索引空间 extents() 的大小为 ​0​，
  • 否则等于 1 加上对于范围 [​0​, extents().rank()) 中的所有 r 的 extents().extent(r) - 1 与 extents_type::index-cast(strides_[r]) 的乘积的和。

此调用等价于 return s;。

返回值

映射所需大小。
