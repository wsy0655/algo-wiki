std::experimental::ranges::Copyable

< cpp‎ | experimental‎ | ranges
在标头 <experimental/ranges/concepts> 定义
template< class T >

concept bool Copyable =
    CopyConstructible<T> &&
    Movable<T> &&

    Assignable<T&, const T&>;
 (范围 TS)

概念 Copyable<T> 指定 T 为 Movable 对象类型，并且亦能复制（即它支持复制构造和复制赋值）。

注解

Copyable<T> 亦有意要求 ranges::Assignable<T&, const T>（从 const 右值赋值）和 ranges::Assignable<T&, T&>（从非 const 左值赋值）得到满足。
