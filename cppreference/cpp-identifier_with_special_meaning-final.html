有特殊含义的 C++ 标识符: final (C++11 起)

< cpp


用法

  • final 说明符

示例

struct b0 final {};
struct d0: b0 {}; // 错误：不能从 final 基类派生
 
struct b1
{
    virtual void f0() final;
    virtual void f1();
};
 
struct d1: b1 // OK
{
    void f0(); // 错误：不能覆盖 final 函数
    void f1(); // OK
};

参阅

  • class, struct, union
  • virtual, override
  • replaceable_if_eligible (C++26 起), trivially_relocatable_if_eligible (C++26 起)
