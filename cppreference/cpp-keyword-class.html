C++ 关键词：class

< cpp‎ | keyword


用法

  • 声明类
  • 声明有作用域枚举类型
 (C++11 起)
  • 在模板声明中，class 可用于引入类型模板形参与模板模板形参
  • 若作用域中存在具有与某个类类型的名字相同的名字的函数或变量，则 class 可附于类名之前以消歧义，这产生一个详述类型说明符

示例

class Foo;  // 类的前置声明
 
class Bar   // 类的定义
{
public:
    Bar(int i) : m_i(i) {}
private:
    int m_i;
};
 
template <class T> // 模板实参
void qux()
{
    T t;
}
 
enum class Pub // 有作用域枚举，C++11 起
{
    b, d, p, q
};
 
int main()
{
    Bar Bar(1); // 变量 Bar 隐藏了类型 Bar
    Bar Bar2(2); // 编译器报错
    class Bar Bar2(2); // 详述的类型
}
