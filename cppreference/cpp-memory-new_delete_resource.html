std::pmr::new_delete_resource

< cpp‎ | memory
在标头 <memory_resource> 定义
std::pmr::memory_resource* new_delete_resource() noexcept;
 (C++17 起)

返回指向使用全局 operator new 与 operator delete 分配内存的 memory_resource 的指针。

返回值

返回指针 p，指向派生自 std::pmr::memory_resource 的类型的静态存储期对象，它拥有下列性质：

  • 其 allocate() 函数用 ::operator new 分配内存；
  • 其 deallocate() 函数用 ::operator delete 解分配内存；
  • 对于任何 memory_resource r，p->is_equal(r) 返回 &r == p。

每次调用此函数时返回相同值。
