std::ranges::slide_view<V>::begin

< cpp‎ | ranges‎ | slide view
constexpr auto begin()
    requires (!(/*simple-view*/<V> && /*slide-caches-nothing*/<const V>));
 (1) (C++23 起)
constexpr auto begin() const
    requires /*slide-caches-nothing*/<const V>;
 (2) (C++23 起)

返回指向 slide_view 首个元素的迭代器。

1) 如果 V 实现 slide-caches-first，等价于

return iterator<false>(ranges::begin(base_),
                       ranges::next(ranges::begin(base_), n_ - 1, ranges::end(base_)),
                       n_);

否则，等价于 return iterator<false>(ranges::begin(base_), n_);。
如果 V 实现 slide-caches-first，则该函数将结果缓存在 cached_begin_ 中以供后续调用。这对于提供 range 所需的均摊常数时间复杂度是必要的。
2) 等价于 return iterator<true>(ranges::begin(base_), n_);。

参数

(无)

返回值

指向 slide_view 首个元素的迭代器，根据有无 const 限定，它指向底层视图类型的大小为 n_ 的子范围，底层视图类型对于重载 (1) 为 V，对于重载 (2) 为 const V。

示例

#include <iostream>
#include <ranges>
#include <string_view>
using namespace std::literals;
 
int main()
{
    static constexpr auto source = {"∀x"sv, "∃y"sv, "ε"sv, "δ"sv};
    auto view{std::ranges::slide_view(source, 2)};
    const auto subrange{*(view.begin())};
    for (std::string_view const s : subrange)
        std::cout << s << ' ';
    std::cout << '\n';
}

输出：

∀x ∃y
