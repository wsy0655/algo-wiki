bitset operator~() const;
 (4) (C++11 起为 noexcept)

进行二进制与（AND）、或（OR）、异或（XOR）及非（NOT）运算。

1) 把各个位设置为 *this 与 other 的对应位对偶上的二进制与的结果。
2) 把各个位设置为 *this 与 other 的对应位对偶上的二进制或的结果。
3) 把各个位设置为 *this 与 other 的对应位对偶上的二进制异或的结果。。
4) 返回 *this 的反转所有位的临时副本（二进制非）。

注意 &=、|= 和 ^= 仅对拥有相同大小 N 的 bitset 有定义。

参数

other - 另一 bitset

返回值

1-3) *this
4) std::bitset<N>(*this).flip()

示例

#include <bitset>
#include <cstddef>
#include <iostream>
#include <string>
 
int main()
{
    const std::string pattern_str{"1001"};
    std::bitset<16> pattern{pattern_str}, dest;
 
    for (std::size_t i = dest.size() / pattern_str.size(); i != 0; --i)
    {
        dest <<= pattern_str.size();
        dest |= pattern;
        std::cout << dest << " (i = " << i << ")\n";
    }
 
    std::cout << ~dest << " (~dest)\n";
}

输出：

0000000000001001 (i = 4)
0000000010011001 (i = 3)
0000100110011001 (i = 2)
1001100110011001 (i = 1)
0110011001100110 (~dest)
