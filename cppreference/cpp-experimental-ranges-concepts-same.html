std::experimental::ranges::Same

< cpp‎ | experimental‎ | ranges
在标头 <experimental/ranges/concepts> 定义
template< class T, class U >
concept bool Same = std::is_same<T, U>::value; // 见下文
 (范围 TS)

概念 Same<T, U> 当且仅当 T 与 U 代表同一类型时才得到满足。

无关乎说明使用，Same<T, U> 与 std::is_same<T, U>::value 间不必有蕴含关系。

为制约检查的目的，Same<T, U> 隐含 Same<U, T>。

注解

Same 的制约检查上的额外要求与 std::is_same 不同。
