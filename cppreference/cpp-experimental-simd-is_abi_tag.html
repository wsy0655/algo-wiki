std::experimental::is_abi_tag

< cpp‎ | experimental‎ | simd
在标头 <experimental/simd> 定义
template< class T >
struct is_abi_tag;
 (并行 TS v2)

若 T 是 simd_abi 命名空间中的 ABI 标签类型，则提供等于 true 的成员常量 value。对于任何其他类型，value 为 false。

模板形参

T - 要检查的类型

辅助变量模板

template< class T >
constexpr bool is_abi_tag_v = is_abi_tag<T>::value;
 (并行 TS v2)

继承自 std::integral_constant

成员常量

value
[静态]
 如果 T 为 ABI 标签类型那么是 true，否则是 false
(公开静态成员常量)

成员函数

operator bool
 将对象转换到 bool，返回 value
(公开成员函数)
operator()
(C++14)
 返回 value
(公开成员函数)

成员类型

类型 定义
value_type bool
type std::integral_constant<bool, value>

示例

#include <experimental/simd>
 
static_assert(std::experimental::is_abi_tag_v<stdx::simd_abi::scalar>);
static_assert(!std::experimental::is_abi_tag_v<int>);
 
int main() {}
