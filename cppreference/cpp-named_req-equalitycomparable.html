C++ 具名要求：可相等比较 (EqualityComparable)

< cpp‎ | named req


类型必须能使用 == 运算符且结果应当具有标准语义。

要求

对于类型 T，给定 T 或(C++11 起) const T 类型的表达式 a、b 和 c，如果以下表达式合法且具有指定的效果，那么 T 满足可相等比较 (EqualityComparable) ：

表达式 类型 要求
a == b 符合可布尔测试 (BooleanTestable)
(C++20 前)		 建立一种等价关系，即满足以下所有性质：
  • 对于 a 的所有值，a == a 都会产生 true。
  • 如果 a == b，那么 b == a。
  • 如果 a == b 且 b == c，那么 a == c。

注解

为满足此要求，没有内建比较运算符的类型必须提供用户定义的 operator==。

对于既可相等比较 (EqualityComparable) 也可小于比较 (LessThanComparable) 的类型，C++ 标准库区分：

  • 相等（即表达式 a == b 的值）和
  • 等价（即表达式 !(a < b) && !(b < a) 的值）
