(C++20 前)

检查 *this 与 other 是否等价。

以下情况下，两个正则表达式记号迭代器相等：

a) 它们都是序列尾迭代器。
b) 它们都是后缀迭代器且后缀相等。
c) 它们都不是序列尾或后缀迭代器且：
  • position == other.position
  • N == other.N
  • subs == other.subs
1) 检查 *this 是否等于 other。
2) 检查 *this 是否不等于 other。

!= 运算符从 operator== 运算符合成。

返回值

2) 若 *this 不等于 other 则为 true，否则为 false。
