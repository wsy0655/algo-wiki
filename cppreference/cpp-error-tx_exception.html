std::tx_exception

< cpp‎ | error
在标头 <stdexcept> 定义
template< class T >
class tx_exception : public std::runtime_error;
 (TM TS)

定义能用于取消并回滚关键词 atomic_cancel 所初始化的原子事务的异常类型。

若 T 非可平凡复制 (TriviallyCopyable) ，则特化 std::tx_exception<T> 的程序非良构。
