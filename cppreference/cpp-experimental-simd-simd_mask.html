std::experimental::simd_mask

< cpp‎ | experimental‎ | simd
在标头 <experimental/simd> 定义
template< class T, class Abi = simd_abi::compatible<T> >
class simd_mask;
 (并行 TS v2)

类模板 simd_mask 是元素类型为 bool 的数据并行类型。给定的 simd_mask 实例化的宽度是常量表达式，由模板形参确定。具体来说，simd_mask<T, Abi>::size() 始终为 simd<T, Abi>::size()。

模板形参

T - 应用 simd_mask 的元素类型
Abi - 应用 simd_mask 的 ABI 类型

辅助别名模版

模板 定义
template< class T, int N > using fixed_size_simd_mask simd_mask<T, simd_abi::fixed_size<N>>
template< class T > using native_simd_mask simd_mask<T, simd_abi::native<T>>
