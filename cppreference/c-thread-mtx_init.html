mtx_init

< c‎ | thread
在标头 <threads.h> 定义
int mtx_init( mtx_t* mutex, int type );
 (C11 起)

创建新的拥有 type 类型的互斥体。设置 mutex 所指向的对象为新创建互斥体的标识符。

type 必须拥有下列值之一：

  • mtx_plain - 创建简单的，非递归互斥体。
  • mtx_timed - 创建非递归的，支持超时的互斥体。
  • mtx_plain | mtx_recursive - 创建递归互斥体。
  • mtx_timed | mtx_recursive - 创建递归的支持超时的互斥体。

参数

mutex - 指向要初始化的互斥体的指针
type - 互斥的类型

返回值

若成功则为 thrd_success，否则为 thrd_error。
