std::chrono::year_month_day

< cpp‎ | chrono
在标头 <chrono> 定义
class year_month_day;
 (C++20 起)

类 year_month_day 表示为特定的年月日。它是基于域的时间点，其分辨率为 std::chrono::days。它直接支持面向 std::chrono::years 与 std::chrono::months 的算术。与 std::chrono::sys_days 间的双向隐式转换允许进行面向 std::chrono::days 的有效算术运算。

year_month_day 是可平凡复制 (TriviallyCopyable) 的标准布局类型 (StandardLayoutType) 。

Example

#include <chrono>
#include <iostream>
 
int main()
{
    const std::chrono::time_point now{std::chrono::system_clock::now()};
 
    const std::chrono::year_month_day ymd{std::chrono::floor<std::chrono::days>(now)};
 
    std::cout << "当前年：" << static_cast<int>(ymd.year()) << ", "
              << "月：" << static_cast<unsigned>(ymd.month()) << ", "
              << "日：" << static_cast<unsigned>(ymd.day()) << "\n"
                 "ymd: " << ymd << '\n';
}

可能的输出：

当前年：2023, 月：6, 日：29
ymd: 2023-09-04
