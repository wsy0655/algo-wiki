空终止字节字符串

< cpp‎ | string


空终止字节字符串（null-terminated byte string, NTBS）是后随拥有零值的字节（空终止字符）的非零字节序列。字节字符串中的每个字节都是一些字符集的编码。例如，字符数组 {'\x63', '\x61', '\x74', '\0'} 是持有以 ASCII 编码的字符串 "cat" 的 NTBS。
