std::deque<T,Allocator>::empty

< cpp‎ | container‎ | deque

bool empty() const;
 (C++11 起为 noexcept)

检查容器是否无元素，即是否 begin() == end()。

返回值

若容器为空则为 true，否则为 false。

复杂度

常数。

示例

下列代码用 empty 检查 std::deque<int> 是否含有任何元素：

#include <deque>
#include <iostream>
 
int main()
{
    std::deque<int> numbers;
    std::cout << std::boolalpha;
    std::cout << "起初, numbers.empty(): " << numbers.empty() << '\n';
 
    numbers.push_back(42);
    numbers.push_back(13317); 
    std::cout << "添加元素后, numbers.empty(): " << numbers.empty() << '\n';
}

输出：

起初, numbers.empty(): true
添加元素后, numbers.empty(): false
