令 current_ 代表指向所适配视图的元素的迭代器的底层元组式 ﻿对象。

1,2) 返回增加 n 后的迭代器 i。等价于： auto r = i; r += n; return r;。
3) 返回减小 n 后的迭代器 i。等价于： auto r = i; r -= n; return r;。
4) 令 DIST(k) 为 difference_type(std::get<k>(i.current_) - std::get<k>(j.current_))。返回 0 <= n < sizeof...(Views) 中所有整数中 DIST(n) 的最小绝对值。

这些函数对常规的无限定或有限定查找不可见，而只能在 zip_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

参数

i, j - 迭代器
n - 相对于当前位置的位置

返回值

1,2) 按 n 推进后的 i 的副本
3) 按 -n 推进后的 i 的副本
4) i 和 j 之间的距离
