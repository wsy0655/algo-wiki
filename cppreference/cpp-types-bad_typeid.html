std::bad_typeid

< cpp‎ | types
在标头 <typeinfo> 定义
class bad_typeid : public std::exception;

此类型的异常在应用 typeid 运算符到多态类型的空指针值时抛出。

示例

#include <iostream>
#include <typeinfo>
 
struct S // 类型必须是多态
{
    virtual void f();
}; 
 
int main()
{
    S* p = nullptr;
    try
    {
        std::cout << typeid(*p).name() << '\n';
    }
    catch (const std::bad_typeid& e)
    {
        std::cout << e.what() << '\n';
    }
}

可能的输出：

Attempted a typeid of NULL pointer!
