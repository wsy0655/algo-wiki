std::ranges::iota_view<W, Bound>::empty

< cpp‎ | ranges‎ | iota view
constexpr bool empty() const;
 (C++20 起)

检查范围是否为空（即起始值是否等于哨位值）。

返回值

value_ == bound_

示例

#include <cassert>
#include <ranges>
 
int main()
{
    auto a = std::ranges::iota_view<int, int>();
    assert(a.empty());
 
    auto b = std::ranges::iota_view(4);
    assert(!b.empty());
 
    auto c = std::ranges::iota_view(4, 8);
    assert(!c.empty());
}
