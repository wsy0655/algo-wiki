进行比较的函数对象。主模板调用 T 类型的 const 左值上的 operator< 并对结果取反。特化 greater_equal<void> 从实参推导函数调用运算符的形参类型（但非返回类型）。

greater_equal 的所有特化均为 Semiregular。

 (1) (仅为主 greater_equal<T> 模板的成员)
template< class T, class U >

    requires ranges::StrictTotallyOrderedWith<T, U> ||
             /* std::declval<T>() < std::declval<U>()
                解析成比较指针的内建运算符 */

constexpr bool operator()(T&& t, U&& u) const;
 (2) (仅为 greater_equal<void> 特化的成员)
1) 比较 x 与 y。等价于 return !ranges::less<>{}(x, y)；
2) 比较 t 与 u。等价于 return !ranges::less<>{}(std::forward<T>(t), std::forward<U>(u));。

注解

不同于 std::greater_equal，ranges::greater_equal 要求六个比较运算符 <、<=、>、>=、== 及 != 均合法（通过 StrictTotallyOrdered 与 StrictTotallyOrderedWith 制约）并且完全以 ranges::less 定义。然而，实现可以自由地直接使用 operator>= ，因为这些运算符要求比较运算符的结果一致。
