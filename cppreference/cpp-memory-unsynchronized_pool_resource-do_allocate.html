std::pmr::unsynchronized_pool_resource::do_allocate

< cpp‎ | memory‎ | unsynchronized pool resource
virtual void* do_allocate( std::size_t bytes, std::size_t alignment );
 (C++17 起)

分配存储。

若为 bytes 大小的块所选的池不足以满足来自其内部数据结构的请求，则在上游内存资源上调用 allocate() 以获得内存。

若请求的大小大于最大的池所能处理者，则在上游内存资源上调用 allocate() 分配内存。

返回值

指向至少有 bytes 字节大小的已分配存储，若支持指定的对齐 alignment 则按此对齐，否则对齐到 alignof(std::max_align_t)。

异常

除非上游内存资源上的 allocate() 调用抛出异常，否则不抛出。
