std::moneypunct<CharT,International>::frac_digits, do_frac_digits

< cpp‎ | locale‎ | moneypunct
在标头 <locale> 定义
public:
int frac_digits() const;
 (1)
protected:
virtual int do_frac_digits() const;
 (2)
1) 公开成员函数，调用最终派生类的成员函数 do_frac_digits。
2) 返回打印货币值时出现在小数点后的位数。

返回值

要显示于小数点后的位数。在常用的美国本地环境中，此为值 2。

示例

#include <iomanip>
#include <iostream>
#include <iterator>
#include <locale>
 
struct space_out : std::moneypunct<char>
{
    pattern do_pos_format() const { return {value, none, none, none}; }
    int do_frac_digits() const { return 0; }
    char_type do_thousands_sep() const { return ' '; }
    string_type do_grouping() const { return "\002"; }
};
 
int main()
{
    std::cout.imbue(std::locale("en_US.UTF-8"));
    std::cout << "美国本地环境: " << std::showbase
              << std::put_money(12345678.0) << '\n';
 
    std::cout.imbue(std::locale(std::cout.getloc(), new space_out));
    std::cout << "带有修改的 moneypunct 的本地环境: "
              << std::put_money(12345678.0) << '\n';
}

输出：

美国本地环境: $123,456.78
带有修改的 moneypunct 的本地环境: 12 34 56 78
