std::numpunct_byname

< cpp‎ | locale
在标头 <locale> 定义
template< class CharT >
class numpunct_byname : public std::numpunct<CharT>;

std::numpunct_byname 是 std::numpunct 刻面，它封装在它构造时指定的本地环境的数值标点偏好。

特化

标准库保证提供以下特化：

在标头 <locale> 定义
std::numpunct_byname<char> 窄字符输入/输出的本地环境限定 std::numpunct 刻面
std::numpunct_byname<wchar_t> 宽字符输入/输出的本地环境限定 std::numpunct 刻面

为名为 name 的本地环境构造新的 std::numpunct_byname 刻面。

refs 用于资源管理：在销毁最后一个保有刻面的 std::locale 对象时，如果 refs == 0，那么实现会销毁刻面对象。否则不销毁对象。

示例

此示例演示如何应用另一语言的数值标点规则而不更改本地环境的剩余部分。

#include <iostream>
#include <locale>
 
int main()
{
    const double number = 1000.25;
    std::wcout << L"默认本地环境：" << number << L'\n';
    std::wcout.imbue(std::locale(std::wcout.getloc(),
                                 new std::numpunct_byname<wchar_t>("ru_RU.UTF8")));
    std::wcout << L"带有俄语数值标点的本地环境：" << number << L'\n';
}

输出：

默认本地环境：1000.25
带有俄语数值标点的本地环境：1 000,25
