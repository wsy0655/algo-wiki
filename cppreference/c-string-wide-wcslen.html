wcslen, wcsnlen_s

< c‎ | string‎ | wide
在标头 <wchar.h> 定义
size_t wcslen( const wchar_t *str );
 (1) (C95 起)
size_t wcsnlen_s(const wchar_t *str, size_t strsz);
 (2) (C11 起)
1) 返回宽字符串的长度，即空终止宽字符之前的非空宽字符数。
2) 同 (1)，除了若 str 为空指针则函数返回零，而若在 src 的前 strsz 个宽字符中找不到空宽字符则返回 strsz。
同所有边界检查函数，wcslen_s，仅若实现定义 __STDC_LIB_EXT1__ 且用户在包含 <stdio.h> 前定义 __STDC_WANT_LIB_EXT1__ 为整数常量 1 才保证可用。.

参数

str - 指向要检验的空终止宽字符串的指针
strsz - 要检验的最大宽字符数

返回值

1) 空终止宽字符串 str 的长度。
2) 成功时为空终止宽字符串 str 的长度，若 str 为空指针则为零，若找不到空宽字符则为 strsz。

注意

strnlen_s 和 wcsnlen_s 是仅有的不调用运行时制约处理的带边界检查函数。它们是纯工具函数，用于为非空终止字符串提供有限制支持。

示例

#include <wchar.h>
#include <stdio.h>
 
int main(void)
{
    wchar_t str[] = L"How many wide characters does this string contain?";
 
    printf("without null character: %zu\n", wcslen(str));
    printf("with null character: %zu\n", sizeof str / sizeof *str);
}

输出：

without null character: 50
with null character: 51
