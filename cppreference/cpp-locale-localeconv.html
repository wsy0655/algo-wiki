std::localeconv

< cpp‎ | locale
在标头 <clocale> 定义
std::lconv* localeconv();

localeconv 函数获得指向 std::lconv 类型的静态对象的指针，该对象表示当前 C 本地环境的数值和货币格式化规则。

参数

（无）

返回值

指向当前 std::lconv 对象的指针。

注解

通过返回的指针修改对象是未定义行为。

std::localeconv 修改静态对象，从不同线程调用它而不同步是未定义行为。

示例

#include <clocale>
#include <iostream>
 
int main()
{
    std::setlocale(LC_ALL, "ja_JP.UTF-8");
    std::lconv* lc = std::localeconv();
    std::cout << "日本货币符号: " << lc->currency_symbol
              << '(' << lc->int_curr_symbol << ")\n";
}

输出：

日本货币符号: ￥(JPY )
