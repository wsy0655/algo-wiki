std::uses_allocator<std::function>

< cpp‎ | utility‎ | functional‎ | function
在标头 <functional> 定义
template< class R, class... ArgTypes, class Alloc >
struct uses_allocator<std::function<R(ArgTypes...)>, Alloc> : std::true_type { };
 (C++11 起)
(C++17 前)

std::uses_allocator 的此特化告知其他库组件，std::function 类型的所有对象都支持使用分配器构造，即使它们无嵌套的 allocator_type。

注解

std::function 的分配器支持说明贫乏，且实现不一致。一些实现完全不提供 uses-allocator 构造 ，一些提供重载但忽略提供的分配器参数，而一些实现提供重载并将提供的分配器用于构造，但不在重赋值 std::function 时使用。结果，C++17 中移除了分配器支持。
