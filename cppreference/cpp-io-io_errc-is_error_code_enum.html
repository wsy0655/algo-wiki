std::is_error_code_enum<std::io_errc>

< cpp‎ | io‎ | io errc
在标头 <ios> 定义
template<>
struct is_error_code_enum<std::io_errc> : public std::true_type {};
 (C++11 起)

此 std::is_error_code_enum 特化指示其他库组件 std::io_errc 的值是保有错误码的枚举，这使得它们可以隐式转换到并赋值给 std::error_code 类型的对象。

示例

e.code() 与 std::io_errc::stream 之间的比较能编译，因为 std::is_error_code_enum<std::io_errc>::value == true。

#include <fstream>
#include <iostream>
 
int main()
{
    std::ifstream f("doesn't exist");
    try
    {
        f.exceptions(f.failbit);
    }
    catch (const std::ios_base::failure& e)
    {
        std::cout << "捕获了 ios_base::failure。\n";
        if (e.code() == std::io_errc::stream)
            std::cout << "错误码为 std::io_errc::stream\n";
    }
}

输出：

捕获了 ios_base::failure。
错误码为 std::io_errc::stream
