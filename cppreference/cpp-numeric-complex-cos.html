std::cos(std::complex)

< cpp‎ | numeric‎ | complex
在标头 <complex> 定义
template< class T >
complex<T> cos( const complex<T>& z );

计算复数值 z 的复余弦。

参数

z - 复数值

返回值

若不出现错误，则返回 z 的复余弦。

如同以 std::cosh(i * z) 实现运算一般处理错误和特殊情况，其中 i 为虚数单位。

注解

余弦在复平面上是整函数，而且无分支切割。

余弦的数学定义是 cos z =
eiz
+e-iz
2
。

示例

#include <cmath>
#include <complex>
#include <iostream>
 
int main()
{
    std::cout << std::fixed;
    std::complex<double> z(1.0, 0.0); // 表现为沿实轴的实余弦
    std::cout << "cos" << z << " = " << std::cos(z)
              << " ( cos(1) = " << std::cos(1) << ")\n";
 
    std::complex<double> z2(0.0, 1.0); // 表现为沿虚轴的实 cosh
    std::cout << "cos" << z2 << " = " << std::cos(z2)
              << " (cosh(1) = " << std::cosh(1) << ")\n";
}

cos(0.000000,1.000000) = (1.543081,-0.000000) (cosh(1) = 1.543081)
