比较底层迭代器。

令 current_ 为指向所适配视图的元素的迭代器的底层元组式 ﻿对象。

1) 返回：
  • x.current_ == y.current_，如果 /*all-bidirectional*/<Const, Views...> 为 true。
  • 否则为 true，如果存在整数 0 <= i < sizeof...(Views) 使得 bool(std::get<i>(x.current_) == std::get<i>(y.current_)) 为 true。
  • 否则为 false。
2) 等价于 return x.current_ <=> y.current_;。

此函数对常规的无限定或有限定查找不可见，而只能在 std::ranges::zip_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

<、<=、>、>= 及 != 运算符分别从 operator<=> 与 operator== 合成。
