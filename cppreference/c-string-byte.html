空终止字节字符串

< c‎ | string


空终止字节字符串（NTBS）是非零字节的序列后随一个零值字节（空终止字符）。字节字符串中的每个字节都是某个字符集中被编码的一个字符。例如，字符数组 {'\x63','\x61','\x74','\0'} 是一个以 ASCII 编码表示字符串 "cat" 的 NTBS。
