示例

展示日历时间。

#include <ctime>
#include <iostream>
 
int main()
{
    std::tm tm{};
    tm.tm_year = 2022 - 1900;
    tm.tm_mday = 1;
    std::mktime(&tm);
 
    std::cout << std::asctime(&tm); // 注意隐含的尾部 '\n'
}

可能的输出：

Sat Jan  1 00:00:00 2022
