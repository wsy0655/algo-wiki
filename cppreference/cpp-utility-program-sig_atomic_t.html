std::sig_atomic_t

< cpp‎ | utility‎ | program
在标头 <csignal> 定义
typedef /* 未指明 */ sig_atomic_t;

即使在存在由信号产生的异步中断的情况下，也可以作为原子实体进行访问的整数类型。

注解

在 C++ 11 引入 std::atomic 和 std::atomic_signal_fence 前，严格符合规范的信号处理程序中能做的事情几乎只有给一个静态的 volatile static std::sig_atomic_t 变量赋值并立即返回。
