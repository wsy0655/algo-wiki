std::fgetpos

< cpp‎ | io‎ | c
在标头 <cstdio> 定义
int fgetpos( std::FILE* stream, std::fpos_t* pos );

获得文件流 stream 的文件位置指示器和当前分析状态（若存在），并将它们存储于 pos 所指向的对象。存储的值仅在作为 std::fsetpos 的输入的情况有意义。

参数

stream - 要检验的文件流
pos - 指向要存储文件位置指示器的 fpos_t 对象的指针

返回值

成功时为 ​0​，否则非零值。失败时还设置 errno。
