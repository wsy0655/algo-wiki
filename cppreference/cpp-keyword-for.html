C++ 关键词：for

< cpp‎ | keyword


用法

  • for 循环：用于该循环的声明
  • 基于范围的 for 循环：用于该循环的声明
 (C++11 起)

示例

#include <iostream>
 
int main() noexcept
{
    // 以下 'for' 循环语句：
    // 1. (初始化语句) 声明了名为 'i' 的整数并以值 '0' 对它初始化。
    // 2. (条件)      检查 i 是否小于 3，若为假则结束循环执行。
    // 3. (语句)      将整数 'i' 的当前值向 stdout 写出。
    // 4. (表达式)     前缀自增整数 'i'（将其值增加 1）。
    // 5.             返回到 2. (条件)
 
                                // 初始化语句: int i {0};
                                // 条件:      i < 3
    for (int i {0}; i < 3; ++i) // 表达式:     ++i
        std::cout << i;         // 语句:      std::cout << i;
}

输出：

012
