 (A)
1-3) 以当前舍入模式舍入浮点数实参 num 到浮点数格式的整数。标准库提供所有以无 cv 限定的浮点数类型作为形参的类型的 std::nearbyint 重载。(C++23 起)
S) SIMD 重载对 v_num 实施逐元素 std::nearbyint。
（参见 math-floating-point 和 deduced-simd-t 的定义。）
 (C++26 起)
A) 为所有整数类型提供额外重载，将它们当做 double。
 (C++11 起)

参数

num - 浮点数或整数

返回值

返回 num 按照当前舍入模式的最接近整数。

错误处理

此函数不受制于任何指定于 math_errhandling 的错误。

如果实现支持 IEEE 浮点数算术（IEC 60559），那么

  • 决不引发 FE_INEXACT
  • 如果 num 是 ±∞，那么返回不修改的该值
  • 如果 num 是 ±0，那么返回不修改的该值
  • 如果 num 是 NaN，那么返回 NaN

注解

std::nearbyint 和 std::rint 之间仅有的区别是 std::nearbyint 永远不会引发 FE_INEXACT。

所有标准浮点数格式的最大可表示浮点数都是整数，所以 std::nearbyint 自身永远不会溢出；然而将结果存储到整数对象时，结果可能溢出任何整数类型（包括 std::intmax_t）。

如果当前舍入模式是 FE_TONEAREST，那么此函数在中点情况向偶数舍入（同 std::rint，但不同于 std::round）。

额外重载不需要以 (A) 的形式提供。它们只需要能够对它们的整数类型实参 num 确保 std::nearbyint(num) 和 std::nearbyint(static_cast<double>(num)) 的效果相同。

示例

#include <cfenv>
#include <cmath>
#include <iostream>
#pragma STDC FENV_ACCESS ON
 
int main()
{
    std::fesetround(FE_TONEAREST);
    std::cout << "就近舍入：\n"
              << "nearbyint(+2.3) = " << std::nearbyint(2.3)
              << "  nearbyint(+2.5) = " << std::nearbyint(2.5)
              << "  nearbyint(+3.5) = " << std::nearbyint(3.5) << '\n'
              << "nearbyint(-2.3) = " << std::nearbyint(-2.3)
              << "  nearbyint(-2.5) = " << std::nearbyint(-2.5)
              << "  nearbyint(-3.5) = " << std::nearbyint(-3.5) << '\n';
 
    std::fesetround(FE_DOWNWARD);
    std::cout << "向下舍入：\n"
              << "nearbyint(+2.3) = " << std::nearbyint(2.3)
              << "  nearbyint(+2.5) = " << std::nearbyint(2.5)
              << "  nearbyint(+3.5) = " << std::nearbyint(3.5) << '\n'
              << "nearbyint(-2.3) = " << std::nearbyint(-2.3)
              << "  nearbyint(-2.5) = " << std::nearbyint(-2.5)
              << "  nearbyint(-3.5) = " << std::nearbyint(-3.5) << '\n';
 
    std::cout << "nearbyint(-0.0) = " << std::nearbyint(-0.0)  << '\n'
              << "nearbyint(-Inf) = " << std::nearbyint(-INFINITY) << '\n';
}

输出：

就近舍入：
nearbyint(+2.3) = 2  nearbyint(+2.5) = 2  nearbyint(+3.5) = 4
nearbyint(-2.3) = -2  nearbyint(-2.5) = -2  nearbyint(-3.5) = -4
向下舍入：
nearbyint(+2.3) = 2  nearbyint(+2.5) = 2  nearbyint(+3.5) = 3
nearbyint(-2.3) = -3  nearbyint(-2.5) = -3  nearbyint(-3.5) = -4
nearbyint(-0.0) = -0
nearbyint(-Inf) = -inf
