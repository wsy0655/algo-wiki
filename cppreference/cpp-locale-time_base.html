std::time_base

< cpp‎ | locale
在标头 <locale> 定义
class time_base;

类 std::time_base 提供日期顺序常量，它为 std::time_get 刻面所继承。

成员类型

成员类型 描述
enum dateorder { no_order, dmy, mdy, ymd, ydm }; 无作用域枚举类型
枚举常量 描述
no_order 未指定的顺序
dmy 日、月、年（欧洲）顺序
mdy 月、日、年（美国）顺序
ymd 年、月、日（亚洲）顺序
ydm 年、日、月
