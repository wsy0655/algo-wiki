返回转换到有符号类型的范围大小。

给定其（可能经实质化的）结果对象以 t 表示的子表达式为 E：

  • 如果 ranges::size(t) 非良构，那么 ranges::ssize(E) 也非良构。
  • 否则，设 Signed 为 make-signed-like-t ﻿<decltype(ranges::size(t))>：
    • 如果 std::ptrdiff_t 比 Signed 宽，那么 ranges::ssize(E) 表达式等价于 static_cast<std::ptrdiff_t>(ranges::size(t))。
    • 否则 ranges::ssize(E) 表达式等价于 static_cast<Signed>(ranges::size(t))。

定制点对象

名字 ranges::ssize 代表一个定制点对象，它是某个字面 semiregular 类类型的 const 函数对象。 细节参见定制点对象 (CustomizationPointObject) 。

注解

只要 ranges::size(e) 对表达式 e 合法，那么返回类型是有符号整数式类型。

示例

#include <array>
#include <iostream>
#include <ranges>
#include <type_traits>
 
int main()
{
    std::array arr{1, 2, 3, 4, 5};
    auto s = std::ranges::ssize(arr);
 
    std::cout << "ranges::ssize(arr) = " << s << '\n'
              << "ranges::ssize "
              << (std::is_signed_v<decltype(s)> ? "有符号" : "无符号")
              << '\n';
 
    std::cout << "逆向的 arr: ";
 
    for (--s; s >= 0; --s)
        std::cout << arr[s] << ' ';
 
    std::cout << "\n" "s = " << s << '\n';
}

输出：

ranges::ssize(arr) = 5
ranges::ssize 有符号
逆向的 arr: 5 4 3 2 1 
s = -1
