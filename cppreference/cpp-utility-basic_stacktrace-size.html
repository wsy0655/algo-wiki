std::basic_stacktrace<Allocator>::size

< cpp‎ | utility‎ | basic stacktrace
size_type size() const noexcept;
 (C++23 起)

返回栈踪迹中的条目个数。

参数

（无）

返回值

栈踪迹中的条目个数。

复杂度

常数。

示例

以下代码用 size 显示当前栈踪迹中的条目个数：

#include <stacktrace>
#include <iostream>
 
int main()
{ 
    auto trace = std::stacktrace::current();
 
    std::cout << "栈中包含 " << trace.size() << " 个条目。\n";
}

可能的输出：

栈中包含 3 个条目。

empty
