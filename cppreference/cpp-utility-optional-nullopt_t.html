std::nullopt_t

< cpp‎ | utility‎ | optional
在标头 <optional> 定义
struct nullopt_t;
 (C++17 起)

std::nullopt_t 是空类类型，用于指示 optional 类型拥有未初始化状态。具体而言，std::optional 有一个以 nullopt_t 为单实参的构造函数，它创建不含值的 optional。

std::nullopt_t 必须是非聚合字面类型 (LiteralType) ，而且不能有默认构造函数或初始化式列表构造函数。

它必须有一个接收实现定义字面类型的 constexpr 构造函数。

注解

nullopt_t 构造函数上的制约，在于支持 op = {}; 和 op = nullopt; 两者都是令 optional 不含对象的语法。

此类的一种可能实现是

struct nullopt_t {
    constexpr explicit nullopt_t(int) {}
};
