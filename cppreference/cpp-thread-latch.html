std::latch

< cpp‎ | thread
在标头 <latch> 定义
class latch;
 (C++20 起)

latch 类是一种 std::ptrdiff_t 类型的向下计数器，它能用于同步线程。在创建时初始化计数器的值。线程可能在闩上阻塞直至计数器减少到零。没有可能增加或重置计数器，这使得 latch 成为一种单次使用的屏障。

同时调用 latch 的成员函数，除了析构函数，不引入数据竞争。

示例

#include <functional>
#include <iostream>
#include <latch>
#include <string>
#include <thread>
 
struct Job
{
    const std::string name;
    std::string product{"未工作"};
    std::thread action{};
};
 
int main()
{
    Job jobs[]{{"Annika"}, {"Buru"}, {"Chuck"}};
 
    std::latch work_done{std::size(jobs)};
    std::latch start_clean_up{1};
 
    auto work = [&](Job& my_job)
    {
        my_job.product = my_job.name + " 已工作";
        work_done.count_down();
        start_clean_up.wait();
        my_job.product = my_job.name + " 已清理";
    };
 
    std::cout << "工作启动... ";
    for (auto& job : jobs)
        job.action = std::thread{work, std::ref(job)};
 
    work_done.wait();
    std::cout << "完成:\n";
    for (auto const& job : jobs)
        std::cout << "  " << job.product << '\n';
 
    std::cout << "清理工作线程... ";
    start_clean_up.count_down();
    for (auto& job : jobs)
        job.action.join();
 
    std::cout << "完成:\n";
    for (auto const& job : jobs)
        std::cout << "  " << job.product << '\n';
}

输出：

工作启动... 完成:
  Annika 已工作
  Buru 已工作
  Chuck 已工作
清理工作线程... 完成:
  Annika 已清理
  Buru 已清理
  Chuck 已清理
