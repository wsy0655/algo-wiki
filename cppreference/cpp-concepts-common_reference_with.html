std::common_reference_with

< cpp‎ | concepts
在标头 <concepts> 定义
template< class T, class U >

concept common_reference_with =
    std::same_as<std::common_reference_t<T, U>, std::common_reference_t<U, T>> &&
    std::convertible_to<T, std::common_reference_t<T, U>> &&

    std::convertible_to<U, std::common_reference_t<T, U>>;
 (C++20 起)

概念 std::common_reference_with<T, U> 指定两个类型 T 与 U 共享一种二者均能转换到的公共引用类型（以 std::common_reference_t 计算）。

语义要求

T 与 U 实现 common_reference_with<T, U>，仅当给定保持相等性的表达式 t1、t2、u1 及 u2，且它们满足 decltype((t1)) 与 decltype((t2)) 均为 T 而 decltype((u1)) 与 decltype((u2)) 均为 U，且满足以下条件：

  • 当且仅当 t1 等于 t2 时 std::common_reference_t<T, U>(t1) 等于 std::common_reference_t<T, U>(t2)；且
  • 当且仅当 u1 等于 u2 时 std::common_reference_t<T, U>(u1) 等于 std::common_reference_t<T, U>(u2)。

换言之，到公共引用类型的转换必须保持相等性。
