std::formatter<std::queue>

< cpp‎ | container‎ | queue

在标头 <queue> 定义
template< class CharT, class T, std::formattable<CharT> Container, class... U >
struct formatter<std::queue<T, Container, U...>, CharT>;
 (C++23 起)

std::formatter 针对容器适配器类型 std::queue 的模板特化，允许用户使用格式化函数把底层容器转换为它作为元素汇集的文本表示。

如果 std::formattable<Container, CharT> 为 true 则启用此特化。
