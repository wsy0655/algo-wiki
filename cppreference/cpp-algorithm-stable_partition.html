std::stable_partition

< cpp‎ | algorithm
在标头 <algorithm> 定义
template< class BidirIt, class UnaryPred >
BidirIt stable_partition( BidirIt first, BidirIt last, UnaryPred p );
 (1) (C++26 起为 constexpr)
template< class ExecutionPolicy, class BidirIt, class UnaryPred >

BidirIt stable_partition( ExecutionPolicy&& policy,

                          BidirIt first, BidirIt last, UnaryPred p );
 (2) (C++17 起)
1) 重排序范围 [first, last) 中的元素，使得所有谓词 p 对其返回 true 的元素均先于谓词 p 对其返回 false 的元素。保持元素的相对顺序。
2) 同 (1)，但按照 policy 执行。
此重载只有在满足以下所有条件时才会参与重载决议：

std::is_execution_policy_v<std::decay_t<ExecutionPolicy>> 是 true。

 (C++20 前)

std::is_execution_policy_v<std::remove_cvref_t<ExecutionPolicy>> 是 true。

 (C++20 起)

如果满足以下任意条件，那么行为未定义：

  • *first 的类型不可交换 (Swappable) 。
 (C++11 前)
  • BidirIt 不可交换值 (ValueSwappable) 。
  • *first 的类型不可移动构造 (MoveConstructible) 。
  • *first 的类型不可移动赋值 (MoveAssignable) 。
 (C++11 起)

参数

first, last - 要重排序的元素范围的迭代器对
policy - 所用的执行策略
p - 如果元素在顺序中应该在其他元素之前则返回 ​true 的一元谓词。

对每个（可为 const 的） VT 类型参数 v ，其中 VT 是 BidirIt 的值类型，表达式 p(v) 必须可转换到 bool，无关乎值类别，而且必须不修改 v 。从而不允许 VT& 类型参数，亦不允许 VT ，除非对 VT 而言移动等价于复制(C++11 起)。 ​
类型要求
-
BidirIt 必须满足老式双向迭代器 (LegacyBidirectionalIterator) 。
-
UnaryPred 必须满足谓词 (Predicate) 。

返回值

指向第二个分组首元素的迭代器。

复杂度

给定 \(\scriptsize N\)N 为 std::distance(first, last)：

1) 应用 \(\scriptsize N\)N 次 p。
在额外内存足够时交换 \(\scriptsize O(N)\)O(N) 次，否则交换最多 \(\scriptsize N \cdot log_{2}(N)\)N⋅log2(N) 次。
2) 应用 \(\scriptsize O(N)\)O(N) 次 p。
交换 \(\scriptsize N \cdot log(N)\)N⋅log(N) 次。

异常

拥有名为 ExecutionPolicy 的模板形参的重载按下列方式报告错误：

  • 如果作为算法一部分调用的函数的执行抛出异常，且 ExecutionPolicy 是标准策略之一，那么调用 std::terminate。对于任何其他 ExecutionPolicy，行为由实现定义。
  • 如果算法无法分配内存，那么抛出 std::bad_alloc。

注解

此函数试图分配临时缓冲区。如果分配失败，那么就会选择较低效的算法。

libc++ 与 libstdc++ 中的实现亦作为扩展接受老式向前迭代器 (LegacyForwardIterator) 所代表的范围。

示例

#include <algorithm>
#include <iostream>
#include <vector>
 
int main()
{
    std::vector<int> v{0, 0, 3, -1, 2, 4, 5, 0, 7};
    std::stable_partition(v.begin(), v.end(), [](int n) { return n > 0; });
    for (int n : v)
        std::cout << n << ' ';
    std::cout << '\n';
}

输出：

3 2 4 5 7 0 0 -1 0
