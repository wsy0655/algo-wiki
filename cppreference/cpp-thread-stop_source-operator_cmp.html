operator==(std::stop_source)

< cpp‎ | thread‎ | stop source


friend bool operator==( const stop_source& lhs, const stop_source& rhs ) noexcept;
 (C++20 起)

比较两个 stop_source 值。

此函数对常规的无限定或有限定查找不可见，而只能在 std::stop_source 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

lhs, rhs - 要比较的 stop_source

返回值

若 lhs 与 rhs 拥有同一停止状态或均无停止状态则为 true，否则为 false。
