std::experimental::ranges::tagged

< cpp‎ | experimental‎ | ranges
在标头 <experimental/ranges/utility> 定义
template< class Base, TagSpecifier... Tags >

    requires sizeof...(Tags) <= std::tuple_size<Base>::value

struct tagged;
 (范围 TS)

类模板 tagged 将标签指定符 Tags... 所指定的具名访问器集合增设到其所公开派生的元组式类 Base。按顺序应用标签：Tags... 中第 i 个标签指定符对应 tuple 的第 i 个元素。

Tags... 中的标签指定符应当是独一无二的。否则行为未定义。

不可能通过指向异于 Base 的任何基类的指针删除 tagged 的实例。

具名元素访问器

对于 Tags... 中对应元素名为 name 的第 N 个标签指定符，tagged 提供如下的名为 name 的具名访问器：

constexpr decltype(auto) name() &       { return std::get<N>(*this); }
constexpr decltype(auto) name() const & { return std::get<N>(*this); }
constexpr decltype(auto) name() &&      { return std::get<N>(std::move(*this)); }
