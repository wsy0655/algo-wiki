std::atomic_flag_notify_one

< cpp‎ | atomic
在标头 <atomic> 定义
void atomic_flag_notify_one( std::atomic_flag* object ) noexcept;
 (1) (C++20 起)
void atomic_flag_notify_one( volatile std::atomic_flag* object ) noexcept;
 (2) (C++20 起)

进行原子提醒操作。

如果有线程被 *object 上的原子等待操作（即 std::atomic_flag_wait()、std::atomic_flag_wait() 或 std::atomic_flag::wait()）阻塞，那么解除锁定至少 ﻿一个这种线程；否则不做任何事。

等价于 object->notify_one()。

参数

object - 指向待提醒的 atomic_flag 对象的指针

返回值

（无）

注解

这种形式的更改检测通常比简单轮询或纯自旋锁高效。
