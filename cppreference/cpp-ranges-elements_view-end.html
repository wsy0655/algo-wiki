返回表示 elements_view 末尾的哨位或迭代器。

令 base_ 为底层视图。等价于：

1) return /*sentinel*/<false>{ranges::end(base_)};。
2) return /*iterator*/<false>{ranges::end(base_)};。
3) return /*sentinel*/<true>{ranges::end(base_)};。
4) return /*iterator*/<true>{ranges::end(base_)};。

参数

（无）

返回值

1,3) 比较等于尾迭代器的哨位。
2,4) 指向末元素后一位置的迭代器。

注解

end() 当且仅当底层视图为 common_range 时返回迭代器：elements_view<V,F> 凡在 V 实现 common_range 时实现该概念。
