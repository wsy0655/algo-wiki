std::ranges::common_view<V>::begin

< cpp‎ | ranges‎ | common view
constexpr auto begin() requires (!/*simple_view*/<V>);
 (1) (C++20 起)
constexpr auto begin() const requires range<const V>;
 (2) (C++20 起)
1) 返回指向 common_view 首元素的迭代器，即：
  • ranges::begin(base_)，若 ranges::random_access_range<V> 与 ranges::sized_range<V> 均得到满足，
  • 否则为 std::common_iterator<ranges::iterator_t<V>, ranges::sentinel_t<V>>(ranges::begin(base_))。
2) 同 (1)，但 V 为 const 限定。

返回值

指向底层视图起始的迭代器。

示例

#include <iostream>
#include <numeric>
#include <ranges>
#include <string_view>
 
int main()
{
    constexpr auto common = std::views::iota(1)
                          | std::views::take(3)
                          | std::views::common
                          ;
 
    for (int i{}; int e : common)
        std::cout << (i++ ? " + " : "") << e;
 
    std::cout << " = " << std::accumulate(common.begin(), common.end(), 0) << '\n';
}

输出：

1 + 2 + 3 = 6
