设置随机数引擎的状态。

1) 状态设置完成时 *this == std::linear_congruential_engine(value) 是 true。
2) 状态设置完成时 *this == std::linear_congruential_engine(seq) 是 true。

参数

value - 用于设置状态的种子值
seq - 用于设置状态的种子序列

异常

2) 如果 SeedSeq 不是 std::seed_seq，那么就会抛出 seq.generate 调用抛出的异常。

复杂度

1) 与 std::linear_congruential_engine(value) 相同。
2) 与 std::linear_congruential_engine(seq) 相同。
