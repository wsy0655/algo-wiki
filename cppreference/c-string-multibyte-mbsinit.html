mbsinit

< c‎ | string‎ | multibyte
在标头 <wchar.h> 定义
int mbsinit( const mbstate_t* ps);
 (C95 起)

若 ps 不是空指针，则 mbsinit 函数确定其所指向的 mbstate_t 对象是否描述初始转换状态。

注意

尽管零初始化的 mbstate_t 始终表示初始转换状态，亦可能有其他 mbstate_t 值表示初始转换状态。

参数

ps - 指向要检验的 mbstate_t 对象的指针

返回值

若 ps 不是空指针且不表示初始迁移状态则为 ​0​，否则为非零。

示例

#include <locale.h>
#include <string.h>
#include <stdio.h>
#include <wchar.h>
 
int main(void)
{
    // 允许 mbrlen() 作用于 UTF-8 多字节字符串
    setlocale(LC_ALL, "en_US.utf8");
    // UTF-8 窄多字节编码
    const char* str = u8"水"; // 即 u8"\u6c34" 或 "\xe6\xb0\xb4"
    static mbstate_t mb; // 零初始化
    (void)mbrlen(&str[0], 1, &mb);
    if (!mbsinit(&mb)) {
        printf("处理 %s 首字节后，\n"
               "转换状态未初始化\n\n", str);
    }
 
    (void)mbrlen(&str[1], strlen(str), &mb);
    if (mbsinit(&mb)) {
        printf("处理 %s 末两字节后，\n"
               "转换状态为初始转换状态\n", str);
    }
}

输出：

处理 水 首字节后，
转换状态未初始化
 
处理 水 末两字节后，
转换状态为初始转换状态
