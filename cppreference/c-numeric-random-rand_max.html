RAND_MAX

< c‎ | numeric‎ | random


在标头 <stdlib.h> 定义
#define RAND_MAX /* 由实现定义 */

展开成等于函数 rand() 最大返回值的整数常量表达式。此值为实现定义。标准保证此值至少为 32767。

示例

#include <limits.h>
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
 
int main(void)
{
    srand(time(NULL)); // 使用当前时间作为随机数生成器的种子
    printf("RAND_MAX: %i\n", RAND_MAX);
    printf("INT_MAX: %i\n", INT_MAX);
    printf("Random value on [0,1]: %f\n", (double)rand() / RAND_MAX);
}

可能的输出：

RAND_MAX: 2147483647
INT_MAX: 2147483647
Random value on [0,1]: 0.362509
