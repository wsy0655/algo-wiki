将 sub_match 与另一 sub_match、std::string、C 风格字符串或单个字符进行比较。

1-7) 直接比较两个 sub_match。
8-20) 比较 sub_match 与 std::basic_string。
21-33) 比较 sub_match 与 C 风格字符串。
34-46) 比较 sub_match 与单个字符。

<、<=、>、>= 及 != 运算符分别从 operator<=> 与 operator== 合成。

 (C++20 起)
47) value-type-of ﻿<BidirIt> 是 BidirIt 的值类型。
48) cat-type-of ﻿<BidirIt> 是 std::sub_match<BidirIt> 的三路比较结果类型。

参数

lhs, rhs, sm - 要比较的 sub_match
str - 要比较的 std::basic_string
s - 指向要比较的 C 风格字符串的指针
ch - 要比较的字符

返回值

设 target 为以下值：

1-7) rhs
8-20) typename std::sub_match<BidirIt>::string_type(str.data(), str.size())
21-33) s
34-46) typename std::sub_match<BidirIt>::string_type(1, ch)

返回值定义如下

  运算符   返回值
重载 (1-14,21-27,34-40)
（带有形参 lhs 的重载）		            重载 (15-20,28-33,41-46)           
（不带形参 lhs 的重载）
== lhs.compare(target) == 0 rhs.compare(target) == 0
!= lhs.compare(target) != 0 rhs.compare(target) != 0
< lhs.compare(target) < 0 rhs.compare(target) > 0
<= lhs.compare(target) <= 0 rhs.compare(target) >= 0
> lhs.compare(target) > 0 rhs.compare(target) < 0
>= lhs.compare(target) >= 0 rhs.compare(target) <= 0
<=>  static_cast<cat-type-of ﻿<BidirIt>>
    (lhs.compare(target) <=> 0) 		 不适用

注解

operator<=> 的返回类型保证是比较类别类型。如果 value-type-of ﻿<BidirIt> 是 char、wchar_t、char8_t、char16_t 或 char32_t，那么 operator<=> 的返回类型是 std::strong_ordering。
