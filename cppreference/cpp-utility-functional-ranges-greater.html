std::ranges::greater::operator()

template< class T, class U >
constexpr bool operator()(T&& t, U&& u) const;

等价于 return ranges::less{}(std::forward<U>(u), std::forward<T>(t));。

此重载只有在满足 std::totally_ordered_with<T, U> 时才会参与重载决议。

注解

与 std::greater 不同，std::ranges::greater 要求全部六个比较运算符 <、<=、>、>=、== 和 != 合法（经由 totally_ordered_with 约束），并且完全以 std::ranges::less 定义。
