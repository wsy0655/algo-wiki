std::filesystem::path::format

< cpp‎ | filesystem‎ | path
enum format {

    native_format,
    generic_format,
    auto_format

};
 (C++17 起)

确定路径名的字符串表示如何为接受字符串的 std::filesystem::path 构造函数所判读。

常量

名称 解释
native_format 原生路径名格式
generic_format 通用路径名格式
auto_format 实现定义路径名格式，可行的场合自动检测

注解

在 POSIX 系统上，原生与通用格式无区别。
