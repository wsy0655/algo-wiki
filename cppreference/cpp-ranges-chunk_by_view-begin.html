std::ranges::chunk_by_view<V,Pred>::begin

< cpp‎ | ranges‎ | chunk by view
constexpr /*iterator*/ begin();
 (C++23 起)

返回指向 chunk_by_view 首元素的迭代器。

等价于：

ranges::iterator_t<V> iter;
 
if (begin_.has_value())
    iter = begin_.value();
else
{
    iter = /*find_next*/(ranges::begin(base()));
    begin_ = iter; // 缓存
}
 
return /*iterator*/(*this, ranges::begin(base()), iter);

如果底层谓词 pred_ 不包含值，则其行为未定义。

参数

（无）

返回值

指向首元素的迭代器。

注解

为了提供 range 概念所需的均摊常数时间复杂度，该函数将结果缓存在数据成员 begin_ 中，供后续调用使用。
