jmp_buf

< c‎ | program
在标头 <setjmp.h> 定义
typedef /* 未指明 */ jmp_buf;

jmp_buf 类型是适合于存储并恢复调用环境的数组类型。存储的信息足以恢复程序在当前块的执行和该块的调用。浮点状态标志、打开的文件或其他任何数据不存储于 jmp_buf 类型对象中。
