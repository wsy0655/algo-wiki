标准库标头 <cstdarg>

< cpp‎ | header


此标头原作为 <stdarg.h> 存在于 C 标准库。

此头文件提供对 C 风格变参函数的支持，并且“默认实参提升”的 C 定义会替换成对应的 C++ 定义。

类型
va_list
 保有 va_start、va_arg、va_end 和 va_copy 所需的信息
(typedef)

宏
va_start
 启用对可变函数实参的访问
(宏函数)
va_arg
 访问下一个可变函数实参
(宏函数)
va_copy
(C++11)
 制造可变函数实参的副本
(宏函数)
va_end
 结束对可变函数实参的遍历
(宏函数)

概要

namespace std {
  using va_list = /* 见描述 */ ;
}
#define va_arg(V, P) /* 见描述 */
#define va_copy(VDST, VSRC) /* 见描述 */
#define va_end(V) /* 见描述 */
#define va_start(V, P) /* 见描述 */
