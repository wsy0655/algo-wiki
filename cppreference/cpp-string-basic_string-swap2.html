 (C++17 前)
template< class CharT, class Traits, class Alloc >

void swap( std::basic_string<CharT, Traits, Alloc>& lhs,

           std::basic_string<CharT, Traits, Alloc>& rhs ) noexcept(/* 见下文 */);
 (C++17 起)
(C++20 起为 constexpr)

对 std::basic_string 特化 std::swap 算法。交换 lhs 与 rhs 的内容。等价于 lhs.swap(rhs)。

参数

lhs, rhs - 要交换内容的字符串

返回值

（无）

复杂度

常数。

异常

noexcept 说明：  
noexcept(noexcept(lhs.swap(rhs)))
 (C++17 起)

示例

#include <iostream>
#include <string>
 
int main()
{
    std::string a = "AAA";
    std::string b = "BBBB";
 
    std::cout << "交换前:\n"
                 "a = " << a << "\n"
                 "b = " << b << "\n\n";
 
    std::swap(a, b);
 
    std::cout << "交换后:\n"
                 "a = " << a << "\n"
                 "b = " << b << '\n';
}

输出：

交换前:
a = AAA
b = BBBB
 
交换后:
a = BBBB
b = AAA
