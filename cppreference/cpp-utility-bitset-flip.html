std::bitset<N>::flip

< cpp‎ | utility‎ | bitset
bitset& flip();
 (1) (C++11 起为 noexcept)
(C++23 起为 constexpr)
bitset& flip( std::size_t pos );
 (2) (C++23 起为 constexpr)

翻转位，即把 true 值改为 false 并把 false 值改为 true。等价于在 bitset 一部分或全体上的逻辑非运算。

1) 翻转所有位（类似 operator~，但在原位操作）。
2) 翻转处于位置 pos 的位。

参数

pos - 要翻转的位的位置

返回值

*this

异常

2) 在 pos 未对应有效位位置时抛出 std::out_of_range。

示例

#include <bitset>
#include <iostream>
 
int main()
{
    std::bitset<4> flops;
 
    std::cout << flops << '\n'
              << flops.flip(0) << '\n'
              << flops.flip(2) << '\n'
              << flops.flip() << '\n';
}

输出：

0000
0001
0101
1010
