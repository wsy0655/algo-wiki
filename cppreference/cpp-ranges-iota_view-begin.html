std::ranges::iota_view<W, Bound>::begin

< cpp‎ | ranges‎ | iota view
constexpr /*iterator*/ begin() const;
 (C++20 起)

获取一个指向起始值的迭代器。

返回值

iterator ﻿{value_ ﻿}

示例

#include <iostream>
#include <ranges>
 
int main()
{
    auto iota{std::views::iota(2, 6)};
    auto iter{iota.begin()};
    while (iter != iota.end())
        std::cout << *iter++ << ' ';
    std::cout << '\n';
}

输出：

2 3 4 5
