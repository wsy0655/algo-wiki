std::basic_streambuf<CharT,Traits>::gbump

< cpp‎ | io‎ | basic streambuf
protected:
void gbump( int count );

通过使 获取指针 增加 count 来跳过获取区中的 count 个字符。不为下溢做检查。

参数

count - 要跳过的字符数

返回值

（无）

注解

因为此函数接收 int，所以它无法操纵大于 std::numeric_limits<int>::max() 个字符的缓冲区（LWG 255）。
