std::enable_shared_from_this<T>::shared_from_this

< cpp‎ | memory‎ | enable shared from this
std::shared_ptr<T> shared_from_this();
 (1) (C++11 起)
std::shared_ptr<T const> shared_from_this() const;
 (2) (C++11 起)

返回与所有指代 *this 的 std::shared_ptr 共享 *this 所有权的 std::shared_ptr<T>。

返回值

std::shared_ptr<T>(weak_this ﻿)

异常

如果调用 shared_from_this 的对象还没有以 std::shared_ptr 共享，那么就会通过 std::shared_ptr 的构造函数抛出 std::bad_weak_ptr。

示例

#include <iostream>
#include <memory>
 
struct Foo : public std::enable_shared_from_this<Foo>
{
    Foo() { std::cout << "Foo::Foo\n"; }
    ~Foo() { std::cout << "Foo::~Foo\n"; } 
    std::shared_ptr<Foo> getFoo() { return shared_from_this(); }
};
 
int main()
{
    Foo *f = new Foo;
    std::shared_ptr<Foo> pf1;
 
    {
        std::shared_ptr<Foo> pf2(f);
        pf1 = pf2->getFoo();  // 与 pf2 的对象共享所有权
    }
 
    std::cout << "pf2 已经消失\n";   
}

输出：

Foo::Foo
pf2 已经消失
Foo::~Foo
