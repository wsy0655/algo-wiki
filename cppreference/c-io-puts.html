puts

< c‎ | io
在标头 <stdio.h> 定义
int puts( const char *str );

写入每个来自空终止字符串 str 的字符及附加换行符 '\n' 到输出流 stdout ，如同以重复执行 fputc 写入。

不写入来自 str 的空终止字符。

参数

str - 要写入的字符串

返回值

成功时返回非负值。

失败时，返回 EOF 并设置 stdout 的错误指示器（见 ferror()）。

注意

puts 函数后附一个换行字符到输出，而 fputs 不这么做。

不同的实现返回不同的非负数：一些返回最后写入的字符，一些返回写入的字符数（或若字符串长于 INT_MAX 则返回它），一些简单地返回非负常量。

在重定向 stdout 到文件时，导致 puts 失败的典型原因是用尽了文件系统的空间。

示例

#include <stdio.h>
 
int main(void)
{
    int rc = puts("Hello World");
 
    if (rc == EOF)
        perror("puts()"); // POSIX 要求设置 errno
}

输出：

Hello World
