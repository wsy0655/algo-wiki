std::abs(std::valarray)

< cpp‎ | numeric‎ | valarray
在标头 <valarray> 定义
template< class T >
valarray<T> abs( const valarray<T>& va );

计算值数组中每个元素的绝对值。

参数

va - 要应用操作到的值数组

返回值

含 va 中值的绝对值的值数组。

注解

用无限定函数 (abs) 进行计算。若该函数不可用，则会由于实参依赖查找而使用 std::abs。

函数可以实现为拥有不同于 std::valarray 的返回类型。此时替换它的类型拥有下列属性：

  • 提供 std::valarray 的所有 const 成员函数。
  • 能从替换类型构造 std::valarray、std::slice_array、std::gslice_array、std::mask_array 和 std::indirect_array。
  • 所有接受一个 const std::valarray& 类型参数的函数，除了 begin() 和 end()(C++11 起)也应该接受替换类型。
  • 所有接受两个 const std::valarray& 类型参数的函数都应该接受 const std::valarray& 和替换类型的每种组合。
  • 返回类型添加不多于两层嵌套在最深层嵌套的参数类型上的模板。

可能的实现

template<class T>
valarray<T> abs(const valarray<T>& va)
{
    valarray<T> other = va;
    for (T& i : other)
        i = abs(i);
 
    return other; // 可以返回代理对象
}

示例

#include <iostream>
#include <valarray>
 
int main()
{
    std::valarray<int> v{1, -2, 3, -4, 5, -6, 7, -8};
    std::valarray<int> v2 = std::abs(v);
    for (auto n : v2)
        std::cout << n << ' ';
    std::cout << '\n';
}

输出：

1 2 3 4 5 6 7 8
