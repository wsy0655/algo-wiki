 (1) (C++11 起)
(C++20 弃用)
template< class T >

void atomic_init
    ( volatile std::atomic<T>* obj,

      typename std::atomic<T>::value_type desired ) noexcept;
 (2) (C++11 起)
(C++20 弃用)

以值 desired 初始化默认构造的原子对象 obj。此函数不是原子的：来自其他线程的并发访问，即使通过原子操作，也是数据竞争。

如果 obj 不是默认构造的，或者在同一 obj 上调用此函数两次，那么行为未定义。

参数

obj - 指向要初始化的原子对象的指针
desired - 用来初始化原子对象的值

返回值

（无）

注解

为与 C 兼容而提供此函数。如果不要求兼容性，可通过默认构造函数来初始化 std::atomic。
