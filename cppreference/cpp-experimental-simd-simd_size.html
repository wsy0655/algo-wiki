std::experimental::simd_size

< cpp‎ | experimental‎ | simd
在标头 <experimental/simd> 定义
template< class T, class Abi = simd_abi::compatible<T> >
struct simd_size;
 (并行 TS v2)

若 T 是可向量化的类型，且 is_abi_tag_v<Abi> 为 true，则提供的成员常量 value 等于 simd<T, Abi> 的宽度（元素的数量），无论 simd<T, Abi> 是否受到支持。对于任何其他类型的 T 和 Abi，不提供成员 value。

如果程序添加了 std::simd_size 或 std::simd_size_v(C++17 起) 的特化，那么行为未定义。
