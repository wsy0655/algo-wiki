template< class CharT, class Traits, class Duration >

std::basic_ostream<CharT, Traits>&
    operator<<( std::basic_ostream<CharT, Traits>& os,

                const std::chrono::sys_days& tp );
 (2) (C++20 起)

输出 tp 到流 os 中。

1) 等价于： 
return os << std::format(os.getloc(), STATICALLY-WIDEN<CharT>("{:L%F %T}"), tp);

其中 STATICALLY_WIDEN<CharT>("{:L%F %T}") 当 CharT 为 char 时为 "{:L%F %T}"，而当 CharT 为 wchar_t 时为 L"{:L%F %T}"。

此重载只有在 std::chrono::treat_as_floating_point_v<typename Duration::rep> 为 false 且 Duration(1) < std::chrono::days(1) 时才会参与重载决议。
2) 等价于 os << std::chrono::year_month_day(tp);。

返回值

os
