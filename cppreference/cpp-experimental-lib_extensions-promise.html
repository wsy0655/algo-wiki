std::experimental::promise (库基础 TS)

< cpp‎ | experimental‎ | lib extensions

此页面提及库基础 TS 所提供的带类型擦除分配器支持的 std::promise 修改版本。对于并发 TS 所提供的支持该 TS 所改进的 std::future 的 promise 版本，见 std::experimental::concurrency_v1::promise。

在标头 <experimental/future> 定义
template< class R > class promise;
 (1) (库基础 TS)
template< class R > class promise<R&>;
 (2) (库基础 TS)
template<>          class promise<void>;
 (3) (库基础 TS)

std::experimental::fundamentals_v1::promise（和 std::experimental::fundamentals_v2::promise）是 std::promise 的修改版，由库基础 TS 提供，带有对类型擦除分配器的支持。
