std::pmr::memory_resource::deallocate

< cpp‎ | memory‎ | memory resource
void deallocate( void* p,

                 std::size_t bytes,

                 std::size_t alignment = alignof(std::max_align_t) );
 (C++17 起)

解分配 p 所指向的存储。p 应当是先前在比较等于 *this 的 memory_resource 上对 allocate(bytes, alignment) 的调用所返回的值，且它指向的存储应当未被解分配。

等价于 do_deallocate(p, bytes, alignment);。

异常

不抛出任何异常。
