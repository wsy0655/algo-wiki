调用签名
template< class T, class U >

    requires /* 见下文 */
constexpr std::partial_ordering

    partial_order( T&& t, U&& u ) noexcept(/* 见下文 */);

用三路比较比较两个值，并产生 std::partial_ordering 类型的结果。

设 t 与 u 为表达式，而 T 与 U 分别代表 decltype((t)) 与 decltype((u))，那么 std::partial_order(t, u) 表达式等价于：

  • 如果 std::is_same_v<std::decay_t<T>, std::decay_t<U>> 是 true：
    • std::partial_ordering(partial_order(t, u))，如果它在不包含 std::partial_order 的声明的语境中进行重载决议的情况下是良构的表达式，
    • 否则是 std::partial_ordering(std::compare_three_way()(t, u))，如果它良构，
    • 否则是 std::partial_ordering(std::weak_order(t, u))，如果它良构。
  • 所有其他情况下，该表达式非良构，这能在出现于模板实例化的立即语境时导致代换失败。

定制点对象

名字 std::partial_order 代表一个定制点对象，它是某个字面 semiregular 类类型的 const 函数对象。 细节参见定制点对象 (CustomizationPointObject) 。
