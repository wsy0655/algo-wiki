std::chrono::local_t

< cpp‎ | chrono
在标头 <chrono> 定义
struct local_t {};
 (C++20 起)

类 local_t 是用作 std::chrono::time_point 的第一模板实参的伪时钟，用于指定指示时间点表示相对于仍未指定的时区的本地时间。local_time 支持流操作和时间点算术的全集。

时间点族

定义于命名空间 std::chrono
template<class Duration>
using local_time = std::chrono::time_point<std::chrono::local_t, Duration>;
 (C++20 起)
using local_seconds = local_time<std::chrono::seconds>;
 (C++20 起)
using local_days = local_time<std::chrono::days>;
