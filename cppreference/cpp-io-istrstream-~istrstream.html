std::istrstream::~istrstream

< cpp‎ | io‎ | istrstream
virtual ~istrstream();
 (C++98 弃用)
(C++26 移除)

销毁 std::istrstream 对象，这亦销毁成员 std::strstreambuf。

参数

（无）

注解

std::istrstream 的构造函数不以动态分配模式创建底层 std::strstreambuf，因此不会发生 std::ostrstream::~ostrstream 或 std::strstream::~strstream 可能发生的内存泄露。

示例

#include <iostream>
#include <strstream>
 
int main()
{
    {
        std::istrstream s("1.234");
        double d;
        s >> d;
        std::cout << d << '\n';
    } // 调用析构函数
}

输出：

1.234
