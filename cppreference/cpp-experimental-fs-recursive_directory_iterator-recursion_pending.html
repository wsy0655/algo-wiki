std::experimental::filesystem::recursive_directory_iterator::recursion_pending

< cpp‎ | experimental‎ | fs‎ | recursive directory iterator
bool recursion_pending() const;
 (文件系统 TS)

如果下一次自增将会迭代到 *this 当前指代的目录之内则返回 true。

在构造或自增后直接调用此函数时返回 true。可以由 disable_recursion_pending() 禁用。

参数

（无）

返回值

如果下一次自增将会迭代到当前指代的目录之内则为 true，否则为 false。

异常

不抛出。
