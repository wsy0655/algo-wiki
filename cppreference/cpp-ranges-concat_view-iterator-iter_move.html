返回对 it_ 包含的底层迭代器应用 ranges::iter_move 的结果。

设 Ref 为 concat-rvalue-reference-t ﻿<maybe-const ﻿<Const, Views>...>。等价于 return std::visit([](const auto& i) -> Ref { return ranges::iter_move(i); }, it.it_ ﻿);。

如果 it.it_.valueless_by_exception() 是 true，那么行为未定义。

此函数对常规的无限定或有限定查找不可见，而只能在 concat_view::iterator ﻿<Const> 为实参的关联类时由实参依赖查找找到。
