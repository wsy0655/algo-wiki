进行比较的函数对象。主模板在 T 类型 const 左值上调用 operator == 并对结果取反。特化 not_equal_to<void> 从实参推导函数调用运算符的形参类型（但非返回类型）。

not_equal_to 的所有特化均为 Semiregular。

template< class T, class U >

    requires EqualityComparableWith<T, U> ||
             /* std::declval<T>() == std::declval<U>()
                解析成比较指针的内建运算符 */

constexpr bool operator()(T&& t, U&& u) const;
 (2) (仅为 not_equal_to<void> 特化的成员)
1) 比较 x 与 y。等价于 return !ranges::equal_to<>{}(x, y)；
2) 比较 t 与 u。等价于 return !ranges::equal_to<>{}(std::forward<T>(t), std::forward<U>(u));。

注解

不同于 std::not_equal_to，ranges::not_equal_to 要求 == 与 != 均合法（通过 EqualityComparable 和 EqualityComparableWith 制约），而它完全用 ranges::equal_to 定义。然而，实现可以自由地直接使用 operator!=，因为这些概念要求 == 与 != 的结果一致。
