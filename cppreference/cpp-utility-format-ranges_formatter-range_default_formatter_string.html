std::range-default-formatter<std::range_format::string>, std::range-default-formatter<std::range_format::debug_string>

< cpp‎ | utility‎ | format‎ | ranges formatter
在标头 <format> 定义
template< std::range_format K, ranges::input_range R, class CharT >

    requires (K == std::range_format::string || K == std::range_format::debug_string)

struct /*range-default-formatter*/<K, R, CharT>;
 (C++23 起)
(仅用于阐述*)

用于范围类型的类模板 /*range-default-formatter*/ 进行了特化，用于当 std::format_kind<R> 为 std::range_format::string 或 std::range_format::debug_string 时将范围作为字符串或经转义字符串进行格式化。

std::range-default-formatter<std::range_format::string>::format
std::range-default-formatter<std::range_format::debug_string>::format

template< class FormatContext >
auto format( /* 见下文 */& r, FormatContext& ctx ) const -> FormatContext::iterator;

如果 ranges::input_range<const R> 为 true，则 r 的类型为 const R&。否则其类型为 R&。

令 s 为一个 std::basic_string<CharT>，如同以 std::basic_string<CharT>(std::from_range, r) 构造 s 使得 ranges::equal(s, r) 为 true。

等价于 return underlying_.format(s, ctx);。

返回输出范围的尾后迭代器。
