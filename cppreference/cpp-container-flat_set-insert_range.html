std::flat_set<Key,Compare,KeyContainer>::insert_range

< cpp‎ | container‎ | flat set

template< container-compatible-range<value_type> R >
void insert_range( R&& rg );
 (C++23 起)

向底层容器 c 中插入范围 rg 中每个元素的副本，如同以如下代码：

for (const auto& e : rg)
    c.insert(c.end(), e);

然后，按 compare 对新插入元素的范围排序。 把所得有序范围和既存元素的有序范围合并为一个有序范围。 最后，擦除每个连续等价元素组中除第一个之外的所有元素。

范围 rg 中的每个元素均恰好背解引用一次。 如果 rg 与容器重叠，则其行为未定义。

迭代器失效上的信息复制自此处

参数

rg - 容器兼容范围，即其元素可以转换为 T 的 input_range

返回值

（无）

复杂度

N + M·log(M)，其中 N 为操作前的 size() 而 M 为 ranges::distance(rg)。

注解

由于 insert_range 进行原位归并操作，这可能会分配内存。

示例

#include <iostream>
#include <flat_set>
 
void println(auto, auto const& container)
{
    for (const auto& elem : container)
        std::cout << elem << ' ';
    std::cout << '\n';
}
 
int main()
{
    auto container = std::flat_set{1, 3, 2, 4};
    const auto rg = {-1, 3, -2};
#ifdef __cpp_lib_containers_ranges
    container.insert_range(rg);
#else
    container.insert(rg.begin(), rg.end());
#endif
    println("{}", container);
}

输出：

-2 -1 1 2 3 4
