std::source_location::column

< cpp‎ | utility‎ | source location
constexpr std::uint_least32_t column() const noexcept;
 (C++20 起)

返回表示从此对象所表示的行开头算起的偏移量的实现定义值（即列号）。列号假定为从 1 开始索引。

参数

（无）

返回值

表示从此对象所表示的行开头算起的偏移量的实现定义值（即列号）。

鼓励实现在列号未知时使用 0。

示例

#include <iostream>
#include <source_location>
 
template<typename T = std::source_location>
inline void pos(const T& location = T::current())
{
    std::cout
        << "("
        << location.line()
        << ':' 
        << location.column()
        << ") ";
}
 
int main()
{
//      ↓: column #9
    pos(); std::cout << "Proxima\n";    // row #18
      pos(); std::cout << "Centauri\n"; // row #19
//        ↑: column #11
}

可能的输出：

(18:9) Proxima
(19:11) Centauri
