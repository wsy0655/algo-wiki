可能 const 限定的 zip_transform_view 的迭代器类型，由 zip_transform_view::begin 返回，以及特定情况下由 zip_transform_view::end 返回。

类型 /*iterator*/<true> 或 /*iterator*/<false> 分别将底层视图当作 const 限定或非 const 限定。
