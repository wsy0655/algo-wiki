std::atomic_flag_clear, std::atomic_flag_clear_explicit

< cpp‎ | atomic
在标头 <atomic> 定义
void atomic_flag_clear( volatile std::atomic_flag* obj ) noexcept;
 (1) (C++11 起)
void atomic_flag_clear( std::atomic_flag* obj ) noexcept;
 (2) (C++11 起)
void atomic_flag_clear_explicit( volatile std::atomic_flag* obj,
                                 std::memory_order order ) noexcept;
 (3) (C++11 起)
void atomic_flag_clear_explicit( std::atomic_flag* obj,
                                 std::memory_order order ) noexcept;
 (4) (C++11 起)

将 obj 指向的 std::atomic_flag 的状态原子地更改为清除（false）。

1,2) 内存同步定序是 std::memory_order_seq_cst。
3,4) 内存同步定序是 order。
如果 order 是 std::memory_order_consume、std::memory_order_acquire 和 std::memory_order_acq_rel 之一，那么行为未定义。

参数

obj - 指向要访问的 std::atomic_flag 的指针
order - 内存同步定序

注解

std::atomic_flag_clear 和 std::atomic_flag_clear_explicit 可以分别以 obj->clear() 和 obj->clear(order) 实现。
