std::coroutine_handle<Promise>::from_address

< cpp‎ | coroutine‎ | coroutine handle
static constexpr coroutine_handle from_address( void *addr );
 (C++20 起)

从空指针值或另一 coroutine_handle 的底层地址创建 coroutine_handle。返回值的底层地址为 addr。

若 addr 既非空指针值亦非 coroutine_handle 的底层地址，则行为未定义。若 addr 为 std::coroutine_handle<P1> 的底层地址，其中 Promise 与 P1 均非 void 且 P1 不同于 Promise，则行为亦为未定义。

此函数不对特化 std::coroutine_handle<std::noop_coroutine_promise> 声明。

参数

addr - 要导入的底层地址

返回值

底层地址为 addr 的 std::coroutine_handle<Promise> 。

注解

若 addr 不是空指针值，则它必须从先前在指代某个协程的 coroutine_handle 上调用 address() 获得。
