static bool eq( char_type a, char_type b );
 (1) (C++11 起为 constexpr)
(C++11 起为 noexcept)
static bool lt( char_type a, char_type b );
 (2) (C++11 起为 constexpr)
(C++11 起为 noexcept)

比较两个字符。

1) 比较 a 与 b 是否相等，行为等同于
  • static_cast<unsigned char>(a) == static_cast<unsigned char>(b)，如果 char_type 是 char，
  • 否则等同于 a == b。
2) 以全序的方式比较 a 是否小于 b，行为等同于
  • static_cast<unsigned char>(a) < static_cast<unsigned char>(b)，如果 char_type 是 char，
  • 否则等同于 a < b。

对字符特征的关于 X::eq 和 X::lt 的通用要求请参考字符特征 (CharTraits) 。

参数

a, b - 要比较的字符值

返回值

1) 在 a 等于 b 时返回 true，否则返回 false。
2) 在 a 小于 b 时返回 true，否则返回 false。

复杂度

常数。
