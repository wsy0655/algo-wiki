构造新的 directory_entry 对象。

1) 默认构造函数。
2) 预置的复制构造函数。
3) 预置的移动构造函数。
4,5) 以路径 p 初始化 directory_entry，并调用 refresh 更新缓存的属性。若发生错误，则不抛出重载令 directory_entry 保有默认构造的路径。

参数

p - directory_entry 将指代的文件系统对象的路径
ec - 不抛出重载中报告错误的输出形参

异常

若内存分配失败，则任何不标记为 noexcept 的重载可能抛出 std::bad_alloc 。

1-4) 抛出 std::filesystem::filesystem_error，构造时以 p 为第一路径实参并以OS 错误码为错误码实参。
5) 若 OS API 调用失败，则设置 std::error_code& 为 OS API 错误码，而未发生错误时则执行 ec.clear()。
