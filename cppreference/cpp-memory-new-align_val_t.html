std::align_val_t

< cpp‎ | memory‎ | new
在标头 <new> 定义
enum class align_val_t : std::size_t {};
 (C++17 起)

在将 new 表达式和 delete 表达式用于对齐要求大于 __STDCPP_DEFAULT_NEW_ALIGNMENT__ 的对象时，该对齐要求作为 std::align_val_t 类型的实参传递给分配/解分配函数。

注解

对齐（以 alignof 取得者）拥有 std::size_t 类型，但采用 std::size_t 作为附加参数的布置形式分配和解分配函数已经被使用了，故使用此类型以代替。
