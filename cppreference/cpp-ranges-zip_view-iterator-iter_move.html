iter_move(ranges::zip_view::iterator)

< cpp‎ | ranges‎ | zip view‎ | iterator


friend constexpr auto iter_move( const iterator& i ) noexcept(/* see below */);
 (C++23 起)

等价于：return /*tuple-transform*/(ranges::iter_move, i.current_);，其中 current_ 代表底层的持有指向所适配视图的元素的迭代器的元组式对象。

此函数对常规的无限定或有限定查找不可见，而只能在 zip_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

参数

i - 迭代器

返回值

若 *i 是左值引用则为 std::move(*i)，否则为 *i。
