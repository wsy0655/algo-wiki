std::experimental::filesystem::path::extension

< cpp‎ | experimental‎ | fs‎ | path
path extension() const;
 (文件系统 TS)

返回路径 *this 的文件名组分的扩展名。

如果路径的 filename() 组分包含一个句点 (.)，且它并非特殊文件系统元素 点 或 点点 之一，则扩展名为一最右句点开始（包括句点）直到路径名末尾的子字符串。

如果路径名位 . 或 ..，或者若 filename() 不包含 . 字符，则返回空路径。

实现可以定义额外的行为，以支持向扩展名附加额外元素（如替代数据流或分区数据集名等）的文件系统。

参数

（无）

返回值

当前路径名的扩展名，或没有扩展名时为空路径。

异常

可能会抛出由实现定义的异常。

注解

此函数所返回的扩展名包括一个句点，以使其可以区分以句点结尾的文件（函数返回 "."）和没有扩展名的文件（函数返回 ""）。

对于任何路径 p，p.stem()+p.extension() == p.filename()。

示例

#include <experimental/filesystem>
#include <iostream>
namespace fs = std::experimental::filesystem;
 
int main()
{
    std::cout << fs::path("/foo/bar.txt").extension() << '\n'
              << fs::path("/foo/bar.").extension() << '\n'
              << fs::path("/foo/bar").extension() << '\n'
              << fs::path("/foo/bar.txt/bar.cc").extension() << '\n'
              << fs::path("/foo/bar.txt/bar.").extension() << '\n'
              << fs::path("/foo/bar.txt/bar").extension() << '\n'
              << fs::path("/foo/.").extension() << '\n'
              << fs::path("/foo/..").extension() << '\n'
              << fs::path("/foo/.hidden").extension() << '\n';
}

输出：

".txt"
"."
""
".cc"
"."
""
""
""
".hidden"
