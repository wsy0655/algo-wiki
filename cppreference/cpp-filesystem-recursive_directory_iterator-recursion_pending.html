std::filesystem::recursive_directory_iterator::recursion_pending

< cpp‎ | filesystem‎ | recursive directory iterator
bool recursion_pending() const;
 (C++17 起)

若下次自增将导致迭代进入 *this 当前所指代的目录，则返回 true。

此函数在构造或自增的随后返回 true。可通过 disable_recursion_pending() 禁用递归。

参数

（无）

返回值

若下次自增将迭代进入当前指代目录，则为 true，否则为 false。

异常

不抛出。
