这些函数取三路比较的结果，并转换为六个关系运算符的结果之一。

具体而言，这些函数返回

1) cmp == 0
2) cmp != 0
3) cmp < 0
4) cmp <= 0
5) cmp > 0
6) cmp >= 0

参数

cmp - 三路比较结果

返回值

对应比较关系的 bool 结果
