std::ranges::elements_view<V,N>::iterator

< cpp‎ | ranges‎ | elements view
template< bool Const >
class /*iterator*/;
 (仅用于阐述*)

elements_view::begin 的返回类型，及 elements_view::end 在底层类型为 common_range 时的返回类型。

const 限定重载返回 /*iterator*/<true>。非 const 限定重载返回 /*iterator*/<false>。

成员类型

成员类型 定义
Base (私有) 若 Const 为 true 则为 const V，否则为 V。
(仅用于阐述的成员类型*)
iterator_concept 代表：
  • 否则若 Base 实现 random_access_range 则该类型为 std::random_access_iterator_tag。
  • 否则若 Base 实现 bidirectional_range 则该类型为 std::bidirectional_iterator_tag。
  • 否则若 Base 实现 forward_range 则该类型为 std::forward_iterator_tag。
  • 否则该类型为 std::input_iterator_tag。
iterator_category
(有条件提供)		 若 Base 不实现 forward_range 则未予定义。
  • 否则若 std::get<N>(*current_) 是右值则改类型为 std::input_iterator_tag。
  • 否则，令 C 为类型 std::iterator_traits<std::iterator_t<Base>>::iterator_category。则若 C 实现 std::derived_from<std::random_access_iterator_tag> 则该类型为 std::random_access_iterator_tag。
  • 否则该类型为 C。
