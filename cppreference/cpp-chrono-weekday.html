类 weekday 表示外推格里高利历中的星期之日。它的正常范围是 [​0​, 6]，从星期日到星期六，但它能保有范围 [​0​, 255] 中的任何值。在 std::chrono 命名空间中对一星期中的七日预定义了七个具名常量。

weekday 是可平凡复制 (TriviallyCopyable) 的标准布局类型 (StandardLayoutType) 。

示例

#include <chrono>
#include <iostream>
 
int main()
{
    std::chrono::weekday x{42 / 13};
    std::cout << x++ << '\n';
    std::cout << x << '\n';
    std::cout << ++x << '\n';
}

输出：

Wed
Thu
Fri
