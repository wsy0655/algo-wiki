std::experimental::swap(std::experimental::propagate_const)

< cpp‎ | experimental‎ | propagate const
template< class T >

constexpr void swap( std::experimental::propagate_const<T>& lhs,

                     std::experimental::propagate_const<T>& rhs ) noexcept(/* 见下文 */);
 (库基础 TS v2)

为 std::experimental::propagate_const 特化 swap 算法。交换 lhs 与 rhs 的指针。等价于 lhs.swap(rhs)。

此重载只有在std::is_swappable_v<T> 为 true时才会参与重载决议。

 (库基础 TS v3)

参数

lhs, rhs - 要交换内容的 propagate_const

返回值

（无）

异常

noexcept 说明：  
noexcept(noexcept(lhs.swap(rhs)))

复杂度

常数。

参阅

swap
 交换两个对象的值
(函数模板)
swap
 交换被包装指针
(公开成员函数)
