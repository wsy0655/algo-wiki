std::ranges::subrange<I,S,K>::advance

< cpp‎ | ranges‎ | subrange
constexpr subrange& advance( std::iter_difference_t<I> n );
 (C++20 起)

自增或自减 begin_：

  • 如果 I 实现了 bidirectional_iterator，并且 n < 0 是 true，那么 begin_ 会自减 -n 个元素。
等价于：ranges::advance(begin_ ﻿, n);
if constexpr (StoreSize ﻿)
    size_ += to-unsigned-like ﻿(-n);
return *this;。
  • 否则，begin_ 会自增 n 个元素，或者直到抵达 end_ ﻿。
等价于：auto d = n - ranges::advance(begin_ ﻿, n, end_ ﻿);
if constexpr (StoreSize ﻿)
    size_ -= to-unsigned-like ﻿(d);
return *this;。


根据 ranges::advance 的前提条件，如果 n < 0 是 true 且 begin_ 无法自减 -n 个元素，那么行为未定义。

参数

n - 迭代器上的最大自增次数

返回值

*this

示例

#include <algorithm>
#include <array>
#include <iostream>
#include <iterator>
#include <ranges>
 
void print(auto name, auto const sub)
{
    std::cout << name << ".size() == " << sub.size() << "; { ";
    std::ranges::for_each(sub, [](int x) { std::cout << x << ' '; });
    std::cout << "}\n";
};
 
int main()
{
    std::array arr{1, 2, 3, 4, 5, 6, 7};
    std::ranges::subrange sub{std::next(arr.begin()), std::prev(arr.end())};
    print("1) sub", sub);
    print("2) sub", sub.advance(3));
    print("3) sub", sub.advance(-2));
}

输出：

1) sub.size() == 5; { 2 3 4 5 6 }
2) sub.size() == 2; { 5 6 }
3) sub.size() == 4; { 3 4 5 6 }
