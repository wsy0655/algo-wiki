std::span<T,Extent>::empty

< cpp‎ | container‎ | span
constexpr bool empty() const noexcept;
 (C++20 起)

检查 span 是否为空。等价于 return size() == 0;。

返回值

span 为空时返回 true；否则返回 false。

示例

#include <iomanip>
#include <iostream>
#include <span>
 
int main()
{
    std::span<const char> span{"ABCDEF"};
 
    while (!span.empty())
    {
        std::cout << std::quoted(span.data()) << '\n';
        span = span.last(span.size() - 1);
    }
}

输出：

"ABCDEF"
"BCDEF"
"CDEF"
"DEF"
"EF"
"F"
""
