std::make_error_condition(std::errc)

< cpp‎ | error‎ | errc
在标头 <system_error> 定义
std::error_condition make_error_condition( std::errc e ) noexcept;
 (C++11 起)

为 errc 值 e 创建一个错误条件。设置错误码为 int(e)，错误类别为 std::generic_category。

参数

e - 标准错误码

返回值

e 的错误条件。

示例

#include <iostream>
#include <string>
#include <system_error>
 
int main()
{
    auto err = std::make_error_condition(std::errc::invalid_argument);
    std::cout << err.message() << '\n';
}

可能的输出：

Invalid argument
