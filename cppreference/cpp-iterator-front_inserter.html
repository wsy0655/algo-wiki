std::front_inserter

< cpp‎ | iterator
在标头 <iterator> 定义
template< class Container >
std::front_insert_iterator<Container> front_inserter( Container& c );
 (C++20 起为 constexpr)

front_inserter 是便利函数模板，为容器 c 构造 std::front_insert_iterator，并拥有从实参类型推导的类型。

参数

c - 支持 push_front 操作的容器

返回值

能用于向容器 c 开头添加元素的 std::front_insert_iterator。

可能的实现

template<class Container>
std::front_insert_iterator<Container> front_inserter( Container& c )
{
    return std::front_insert_iterator<Container>(c);
}

示例

#include <algorithm>
#include <deque>
#include <iostream>
#include <iterator>
#include <vector>
 
int main()
{
    std::vector<int> v{1, 2, 3, 4, 5};
    std::deque<int> d;
    std::copy(v.begin(), v.end(), std::front_inserter(d));
    for (int n : d)
        std::cout << n << ' ';
    std::cout << '\n';
}

输出：

5 4 3 2 1
