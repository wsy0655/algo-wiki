比较 lhs 与 rhs 的底层迭代器。

1) 此重载只有在 lhs.base() == rhs.base() 良构并且可转换到 bool 时才会参与重载决议。
3-6) 此重载只有在 lhs.base() < rhs.base() 良构并且可转换到 bool 时才会参与重载决议。

!= 运算符从 operator== 运算符合成。

 (C++20 起)

参数

lhs, rhs - 要比较的迭代器适配器

返回值

1) lhs.base() == rhs.base()
2) lhs.base() != rhs.base()
3) lhs.base() > rhs.base()
4) lhs.base() >= rhs.base()
5) lhs.base() < rhs.base()
6) lhs.base() <= rhs.base()
7) rhs.base() <=> lhs.base()

示例

#include <cassert>
#include <iterator>
 
int main()
{
    int a[]{9, 8, 7, 6};
    //            │  └───── x, y
    //            └──────── z
 
    // “x” 与 “y” 相等，但 “x” 大于 “z”
    std::move_iterator<int*>
        x{std::end(a) - 1},
        y{std::end(a) - 1},
        z{std::end(a) - 2};
 
    // 双路比较
    assert(  x == y );
    assert(!(x != y));
    assert(!(x <  y));
    assert(  x <= y );
    assert(!(x == z));
    assert(  x != z );
    assert(!(x <  z));
    assert(!(x <= z));
 
    // 三路比较
    assert(  x <=> y == 0 );
    assert(!(x <=> y <  0));
    assert(!(x <=> y >  0));
    assert(!(x <=> z == 0));
    assert(!(x <=> z <  0));
    assert(  x <=> z >  0 );
}
