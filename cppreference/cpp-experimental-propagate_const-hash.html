std::hash(std::experimental::propagate_const)

< cpp‎ | experimental‎ | propagate const
在标头 <experimental/propagate_const> 定义
template< class T > struct hash<std::experimental::propagate_const<T>>;
 (库基础 TS v2)

std::hash 对 std::experimental::propagate_const<T> 的类模板特化允许用户获得 std::experimental::propagate_const<T> 类型对象的散列。

对于给定的 std::experimental::propagate_const<T> p，此特化确保 std::hash<std::experimental::propagate_const<T>>()(p) == std::hash<T>()(p.t_)，其中 p.t_ 是 p 所包装的指针式对象。
