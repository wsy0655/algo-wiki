1,2) 将 dm.count() 月加到 ymdl 所表示的日期。结果与 std::chrono::year_month(ymdl.year(), ymdl.month()) + dm 拥有相同的 year() 和 month()。
3,4) 将 dy.count() 年加到 ymdl 所表示的日期。结果等价于 std::chrono::year_month_day_last(ymdl.year() + dy, ymdl.month_day_last())。
5) 从 ymdl 所表示的日期减去 dm.count() 月。等价于 ymdl + -dm。
6) 从 ymdl 所表示的日期减去 dy.count() 年。等价于 ymdl + -dy。

对于能转换到 std::chrono::years 与 std::chrono::months 两者的时长，若调用本会有歧义则偏好 years 重载 (3,4,6)。

示例

#include <cassert>
#include <chrono>
#include <iostream>
 
int main()
{
    auto ymdl{11/std::chrono::last/2020};
    std::cout << ymdl << '\n';
 
    ymdl = std::chrono::years(10) + ymdl;
    std::cout << ymdl << '\n';
    assert(ymdl == std::chrono::day(30)/
                   std::chrono::November/
                   std::chrono::year(2030));
 
    ymdl = ymdl - std::chrono::months(6);
    std::cout << ymdl << '\n';
    assert(ymdl == std::chrono::day(31)/
                   std::chrono::May/
                   std::chrono::year(2030));
}
