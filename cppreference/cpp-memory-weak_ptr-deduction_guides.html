std::weak_ptr 的推导指引

< cpp‎ | memory‎ | weak ptr


在标头 <memory> 定义
template<class T>
weak_ptr(std::shared_ptr<T>) -> weak_ptr<T>;
 (C++17 起)

为 std::weak_ptr 提供一个推导指引以处理隐式推导指引所遗漏的特殊情况。

示例

#include <memory>
 
int main()
{
    auto p = std::make_shared<int>(42);
    std::weak_ptr w{p};    // 用于此情况的显式推导指引
}
