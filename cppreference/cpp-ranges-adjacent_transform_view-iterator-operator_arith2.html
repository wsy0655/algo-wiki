1,2) 等价于 return /*iterator*/(*i.parent_, i.inner_ + n);。

这些函数对常规的无限定或有限定查找不可见，而只能在 adjacent_transform_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

参数

i, j - 迭代器
n - 相对于当前位置的偏移量

返回值

1,2) 增加后的迭代器。
3) 减少后的迭代器。
4) 给定迭代器之间的距离。
