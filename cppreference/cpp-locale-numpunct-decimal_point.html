std::numpunct<CharT>::decimal_point, do_decimal_point

< cpp‎ | locale‎ | numpunct
在标头 <locale> 定义
public:
char_type decimal_point() const;
 (1)
protected:
virtual char_type do_decimal_point() const;
 (2)
1) 公开成员函数，调用最终派生类的成员函数 do_decimal_point。
2) 返回要用作整数和小数部分间的小数分隔符的字符。

返回值

用作小数分隔符的 char_type 类型值。std::numpunct 的标准特化返回 '.' 和 L'.'。

示例

#include <iostream>
#include <locale>
 
struct slash : std::numpunct<char>
{
    char do_decimal_point() const { return '/'; }  // 以斜杠分隔
};
 
int main()
{
    std::cout.precision(10);
    std::cout << "默认本地环境: " << 1234.5678 << '\n';
    std::cout.imbue(std::locale(std::cout.getloc(), new slash));
    std::cout << "带有修改的 numpunct 的本地环境: " << 1234.5678 << '\n';
}

输出：

默认本地环境: 1234.5678
带有修改的 numpunct 的本地环境: 1234/5678
