C++ 关键词：consteval (C++20 起)

< cpp‎ | keyword


用法

  • consteval 声明说明符
  • 显式指定函数调用运算符为立即函数的 lambda-声明符
  • consteval if 语句
 (C++23 起)

注解

功能特性测试宏 值 标准 功能特性

202211L (C++23)
(DR20)		 使 consteval 向上传播
