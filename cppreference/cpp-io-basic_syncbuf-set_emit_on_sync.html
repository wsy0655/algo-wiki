std::basic_syncbuf<CharT,Traits,Allocator>::set_emit_on_sync

< cpp‎ | io‎ | basic syncbuf
void set_emit_on_sync( bool b ) noexcept;

更改当前的同步时发射（emit-on-sync）策略。

值 false（默认值）指示任何冲入都将被延迟到对 emit 的调用。

值 true 使得立即应用冲洗。

参数

b - 同步时发射策略的新值
