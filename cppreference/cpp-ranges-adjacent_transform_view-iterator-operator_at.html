返回位于指定位置的元素。

令 parent_ 与 inner_ 为迭代器的数据成员。等价于：

return apply([&](const auto&... iters) -> decltype(auto)
             {
                return invoke(*parent_->fun_, iters[n]...);
             },
             inner_.current_);

参数

n - 相对于当前位置的位移

返回值

相对于当前位置移动 n 位后的元素。
