std::numeric_limits<T>::min_exponent

< cpp‎ | types‎ | numeric limits
static const int min_exponent;
 (C++11 前)
static constexpr int min_exponent;
 (C++11 起)

std::numeric_limits<T>::min_exponent 的值是满足 \(\scriptsize r^{n-1}\)rn-1
 是浮点类型 T 合法正规值的最低负数 n，其中 r 是 std::numeric_limits<T>::radix。

示例

演示对于 float 类型的 min_exponent、min_exponent10、min() 及 radix 的关系：

#include <iostream>
#include <limits>
 
int main()
{
    std::cout << "min() = " << std::numeric_limits<float>::min() << '\n'
              << "min_exponent10 = " << std::numeric_limits<float>::min_exponent10 << '\n'
              << std::hexfloat << '\n'
              << "min() = " << std::numeric_limits<float>::min() << '\n'
              << "min_exponent = " << std::numeric_limits<float>::min_exponent << '\n';
}

输出：

min() = 1.17549e-38
min_exponent10 = -37
 
min() = 0x1p-126
min_exponent = -125
