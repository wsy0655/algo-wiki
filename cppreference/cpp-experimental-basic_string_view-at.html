std::experimental::basic_string_view<CharT,Traits>::at

< cpp‎ | experimental‎ | basic string view
constexpr const_reference at(size_type pos) const;
 (库基础 TS)

返回到指定位置 pos 的字符的引用。实施边界检查，非法访问时抛出 std::out_of_range 类型的异常。

参数

pos - 要返回字符的位置

返回值

到所请求字符的引用。

异常

当 pos >= size() 时抛出 std::out_of_range。

复杂度

常数。

参阅

operator[]
 访问指定字符
(公开成员函数)
