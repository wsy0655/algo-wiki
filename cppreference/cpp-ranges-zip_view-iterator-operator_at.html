获取由处于当前位置的给定偏移量处的被指向底层元素所构成的 std::tuple。

等价于：

return /*tuple-transform*/([&]<class I>(I& i) -> decltype(auto) {
           return i[iter_difference_t<I>(n)];
       }, current_);

参数

n - 相对于当前位置的位置

返回值

所获得的元组式元素。
