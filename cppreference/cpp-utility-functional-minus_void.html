std::minus<void>

< cpp‎ | utility‎ | functional
在标头 <functional> 定义
template<>
class minus<void>;
 (C++14 起)

std::minus<void> 是会推导形参类型和返回类型的 std::minus 特化。

成员类型

类型 定义
is_transparent 未指定

成员函数

operator()
 返回两个实参的差
(公开成员函数)

std::minus<void>::operator()

template< class T, class U >

constexpr auto operator()( T&& lhs, U&& rhs ) const

    -> decltype(std::forward<T>(lhs) - std::forward<U>(rhs));

返回 lhs 与 rhs 的差。

参数

lhs, rhs - 要相减的值

返回值

std::forward<T>(lhs) - std::forward<U>(rhs)。

示例

#include <complex>
#include <functional>
#include <iostream>
 
int main()
{
    auto complex_minus = std::minus<void>{}; // 可以省略 “void”
    constexpr std::complex<int> z(4, 2);
    std::cout << complex_minus(z, 1) << '\n';
    std::cout << (z - 1) << '\n';
}

输出：

(3,2)
(3,2)
