std::relation

< cpp‎ | concepts
在标头 <concepts> 定义
template< class R, class T, class U >

concept relation =
    std::predicate<R, T, T> && std::predicate<R, U, U> &&

    std::predicate<R, T, U> && std::predicate<R, U, T>;
 (1) (C++20 起)

概念 relation<R, T, U> 指定 R 在由 T 或 U 编码类型和值类别的表达式上的定义了一种二元关系。
