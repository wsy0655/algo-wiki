有特殊含义的 C++ 标识符: module (C++20 起)

< cpp


用法

  • module 声明：声明当前翻译单元是 模块单元
  • 开始模块单元的全局模块片段
  • 开始模块单元的私有模块片段

注解

功能特性测试宏 值 标准 功能特性
__cpp_modules 201907L (C++20) 模块——核心语言支持

示例

module;            // 开始全局模块片段
 
#include <string>
 
export module foo; // 结束全局模块片段
                   // 为具名模块 foo 声明主模块接口单元
                   // 开始模块单元范围
 
export std::string f();
 
module : private;  // 结束模块接口单元中可以影响其他翻译单元行为的部分
                   // 开始私有模块片段
 
std::string f()
{
    return "foo";
}

参阅

  • import, export
