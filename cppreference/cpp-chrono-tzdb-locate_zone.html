std::chrono::tzdb::locate_zone

< cpp‎ | chrono‎ | tzdb
const std::chrono::time_zone* locate_zone( std::string_view tz_name ) const;
 (C++20 起)

获得指向此数据库中表示 tz_name 所指代的时区的 std::chrono::time_zone 的指针。若 p 是所返回的指针，则 p->name() == tz_name 或在此数据库中有一个 std::chrono::time_zone_link l 使得 p->name() == l.target() && l.name() == tz_name。

参数

tz_name - 时区名称

返回值

指向此数据库中表示 tz_name 所指代的时区的 std::chrono::time_zone 的指针。

异常

若找不到这种 time_zone 则为 std::runtime_error。
