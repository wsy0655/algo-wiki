std::experimental::filesystem::directory_options

< cpp‎ | experimental‎ | fs
在标头 <experimental/filesystem> 定义
enum class directory_options {

    none,
    follow_directory_symlink,
    skip_permission_denied

};
 (文件系统 TS)

此类型标识控制 directory_iterator 和 recursive_directory_iterator 的行为的可用选项。

directory_options 满足位掩码类型 (BitmaskType) 的规定（这表示按位运算符 operator&、operator|、operator^、operator~、operator&=、operator|= 和 operator^= 对此类型有定义）。

成员常量

成员常量 值 含义
none ​0​ （默认）跳过目录符号链接，视权限拒绝为错误。
follow_directory_symlink 1 跟随而非跳过目录符号链接。
skip_permission_denied 2 跳过会导致权限拒绝错误的目录。
