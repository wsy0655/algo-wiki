thrd_equal

< c‎ | thread
在标头 <threads.h> 定义
int thrd_equal( thrd_t lhs, thrd_t rhs );
 (C11 起)

检查 lhs 与 rhs 是否表示同一线程。

参数

lhs, rhs - 要比较的线程

返回值

若 lhs 与 rhs 表示同一线程则为非零值，否则为 ​0​。
