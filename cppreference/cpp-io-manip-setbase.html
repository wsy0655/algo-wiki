std::setbase

< cpp‎ | io‎ | manip
在标头 <iomanip> 定义
/*未指定*/ setbase( int base );

设置流的数值底。用于表达式 out << setbase(base) 或 in >> setbase(base) 时，取决于 base 的值，更改流 out 或 in 的 basefield 标志：

  • 值 16 设置 basefield 为 std::ios_base::hex。
  • 值 8 设置 std::ios_base::oct。
  • 值 10 设置 std::ios_base::dec。

8、10 或 16 以外的 base 值会重置 basefield 为零，这对应十进制输出和依赖前缀的输入。

参数

base - basefield 的新值

返回值

一个满足以下条件但未指定类型的对象：

  • 如果 out 是具有 std::basic_ostream<CharT, Traits> 类型的对象，那么表达式 out << setbase(base)：
    • 具有 std::basic_ostream<CharT, Traits>& 类型
    • 值为 out
    • 行为如同调用 f(out, base)
  • 如果 in 是具有 std::basic_istream<CharT, Traits> 类型的对象，那么表达式 in >> setbase(base)：
    • 具有 std::basic_istream<CharT, Traits>& 类型
    • 值为 in
    • 行为如同调用 f(in, base)

其中函数 f 定义如下：

void f(std::ios_base& str, int base)
{
    // 设置 basefield
    str.setf(base == 8 ? std::ios_base::oct :
        base == 10 ? std::ios_base::dec :
        base == 16 ? std::ios_base::hex :
        std::ios_base::fmtflags(0), std::ios_base::basefield);
}

示例

#include <iomanip>
#include <iostream>
#include <sstream>
 
int main()
{
    std::cout << "解析字符串 \"10 0x10 010\"\n";
 
    int n1, n2, n3;
    std::istringstream s("10 0x10 010");
 
    s >> std::setbase(16) >> n1 >> n2 >> n3;
    std::cout << "以十六进制解析：" << n1 << ' ' << n2 << ' ' << n3 << '\n';
 
    s.clear();
    s.seekg(0);
 
    s >> std::setbase(0) >> n1 >> n2 >> n3;
    std::cout << "依赖前缀的解析：" << n1 << ' ' << n2 << ' ' << n3 << '\n';
 
    std::cout << "以十六进制输出：" << std::setbase(16)
              << std::showbase << n1 << ' ' << n2 << ' ' << n3 << '\n';
}

输出：

解析字符串 "10 0x10 010"
以十六进制解析：16 16 16
依赖前缀的解析：10 16 8
以十六进制输出：0xa 0x10 0x8
