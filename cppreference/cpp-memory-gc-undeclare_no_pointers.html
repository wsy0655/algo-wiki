std::undeclare_no_pointers

< cpp‎ | memory
在标头 <memory> 定义
void undeclare_no_pointers( char *p, std::size_t n );
 (C++11 起)
(C++23 移除)

解除注册先前以 std::declare_no_pointers() 注册的范围。

参数

p - 指向先前以 std::declare_no_pointers 注册的范围起始的指针
n - 范围的字节数，与先前用于 std::declare_no_pointers 的值相同

返回值

（无）

异常

不抛出。

参阅

declare_no_pointers
(C++11)(C++23 移除)
 声明该内存区域不含可追踪指针
