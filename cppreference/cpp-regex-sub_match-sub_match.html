std::sub_match<BidirIt>::sub_match

< cpp‎ | regex‎ | sub match
constexpr sub_match();
 (C++11 起)

默认构造 std::sub_match。设置 matched 成员为 false 并值初始化继承的成员 first 与 second。

这是仅有的可公开访问且有定义的构造函数。

示例

#include <cassert>
#include <regex>
 
int main()
{
    std::sub_match<const char*> s;
    assert(!s.matched);
}
