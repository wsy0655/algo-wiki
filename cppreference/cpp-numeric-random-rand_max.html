RAND_MAX

< cpp‎ | numeric‎ | random


在标头 <cstdlib> 定义
#define RAND_MAX /* 由实现定义 */

展开成等于函数 std::rand 返回的最大值的整数常量表达式。此值依赖实现。保证此值至少为 32767。

示例

#include <climits>
#include <cstdlib>
#include <ctime>
#include <iostream>
 
int main()
{
    // 使用当前时间为随机数生成器的种子
    std::srand(std::time(NULL));
 
    std::cout << "RAND_MAX: " << RAND_MAX << '\n'
              << "INT_MAX: " << INT_MAX << '\n'
              << "[0,1] 中的随机值: "
              << static_cast<double>(std::rand()) / RAND_MAX << '\n';
}

可能的输出：

RAND_MAX: 2147483647
INT_MAX: 2147483647
[0,1] 中的随机值: 0.618608
