注意：在 Unicode 中，ASCII 字符区块被称作 U+0000..U+007F 基础拉丁（Basic Latin）。

示例

#include <iostream>
 
int main()
{
    std::cout << "Printable ASCII [32..126]:\n";
    for (char c{' '}; c <= '~'; ++c)
        std::cout << c << ((c + 1) % 32 ? ' ' : '\n');
    std::cout << '\n';
}
