operator==(std::layout_stride::mapping)

< cpp‎ | container‎ | mdspan‎ | layout stride‎ | mapping


template< class OtherMapping >

friend constexpr bool operator==( const mapping& lhs,

                                  const OtherMapping& rhs ) noexcept;
 (C++23 起)

基于底层尺度和步长，比较相同布局类型的两个布局映射。

如果下列条件均为 true，那么布局映射 lhs 和 rhs 比较为相等：

  • lhs.extents() 和 rhs.extents() 相等，
  • OFFSET(rhs) 等于 ​0​，并且
  • 对于范围 [​0​, lhs.extents().rank()) 中的每个秩索引 r，lhs.stride(r) 都等于 rhs.stride(r)

其中 OFFSET(rhs)

  • 等于 rhs()，如果 rhs.extents().rank() 为 ​0​，
  • 否则等于 ​0​，如果多维索引空间 rhs.extents() 的大小为 ​0​，
  • 否则等于 rhs(z...)，其中整数包 z 是 rhs.extents() 中的多维索引，且 z 的每个元素都等于 ​0​。

此重载只有在以下约束条件均被满足：

  • layout-mapping-alike <OtherMapping> 被满足时才会参与重载决议：
  • rank_ == OtherMapping::extents_type::rank() 为 true，并且
  • OtherMapping::is_always_strided() 为 true。

（rank_ 是 std::layout_stride::mapping 中定义的仅用于阐释的静态成员常量。）

If OtherMapping does not meet the 布局映射 (LayoutMapping) requirements, the behavior is undefined.

参数

lhs, rhs - 要比较尺度和步长的布局映射

返回值

如果 lhs 和 rhs 根据上文规定相等时返回 true，否则返回 false。
