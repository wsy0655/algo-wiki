std::bit_width

< cpp‎ | numeric
在标头 <bit> 定义
template< class T >
constexpr int bit_width( T x ) noexcept;
 (C++20 起)

若 x 非零，则计算存储值 x 所需的位数，即 \(1 + \lfloor \log_2(x) \rfloor\)1 + floor(log2(x))。若 x 为零，则返回零。

此重载只有在 T 为无符号整数类型（即 unsigned char、unsigned short、unsigned int、unsigned long、unsigned long long 或扩展无符号整数类型）时才会参与重载决议。

返回值

若 x 为零则为零；否则为一加 x 的 2 为底的对数，舍弃任何小数部分。

注解

此函数等价于 return std::numeric_limits<T>::digits - std::countl_zero(x);。

功能特性测试宏 值 标准 功能特性
__cpp_lib_int_pow2 202002L (C++20) 2 的整数次幂运算

示例

#include <bit>
#include <bitset>
#include <iostream>
 
int main()
{
    for (unsigned x{}; x != 010; ++x)
        std::cout << "bit_width( "
                  << std::bitset<4>{x} << " ) = "
                  << std::bit_width(x) << '\n';
}

输出：

bit_width( 0000 ) = 0
bit_width( 0001 ) = 1
bit_width( 0010 ) = 2
bit_width( 0011 ) = 2
bit_width( 0100 ) = 3
bit_width( 0101 ) = 3
bit_width( 0110 ) = 3
bit_width( 0111 ) = 3
