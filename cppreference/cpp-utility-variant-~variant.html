std::variant<Types...>::~variant

< cpp‎ | utility‎ | variant
~variant();
 (C++17 起)
(C++20 起为 constexpr)

若 valueless_by_exception() 为 true，则不做任何事。否则销毁当前所含对象。

若对于 Types... 中的所有 T_i，std::is_trivially_destructible_v<T_i> 均为 true，则此析构函数为平凡。

注解

功能特性测试宏 值 标准 功能特性
__cpp_lib_variant 202106L (C++20)
(DR)		 完全 constexpr 的 std::variant

示例

#include <cstdio>
#include <variant>
 
int main()
{
    struct X { ~X() { puts("X::~X();"); } };
    struct Y { ~Y() { puts("Y::~Y();"); } };
 
    {
        puts("进入块 #1");
        std::variant<X,Y> var;
        puts("离开块 #1");
    }
 
    {
        puts("进入块 #2");
        std::variant<X,Y> var{ std::in_place_index_t<1>{} }; // 构造 var(Y)
        puts("离开块 #2");
    }
}

输出：

进入块 #1
离开块 #1
X::~X();
进入块 #2
离开块 #2
Y::~Y();
