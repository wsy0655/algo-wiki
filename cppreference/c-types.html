定宽整数类型 (C99 起)

数值极限

注解

true 与 false 的类型为 int 而非 _Bool。

程序可以解除定义，并且可以在之后重新定义宏 bool、true 与 false。然而这种能力是一项弃用的特性。

 (C99 起)
(C23 前)

true 与 false 的类型为 bool。bool、_Bool、true 或 false 的任何一个是否实现为预定义宏是未指定的。

若 bool、true 或 false（但不含 _Bool）被定义为预定义宏，则程序可以解除定义，并且可以在之后重新定义它。

 (C23 起)

示例

#include <stdalign.h>
#include <stdbool.h>
#include <stdio.h>
 
int main(void)
{
    printf("%d %d %d\n", true && false, true || false, !false);
    printf("%d %d\n", true ^ true, true + true);
    printf("%zu\n", alignof(short));
}

可能的输出：

0 1 1
0 2
2
