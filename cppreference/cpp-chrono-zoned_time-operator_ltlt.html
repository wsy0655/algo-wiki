std::chrono::operator<<(std::chrono::zoned_time)

< cpp‎ | chrono‎ | zoned time
在标头 <chrono> 定义
template< class CharT, class Traits, class Duration, class TimeZonePtr >

std::basic_ostream<CharT, Traits>&
    operator<<( std::basic_ostream<CharT, Traits>& os,

                const std::chrono::zoned_time<Duration, TimeZonePtr>& tp );
 (C++20 起)

如同用 std::format(os, fmt, tp) 来输出 tp 到流 os，其中 fmt 当 CharT 为 char 时为 "{:L%F %T %Z}"，或当 CharT 为 wchar_t 时为 L"{:L%F %T %Z}"。

参数

os - 输出流
tp - 要输出的 zoned_time

返回值

os
