std::experimental::pmr::null_memory_resource

< cpp‎ | experimental
在标头 <experimental/memory_resource> 定义
memory_resource* null_memory_resource() noexcept;
 (库基础 TS)

返回指向不进行任何分配的 memory_resource 的指针。

返回值

返回指向静态存储器对象的指针 p，其类型派生于 std::experimental::pmr::memory_resource 并具有下列属性：

  • 其 allocate() 函数总是抛出 std::bad_alloc；
  • 其 deallocate() 函数没有效果；
  • 对于任何 memory_resource r，p->is_equal(r) 返回 &r == p。

每次调用此函数均返回相同的值。
