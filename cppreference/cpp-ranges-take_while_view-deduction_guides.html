std::ranges::take_while_view 的推导指引

< cpp‎ | ranges‎ | take while view


在标头 <ranges> 定义
template< class R, class Pred >
take_while_view( R&&, Pred ) -> take_while_view<views::all_t<R>, Pred>;
 (C++20 起)

为 std::ranges::take_while_view 提供推导指引以允许从 range 与谓词推导。

若 R 为 view，则推导的类型为 take_while_view<R>；否则推导的类型通常为 take_while_view<ranges::ref_view<R>>。
