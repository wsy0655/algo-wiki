std::fpos<State>::state

< cpp‎ | io‎ | fpos
State state() const;
 (1)
void state( State st );
 (2)

管理文件位置状态。

1) 返回文件位置状态的值。
2) 以 st 的值替换文件位置状态。

对于 std::fpos 用于标准库的特化，State 始终是 std::mbstate_t。

参数

st - 状态的新值

返回值

1) fpos 状态的当前值
2) （无）

示例

#include <cwchar>
#include <iostream>
#include <sstream>
 
int main()
{
    std::istringstream s("test");
    std::mbstate_t st = s.tellg().state();
 
    if (std::mbsinit(&st))
        std::cout << "流处于初始迁移状态\n";
}

输出：

流处于初始迁移状态
