std::ctype<char>::classic_table

< cpp‎ | locale‎ | ctype char
在标头 <locale> 定义
static const mask* classic_table() throw();
 (C++11 前)
static const mask* classic_table() noexcept;
 (C++11 起)

返回匹配最小 "C" 本地环境所用分类的分类表。

参数

（无）

返回值

指向分类表（大小为 std::ctype<char>::table_size 的数组）中首元素的指针。

注意

默认构造的 std::ctype<char> 刻面用此表分类。
