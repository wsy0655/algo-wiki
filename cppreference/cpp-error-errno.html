errno

< cpp‎ | error


在标头 <cerrno> 定义
#define errno /* 由实现定义 */

errno 是用于错误指示的预处理器宏。它会展开成一个静态(C++11 前)线程局部(C++11 起)的 int 类型的可修改左值。

数个标准库函数通过向 errno 写入正整数来指示错误。errno 的值通常被设置为 <cerrno> 所列出的以字母 E 起始，后随大写字母或数字的宏常量。

程序启动时 errno 的值是 ​0​，而且尽管库函数可以写入正整数到 errno（不管是否有错误发生），库函数决不会将 ​0​ 存储到 errno。

示例

#include <cerrno>
#include <clocale>
#include <cmath>
#include <cstring>
#include <iostream>
 
int main()
{
    const double not_a_number = std::log(-1.0);
    std::cout << not_a_number << '\n';
 
    if (errno == EDOM)
    {
        std::cout << "log(-1) 失败：" << std::strerror(errno) << '\n';
        std::setlocale(LC_MESSAGES, "de_DE.utf8");
        std::cout << "德语错误信息：" << std::strerror(errno) << '\n';
    }
}

可能的输出：

nan
log(-1) 失败：Numerical argument out of domain
德语错误信息：Das numerische Argument ist ausserhalb des Definitionsbereiches
