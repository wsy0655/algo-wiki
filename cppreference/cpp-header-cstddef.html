标准库标头 <cstddef>

< cpp‎ | header


此标头原作为 <stddef.h> 存在于 C 标准库。

此头文件是工具库的一部分。

宏
NULL
 实现定义的空指针常量
(宏常量)
offsetof
 从标准布局类型的起始到其指定成员的字节偏移量
(宏函数)

类型
size_t
 sizeof 运算符返回的无符号整数类型
(typedef)
ptrdiff_t
 两个指针相减返回的有符号整数类型
(typedef)
nullptr_t
(C++11)
 空指针字面量 nullptr 的类型
(typedef)
max_align_t
(C++11)
 具有不小于任何基础类型的内存对齐需求的平凡类型
(typedef)
byte
(C++17)
 字节类型
(枚举)

函数
to_integer
(C++17)
 转换 std::byte 为整数
(函数模板)
