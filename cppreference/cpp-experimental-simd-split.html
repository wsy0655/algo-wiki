分割输入的 simd 或 simd_mask 对象到多个 simd 或 simd_mask 对象中。

1-2) 分割输入对象到潜在地类型不同的对象（例如拥有不同大小的类型）中。这些函数仅若 Sizes... 求和为 v.size() 才参与重载决议。
3-4) 分割输入对象到对象的数组中。这些函数不参与重载决议，除非
  • 对于为 simd 或 simd_mask 的 v，数组元素必须分别拥有 simd 或 simd_mask 类型；
  • simd_size_v<typename V::value_type, Abi> 是 V::size() 的整数倍。
5,6) 分割输入对象到对象的数组中。除非 simd_size_v<T, A> 是 N 的整数倍，否则这些函数不参与重载决议。

参数

v - 输入的 simd 或 simd_mask 对象

返回值

1,2) 含有分割的 simd 或 simd_mask 对象的元组。
3-6) 含有分割的 simd 或 simd_mask 对象的数组。
