std::ranges::views::values, std::ranges::values_view

在标头 <ranges> 定义
template< class R >

 (1) (C++20 起)

    inline constexpr auto values = ranges::elements<1>;

}
 (2) (C++20 起)

接收元组式值（例如 std::tuple 或 std::pair）的 view，并产生以被适配的视图的值类型的第二个元素为值类型 ﻿的视图。

1) ranges::elements_­view<R, 1> 的别名。
2) 范围适配器对象 (RangeAdaptorObject) （也是 范围适配器闭包对象 (RangeAdaptorClosureObject) ）。表达式 views​::values(e) 表达式对任何适合的子表达式 e 表达式等价于 values_­view<views​::​all_­t<decltype((e))>>{e}。

注解

values_view 能用于从关联容器提取值，例如

std::map<int, std::string> map{{1, "alpha"}, {2, "beta"}};
for (auto const& value : std::views::values(map))
    std::cout << value << ' ';
// 打印: alpha beta

示例

#include <iostream>
#include <map>
#include <ranges>
 
int main()
{
    const auto list = {std::pair{1, 11.1}, {2, 22.2}, {3, 33.3}};
    std::cout << "list 中的 pair::second 值: ";
    for (double value : list | std::views::values)
        std::cout << value << ' ';
 
    std::map<char, int> map{{'A', 1}, {'B', 2}, {'C', 3}, {'D', 4}, {'E', 5}};
    auto odd = [](int x) { return 0 != (x & 1); };
    std::cout << "\nmap 中的奇数: ";
    for (int value : map | std::views::values | std::views::filter(odd))
        std::cout << value << ' ';
    std::cout << '\n';
}

输出：

list 中的 pair::second 值: 11.1 22.2 33.3
map 中的奇数: 1 3 5
