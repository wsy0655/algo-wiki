std::bitset<N>::test

< cpp‎ | utility‎ | bitset
bool test( std::size_t pos ) const;
 (C++23 起为 constexpr)

返回位于位置 pos 的位（从 0 计数）的值。

与 operator[] 不同，它会进行边界检查。

参数

pos - 要返回的位的位置（从 0 计数）

返回值

在位于位置 pos 的位被设置时返回 true，否则返回 false。

异常

在 pos 未对应有效位位置时抛出 std::out_of_range。

示例

#include <bit>
#include <bitset>
#include <cassert>
#include <iostream>
#include <stdexcept>
 
int main()
{
    std::bitset<10> b1("1111010000");
 
    std::size_t idx = 0;
    while (idx < b1.size() && !b1.test(idx))
        ++idx;
 
    assert(static_cast<int>(idx) == std::countr_zero(b1.to_ulong()));
 
    if (idx < b1.size())
        std::cout << "第一个设置的位的索引是 " << idx << '\n';
    else
        std::cout << "没有设置任何位\n";
 
    try
    {
        std::bitset<0B10'1001'1010> bad;
        if (bad.test(bad.size()))
            std::cout << "不符合预期!\n";
    }
    catch (std::out_of_range const& ex)
    {
        std::cout << "异常: " << ex.what() << '\n';
    }
}

可能的输出：

第一个设置的位的索引是 4
异常: bitset::test: __position (which is 666) >= _Nb (which is 666)
