std::deque<T,Allocator>::append_range

< cpp‎ | container‎ | deque

template< container-compatible-range<T> R >
void append_range( R&& rg );
 (C++23 起)

以非逆序插入范围 rg 的各元素的副本到 end() 之前。

所有迭代器（包括 end() 迭代器）都会失效。没有引用会失效。

rg 中的每个迭代器均恰好被解引用一次。

参数

rg - 容器兼容范围，即各元素可以转换为 T 的 input_range
类型要求
-
T 必须为从 *ranges::begin(rg) 可就位构造 (EmplaceConstructible) 到 deque。否则，行为未定义。

复杂度

与 rg 的大小呈线性。对 T 构造函数的调用次数恰好等于 std::ranges::size(rg))。

异常

如果由除了 T 的复制构造函数、移动构造函数、复制赋值运算符或移动赋值运算符之外抛出了异常，则没有效果。如果在向任何一端插入单个元素时抛出了异常，则没有效果。否则，如果非可复制插入 (CopyInsertable) T 的移动构造函数抛出了异常，则其效果未指明。

注解

功能特性测试宏 值 标准 功能特性
__cpp_lib_containers_ranges 202202L (C++23) 按范围构造和插入

示例

#include <cassert>
#include <deque>
#include <list>
 
int main()
{
    auto head = std::deque{1, 2, 3, 4};
    const auto tail = std::list{-5, -6, -7};
#ifdef __cpp_lib_containers_ranges
    head.append_range(tail);
#else
    head.insert(head.end(), tail.cbegin(), tail.cend());
#endif
    assert((head == std::deque{1, 2, 3, 4, -5, -6, -7}));
}
