std::pmr::memory_resource::do_is_equal

< cpp‎ | memory‎ | memory resource
virtual bool do_is_equal( const std::pmr::memory_resource& other ) const noexcept = 0;
 (C++17 起)

比较 *this 与 other 的相等性。

两个 memory_resource 比较相等，当且仅当从一个 memory_resource 分配的内存能从另一个解分配，反之亦然。

注解

other 的最终派生类型可能不匹配 *this 的最终派生类型。典型的派生类实现必须用 dynamic_cast 检查 *this 与 other 的最终派生类型是否匹配，而若转型失败则立即返回 false。
