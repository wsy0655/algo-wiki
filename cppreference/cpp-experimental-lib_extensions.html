C++ 标准库扩展

< cpp‎ | experimental


C++ 库基础扩展之版本 1 (The C++ Extensions for Library Fundamentals)，ISO/IEC TS 19568:2015 为 C++ 标准库定义下列新组件：

未选择包含于 C++17 中的部分

C++17 未选择包含 ISO/IEC TS 19568:2015 的下列组件。

多态分配器与内存资源

此节中的实体声明于 std::experimental::pmr 命名空间。

在标头 <experimental/memory_resource> 定义
memory_resource
 封装内存资源的类的抽象接口
(类)
synchronized_pool_resource
 线程安全的，在不同块大小的池中管理分配的 memory_resource
(类)
unsynchronized_pool_resource
 线程不安全的，在不同块大小的池中管理分配的 memory_resource
(类)
monotonic_buffer_resource
 特殊用途的 memory_resource，它仅在销毁资源时释放分配的内存
(类)
polymorphic_allocator
 支持运行时多态的分配器，多态基于构造所用的 memory_resource
(类模板)
new_delete_resource
 返回一个静态的程序范围 memory_resource，它使用全局 operator new 与 operator delete 分配和解分配内存
(函数)
null_memory_resource
 返回一个不进行任何分配的静态 memory_resource
(函数)
get_default_resource
 获取缺省 memory_resource
(函数)
set_default_resource
 设置缺省 memory_resource
(函数)
