std::swap(std::vector)

< cpp‎ | container‎ | vector

在标头 <vector> 定义
template< class T, class Alloc >

void swap( std::vector<T, Alloc>& lhs,

           std::vector<T, Alloc>& rhs );
 (C++17 前)
template< class T, class Alloc >

void swap( std::vector<T, Alloc>& lhs,
           std::vector<T, Alloc>& rhs )

               noexcept(/* 见下文 */);
 (C++17 起)
(C++20 起为 constexpr)

为 std::vector 特化了 std::swap 算法。交换 lhs 与 rhs 的内容。调用 lhs.swap(rhs)。


参数

lhs, rhs - 要交换内容的容器

返回值

（无）

复杂度

常数。

异常

noexcept 说明：  
noexcept(noexcept(lhs.swap(rhs)))
 (C++17 起)

示例

#include <algorithm>
#include <iostream>
#include <vector>
 
int main()
{
    std::vector<int> alice{1, 2, 3};
    std::vector<int> bob{7, 8, 9, 10};
 
    auto print = [](const int& n) { std::cout << ' ' << n; };
 
    // 打印交换前的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
 
    std::cout << "-- SWAP\n";
    std::swap(alice, bob);
 
    // 打印交换后的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
}

输出：

Alice: 1 2 3
Bobby: 7 8 9 10
-- SWAP
Alice: 7 8 9 10
Bobby: 1 2 3
