operator==(std::philox_engine)

< cpp‎ | numeric‎ | random‎ | philox engine


friend bool operator==( const philox_engine& lhs, const philox_engine& rhs );
 (C++26 起)

比较两个伪随机数引擎。两个引擎在内部状态等价，即对任何数量的 operator() 调用都会生成相等的值的情况下相等。

此函数对常规的无限定或有限定查找不可见，而只能在 decltype(*this) 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

lhs, rhs - 要比较的引擎

返回值

引擎等价时返回 true，否则返回 false。
