swap(std::stop_token)

< cpp‎ | thread‎ | stop token


friend void swap( stop_token& lhs, stop_token& rhs ) noexcept;
 (C++20 起)

对 std::stop_token 重载 std::swap 算法。交换 lhs 与 rhs 的关联停止状态。相当于调用 lhs.swap(rhs)。

此函数对常规的无限定或有限定查找不可见，而只能在 std::stop_token 为实参的关联类时由实参依赖查找找到。

参数

lhs, rhs - 要交换的 stop_token

返回值

（无）
