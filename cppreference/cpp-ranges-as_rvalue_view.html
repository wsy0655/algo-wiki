1) 范围适配器，用于表示底层 view 的视图，其元素为右值。
2) 范围适配器对象 (RangeAdaptorObject) 。令 e 是一个子表达式，并令 T 是 decltype((e))。那么表达式 views::as_rvalue(e) 表达式等价于：

  • 否则是 as_rvalue_view{e}。

示例

#include <algorithm>
#include <iostream>
#include <ranges>
#include <string>
#include <vector>
 
int main()
{
    std::vector<std::string> words =
        {"Quick", "red", "\N{FOX FACE}", "jumped", "over", "a", "pterodactyl"};
    std::vector<std::string> new_words;
 
    std::ranges::copy(
        words | std::views::as_rvalue,
        std::back_inserter(new_words)); // 从 words 移动字符串到 new_words
 
    auto quoted = std::views::transform([](auto&& s) { return "“" + s + "”"; });
 
    std::cout << "旧单词: ";
    for (auto&& word : words | std::views::as_rvalue | quoted)
        std::cout << word << ' ';
 
    std::cout << "\n新单词: ";
    for (auto&& word : new_words | std::views::as_rvalue | quoted)
        std::cout << word << ' ';
}

可能的输出：

旧单词: “” “” “” “” “” “” “” 
新单词: “Quick” “red” “🦊” “jumped” “over” “a” “pterodactyl”
