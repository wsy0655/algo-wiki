std::chrono::weekday_last::weekday

< cpp‎ | chrono‎ | weekday last
constexpr std::chrono::weekday weekday() const noexcept;
 (C++20 起)

取得存储于 *this 的 std::chrono::weekday 对象的副本。

返回值

存储于 *this 的 std::chrono::weekday 对象的副本。

示例

#include <chrono>
#include <iostream>
using namespace std::chrono;
 
int main()
{
    std::cout << std::boolalpha;
 
    auto wdl{Tuesday[last]}; // 某月最后一个星期二
    std::cout << (year_month_day{wdl/10/2019} ==
                  year_month_day{October/29/2019}) << ' ';
 
    wdl = {(wdl.weekday() + days(2))[last]}; // 最后一个星期二现在为最后一个星期四
    std::cout << (year_month_day{wdl/10/2019} ==
                  year_month_day{October/31/2019}) << '\n';
}

输出：

true true
