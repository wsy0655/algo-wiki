进行比较的函数对象。主模板会调用类型 T 上的 operator<=。

特化

less_equal<void>
(C++14)
 实现 x <= y 的函数对象，推导形参类型和返回类型
(类模板特化)

成员类型

类型 定义
result_type (C++17 弃用)(C++20 移除) bool
first_argument_type (C++17 弃用)(C++20 移除) T
second_argument_type (C++17 弃用)(C++20 移除) T

这些成员类型是通过公开继承 std::binary_function<T, T, bool> 获得的。

 (C++11 前)

成员函数

operator()
 检查第一实参是否小于 ﻿或等于 ﻿第二个
(公开成员函数)

std::less_equal::operator()

bool operator()( const T& lhs, const T& rhs ) const;
 (C++14 起为 constexpr)

检查 lhs 是否小于 ﻿或等于 rhs。

参数

lhs, rhs - 要比较的值

返回值

lhs <= rhs。

如果 T 是指针类型，那么结果会与由实现定义的指针严格全序保持一致。

异常

可能会抛出由实现定义的异常。

可能的实现

constexpr bool operator()(const T &lhs, const T &rhs) const 
{
    return lhs <= rhs; // 假定实现会处理指针全序
}
