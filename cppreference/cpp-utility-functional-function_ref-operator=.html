1) 复制赋值运算符被显式预置。std::function_ref 满足 copyable 且可平凡复制 (TriviallyCopyable) 。这个预置的赋值运算符进行存储的 thunk-ptr 和 bound-entity 的浅复制。
2) 用户定义的赋值运算符，当 T 与 std::function_ref 不同，std::is_pointer_v<T> 为 false，并且 T 不是 std::nontype_t 的特化时被显式弃置。此重载仅当以上约束条件均满足时才参与重载决议。
