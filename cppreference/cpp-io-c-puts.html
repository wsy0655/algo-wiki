std::puts

< cpp‎ | io‎ | c
在标头 <cstdio> 定义
int puts( const char *str );

写入空终止字符串 str 的每个字符和一个额外的换行符 '\n' 到输出流 stdout，如同以重复执行 std::putc 进行写入。

不写入 str 的空终止字符。

参数

str - 要写入的字符串

返回值

成功时返回非负值

失败时，返回 EOF 并设置 stdout 的错误 ﻿指示器（见 std::ferror()）。

注解

std::puts 函数向输出后附一个换行字符，而 std::fputs 不这么做。

不同的实现返回不同的非负数：一些返回最后写入的字符，一些返回写入的字符数（或若字符串长于 INT_MAX 则返回它），一些简单地返回非负常量。

当 stdout 被重定向到文件时，一种导致 std::puts 失败的典型原因是用尽了文件系统的空间。

示例

#include <cstdio>
 
int main()
{
    int rc = std::puts("Hello World");
 
    if (rc == EOF)
        std::perror("puts()"); // POSIX 要求设置 errno
}

输出：

Hello World
