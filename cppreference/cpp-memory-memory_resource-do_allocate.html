std::pmr::memory_resource::do_allocate

< cpp‎ | memory‎ | memory resource
virtual void* do_allocate( std::size_t bytes, std::size_t alignment ) = 0;
 (C++17 起)

分配至少有 bytes 字节的存储，对齐到指定的 alignment。

alignment 应当是二的幂。

异常

若不能取得拥有请求的大小和对齐的存储，则抛出异常。
