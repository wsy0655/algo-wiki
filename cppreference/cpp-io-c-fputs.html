std::fputs

< cpp‎ | io‎ | c
在标头 <cstdio> 定义
int fputs( const char* str, std::FILE* stream );

将空终止字符串 str 的每个字符写入到输出流 stream，如同通过重复执行 std::fputc 进行。

不写入 str 的终止空字符。

参数

str - 要写入的空终止字符串
stream - 输出流

返回值

成功时，返回非负值。

失败时，返回 EOF 并设置 stream 上的错误指示器（见 std::ferror）。

注解

相关函数 std::puts 会向输出后附换行符，而 std::fputs 则写入不修改的字符串。

不同的实现返回不同的非负数：一些返回最后写入的字符，一些返回写入的字符数（或若字符串长于 INT_MAX 则为该值），一些简单地返回如零这样的非负常量。

示例

#include <cstdio>
 
int main(void)
{
    int rc = std::fputs("Hello World", stdout);
 
    if (rc == EOF)
        std::perror("fputs()"); // POSIX 要求设置 errno
}

输出：

Hello World
