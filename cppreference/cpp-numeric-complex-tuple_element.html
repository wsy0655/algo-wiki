std::tuple_element<std::complex>

< cpp‎ | numeric‎ | complex
在标头 <complex> 定义
template< std::size_t I, class T >
struct tuple_element<I, std::complex<T>>;
 (C++26 起)

std::tuple_element 针对 std::complex 的部分特化提供了在编译时使用元组式语法来访问一个 complex 的底层实数和虚数类型的方法。为支持结构化绑定而提供它们。如果 I >= 2 则程序非良构。

成员类型

成员类型 定义
type T

注解

功能特性测试宏 值 标准 功能特性
__cpp_lib_tuple_like 202311L (C++26) 为 std::complex 添加元组协议

示例

#include <complex>
#include <type_traits>
 
static_assert([z = std::complex<float>()]
{
    using T = decltype(z);
    return
#if __cpp_lib_tuple_like >= 202311L
        std::is_same_v<std::tuple_element_t<0, T>, float> &&
        std::is_same_v<std::tuple_element_t<1, T>, float> &&
#endif
        std::is_same_v<T::value_type, float>;
}());
 
int main() {}
