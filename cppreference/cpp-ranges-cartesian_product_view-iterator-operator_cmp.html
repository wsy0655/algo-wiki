比较两个迭代器或一个迭代器和一个哨位。

令 current_ 代表底层的迭代器元组。

1) 等价于：return x.current_ == y.current_;
2) 若 std::get<i>(x.current_) == ranges::end(std::get<i>(x.parent_->bases_)) 对每个整数 0 ≤ i ≤ sizeof...(Vs) 为 true，则返回 true。否则，返回 false。
3) 等价于：return x.current_ <=> y.current_;

这些函数对常规的无限定或有限定查找不可见，而只能在 std::ranges::cartesian_product_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

x, y - 要比较的迭代器或哨位
