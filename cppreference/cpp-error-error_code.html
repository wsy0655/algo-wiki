std::error_code

< cpp‎ | error
在标头 <system_error> 定义
class error_code;
 (C++11 起)

std::error_code 表示依赖平台的错误码值。每个 std::error_code 对象，保有一个源于操作系统或某些低层接口的错误码值，和一个指向 std::error_category 类型对象的指针，它对应前述接口。错误码的值在不同错误类别之间可以不唯一。
