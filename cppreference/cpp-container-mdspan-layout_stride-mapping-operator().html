std::layout_stride::mapping<Extents>::operator()

< cpp‎ | container‎ | mdspan‎ | layout stride‎ | mapping
template< class... Indices >
constexpr index_type operator()( Indices... indices ) const noexcept;
 (C++23 起)

将多维索引 indices 映射为偏移量值。

等价于 return ((static_cast<index_type>(indices) * stride(P)) + ... + 0);，其中 P 为使得 std::is_same_v<std::index_sequence_for<Indices...>, std::index_sequence<P...>> 为 true 的包。

此重载只有在:

  • sizeof...(Indices) == extents_type​::​rank() 为 true，
  • (std::is_convertible_v<Indices, index_type> && ...) 为 true，并且
  • (std::is_nothrow_constructible_v<index_type, Indices> && ...) 为 true

时才会参与重载决议。 如果 extents_type​::index-cast(i) 不是 extents() 中的多维索引，那么其行为未定义。

参数

indices - 底层尺度对象中的多维索引

返回值

偏移值。
