std::istreambuf_iterator<CharT,Traits>::equal

< cpp‎ | iterator‎ | istreambuf iterator
bool equal( const istreambuf_iterator& it ) const;

检查 *this 与 it 是否都有效或都无效，无关乎它们所用的流缓冲。

参数

it - 要比较的另一流缓冲迭代器

返回值

*this 与 it 都有效或都无效时返回 true，否则返回 false。

异常

可能会抛出由实现定义的异常。
