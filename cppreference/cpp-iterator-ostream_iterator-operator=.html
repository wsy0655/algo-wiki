std::ostream_iterator<T,CharT,Traits>::operator=

< cpp‎ | iterator‎ | ostream iterator
ostream_iterator& operator=( const ostream_iterator& );
 (1)
ostream_iterator& operator=( const T& value );
 (2)
1) 复制赋值运算符。赋值 other 的内容。
2) 插入 value 到关联的流，然后插入分隔符，若在构造时指定它。

若 out_stream 是指向关联 std::basic_ostream 的指针而 delim 是在此对象构造时指定的分隔符，则效果等价于

*out_stream << value;
if(delim != 0)
    *out_stream << delim;
return *this;

参数

value - 要插入的对象

返回值

*this

注解

T 可以是任何拥有用户定义 operator<< 的类。

C++20 以前，复制赋值运算符的存在依赖于已弃用的隐式生成。

示例

#include <iostream>
#include <iterator>
 
int main()
{
    std::ostream_iterator<int> i1(std::cout, ", ");
    *i1++ = 1; // 通常形式，为标准算法所用
    *++i1 = 2;
    i1 = 3; // * 或 ++ 皆非必须
    std::ostream_iterator<double> i2(std::cout);
    i2 = 3.14;
    std::cout << '\n';
}

输出：

1, 2, 3, 3.14
