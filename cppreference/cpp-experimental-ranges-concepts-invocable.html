std::experimental::ranges::Invocable, std::experimental::ranges::RegularInvocable

< cpp‎ | experimental‎ | ranges
在标头 <experimental/ranges/concepts> 定义
template< class F, class... Args >

concept bool Invocable =
    requires(F&& f, Args&&... args) {
        ranges::invoke(std::forward<F>(f), std::forward<Args>(args)...);
            /* 不要求保持相等性 */

    };
 (范围 TS)
template< class F, class... Args >
concept bool RegularInvocable = Invocable<F, Args...>;
 (范围 TS)

Invocable 概念指定能用函数 ranges::invoke 以实参类型 Args... 调用可调用类型 F。

RegularInvocable 概念对 Invocable 概念添加 invoke 表达式保持相等性且不修改函数对象或实参的要求。

相等性保持

若表达式对给定的相等输入产生相等输出，则它保持相等性。

  • 表达式的输入由其操作数组成。
  • 表达式的输出由其结果和表达式所修改的所有操作数（若存在）组成。

每个要求保持相等性的表达式还进一步要求是稳定的：以相同输入对象对这种表达式进行两次求值必须拥有相等的输出，而无任何对这些输入对象的显式中间修改。

注解

Invocable 与 RegularInvocable 间的区别是纯语义上的。

随机数生成器可满足 Invocable，但不能满足 RegularInvocable（除了搞笑 的）。
