返回到当前元素的右值引用或指针。

返回值

1) static_cast<reference>(*current ﻿)(C++20 前)ranges::iter_move(current ﻿)(C++20 起)
2) current

注解

operator-> 被弃用的原因是在解引用它的结果时可能会产生左值。这可能导致意外行为。

示例

#include <iomanip>
#include <iostream>
#include <iterator>
#include <string>
#include <vector>
 
void print(auto rem, const auto& v)
{
    for (std::cout << rem; const auto& e : v)
        std::cout << std::quoted(e) << ' ';
    std::cout << '\n';
}
 
int main()
{
    std::vector<std::string> p{"alpha", "beta", "gamma", "delta"}, q;
    print("1) p: ", p);
    for (std::move_iterator it{p.begin()}, end{p.end()}; it != end; ++it)
    {
        it->push_back('!'); // 调用 -> string::push_back(char)
        q.emplace_back(*it); // *it <- 重载 (1)
    }
    print("2) p: ", p);
    print("3) q: ", q);
 
    std::vector v{1, 2, 3};
    std::move_iterator it{v.begin()};
    // *it = 13; // 错误：将右值用作左值
}

可能的输出：

1) p: "alpha" "beta" "gamma" "delta"
2) p: "" "" "" ""
3) q: "alpha!" "beta!" "gamma!" "delta!"
