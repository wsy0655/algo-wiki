std::experimental::basic_string_view<CharT,Traits>::data

< cpp‎ | experimental‎ | basic string view
constexpr const_pointer data() const noexcept;
 (库基础 TS)

返回指向底层字符数组的指针。该指针使得范围 [data(), data() + size()) 有效且其中各值对应于视图的各值。（新手注意：与 basic_string::data() 和字符串字面量不同，data() 可能返回指向并非空终止的缓冲区的指针。因此将 data() 传递给接受这种 const CharT* 且期待空终止字符串的例程通常是错误的。）

参数

（无）

返回值

指向底层字符数组的指针。

复杂度

常数。

参阅

front
 访问首字符
(公开成员函数)
back
 访问最末字符
(公开成员函数)
