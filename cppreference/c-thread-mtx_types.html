mtx_plain, mtx_recursive, mtx_timed

< c‎ | thread

enum {

    mtx_plain = /* 未指明 */,
    mtx_recursive = /* 未指明 */,
    mtx_timed = /* 未指明 */

};
 (C11 起)

传递给 mtx_init 时，标识要创建的互斥体类型。

常量 解释
mtx_plain 平常互斥体
mtx_recursive 递归互斥体
mtx_timed 定时互斥体
