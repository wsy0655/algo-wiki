std::experimental::function<R(Args...)>::get_allocator

< cpp‎ | experimental‎ | function
std::pmr::polymorphic_allocator<> get_allocator() const noexcept;
 (库基础 TS v3)

返回在构造 function 对象时存储的 std::pmr::polymorphic_allocator<>。若未显式指定分配器，则存储的分配器在移动构造中等于源分配器，否则为默认构造。

参数

（无）

返回值

在构造 function 对象时存储的 std::pmr::polymorphic_allocator<>。

参阅

get_memory_resource
(库基础 TS v3 中移除)
 获得指向此对象用来分配内存的 memory_resource 的指针
(公开成员函数)
