std::basic_filebuf<CharT,Traits>::imbue

< cpp‎ | io‎ | basic filebuf
protected:
virtual void imbue( const std::locale& loc )

更改关联的本地环境，使得此调用后（和另一 imbue() 调用前）所有插入或提取的字符均以 loc 的 std::codecvt 平面转换。

若旧本地环境的编码依赖状态，且文件未被寻位到起始，则新的本地环境必须与先前浸染者拥有相同的 std::codecvt 平面。

参数

loc - 要与流浸染的本地环境

返回值

（无）
