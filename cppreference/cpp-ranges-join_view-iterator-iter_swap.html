iter_swap(ranges::join_view::iterator)

< cpp‎ | ranges‎ | join view‎ | iterator


friend constexpr void iter_swap( const /*iterator*/& x, const /*iterator*/& y )

    noexcept( /* 见下文 */ )

        requires std::indirectly_swappable<InnerIter>;
 (C++20 起)

交换由两个底层迭代器（表示为 inner_）指向的对象。

等价于：ranges::iter_swap(x.inner_, y.inner_);。

此函数对常规的无限定或有限定查找不可见，而只能在 join_view::iterator<Const> 为实参的关联类时由实参依赖查找找到。

参数

x, y - iterators

返回值

（无）

异常

noexcept 说明：  
noexcept(noexcept(ranges::iter_swap(x.inner_, y.inner_)))
