std::vector<bool,Allocator>::flip

< cpp‎ | container‎ | vector bool
在标头 <vector> 定义
void flip();
 (C++20 起为 constexpr)

翻转 vector 中的每个 bool（以其相反值替换）。

示例

#include <iostream>
#include <vector>
 
void print(const std::vector<bool>& vb)
{
    for (const bool b : vb)
        std::cout << b;
    std::cout << '\n';
}
 
int main()
{
    std::vector<bool> v{0, 1, 0, 1};
    print(v);
    v.flip();
    print(v);
}

输出：

0101
1010
