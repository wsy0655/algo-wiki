std::experimental::erased_type

< cpp‎ | experimental
在标头 <experimental/utility> 定义
struct erased_type { };
 (库基础 TS)
(库基础 TS v3 中移除)

类 erased_type 是空结构体，作为类型占位符用于在运行时确定实际类型的情形。例如，使用类型擦除分配器的类中，嵌套 typedef allocator_type 是 erased_type 的别名。

注解

erased_type 在 LFTS v3 中被移除，因为 std::pmr::polymorphic_allocator<> 对于类型擦除更合适。
