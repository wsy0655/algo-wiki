计算整数 num 的绝对值。如果结果不能以返回类型表示，那么行为未定义。

如果以不能由整数提升转换成 int 的无符号整数类型参数调用 std::abs，那么程序非良构。

当且仅当 std::intmax_t 是扩展整数类型时，<cinttypes> 中会提供 std::abs 对 std::intmax_t 的重载 (6)。

 (C++11 起)

参数

num - 整数

返回值

在可表示的情况下返回 num 的绝对值（即 |num|）。

注解

补码系统中，最小负值的绝对值在范围外，例如对于 32 位补码类型 int，INT_MIN 是 -2147483648，但本应有的结果 2147483648 大于 INT_MAX，即 2147483647。

示例

#include <climits>
#include <cstdlib>
#include <iostream>
 
int main()
{
    std::cout << std::showpos
              << "abs(+3) = " << std::abs(3) << '\n'
              << "abs(-3) = " << std::abs(-3) << '\n';
 
//  std::cout << abs(INT_MIN); // 补码系统上行为未定义
}

输出：

abs(+3) = 3
abs(-3) = 3
