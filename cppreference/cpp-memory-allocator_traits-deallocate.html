std::allocator_traits<Alloc>::deallocate

< cpp‎ | memory‎ | allocator traits
在标头 <memory> 定义
static void deallocate( Alloc& a, pointer p, size_type n );
 (C++11 起)
(C++20 起为 constexpr)

用分配器 a 解分配 p 所引用的存储，通过调用 a.deallocate(p, n)。

参数

a - 使用的分配器
p - 指向先前分配存储的指针
n - 分配存储的对象数

返回值

（无）
