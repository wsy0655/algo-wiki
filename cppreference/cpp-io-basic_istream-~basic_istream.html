std::basic_istream<CharT,Traits>::~basic_istream

< cpp‎ | io‎ | basic istream
virtual ~basic_istream();

析构输入流。

注解

此析构函数不在底层流缓冲（rdbuf()）上进行任何操作：派生输入流，如 std::basic_ifstream 和 std::basic_istringstream 的析构函数负责调用流缓冲的析构函数。

示例

#include <iostream>
#include <sstream>
 
void print_stringbuf(std::streambuf* p)
{
    std::istream buf(p); // buf 与 s1 共享缓冲
    int n;
    buf >> n;
    std::cout << n;
} // 调用 buf 的析构函数。 p 保持不受影响
 
int main()
{
    std::istringstream s1("10 20");
    print_stringbuf(s1.rdbuf());
    int n;
    s1 >> n;
    std::cout << ',' << n << '\n';
}

输出：

10,20
