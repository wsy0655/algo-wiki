std::is_error_condition_enum

< cpp‎ | error‎ | error condition
在标头 <system_error> 定义
template< class T >
struct is_error_condition_enum;
 (C++11 起)

如果 T 是错误条件枚举（例如 std::errc），那么此模板提供等于 true 的成员常量 value。对于任何其他类型，value 是 false。

此模板可以为由程序定义的类型特化，以指示该类型适用 std::error_condition 隐式转换。

辅助变量模板

template< class T >

inline constexpr bool is_error_condition_enum_v =

    is_error_condition_enum<T>::value;
 (C++17 起)
