std::stack<T,Container>::swap

< cpp‎ | container‎ | stack

void swap( stack& other ) noexcept(/* 见下文 */);
 (C++11 起)
交换容器适配器与 other 的内容。相当于调用 
using std::swap;
swap(c, other.c);

参数

other - 要交换内容的容器适配器

返回值

（无）

异常

noexcept 说明：  
noexcept(noexcept(swap(c, other.c)))

上述表达式中，用与 C++17 std::is_nothrow_swappable 特性所用的相同方式查找标识符 swap。

 (C++11 起)
(C++17 前)
noexcept 说明：  
noexcept(std::is_nothrow_swappable_v<Container>)
 (C++17 起)

复杂度

与底层容器相同（常为常数）。

注解

一些实现（例如 libc++）为 C++11 前模式提供 swap 成员函数作为扩展。
