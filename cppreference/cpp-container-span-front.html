std::span<T,Extent>::front

< cpp‎ | container‎ | span
constexpr reference front() const;
 (C++20 起)

返回到 span 中首元素的引用。

如果 empty() 是 true，那么行为未定义。

 (C++26 前)

如果 empty() 是 true，那么：

  • 如果实现是硬化实现，那么就会发生契约违背。并且契约违背处理函数在“观察”求值语义下返回时行为未定义。
  • 如果实现不是硬化实现，那么行为未定义。
 (C++26 起)

返回值

到首元素的引用。

复杂度

常数。

注解

对于 span c，表达式 c.front() 等价于 *c.begin()。

示例

#include <iostream>
#include <span>
 
void print(std::span<const int> const data)
{
    for (auto offset{0U}; offset != data.size(); ++offset)
        std::cout << data.subspan(offset).front() << ' ';
    std::cout << '\n';
}
 
int main()
{
    constexpr int data[]{0, 1, 2, 3, 4, 5, 6};
    print({data, 4});
}

输出：

0 1 2 3
