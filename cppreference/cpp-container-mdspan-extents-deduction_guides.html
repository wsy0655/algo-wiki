在标头 <mdspan> 定义
template< class... Integrals >
explicit extents( Integrals... ) -> /* 见下文 */;
 (C++23 起)

为 std::extents 提供了推导指引，以允许从整型实参进行推导。

所推导类型等价于

std::dextents<std::size_t, sizeof...(Integrals)>

 (C++26 前)

std::extents<std::size_t, maybe-static-ext ﻿<Integrals>...>

 (C++26 起)

此重载只有在(std::is_convertible_v<Integrals, std::size_t> && ...) 为 true时才会参与重载决议。
