std::swap(std::flat_multiset)

< cpp‎ | container‎ | flat multiset

在标头 <flat_set> 定义
friend void swap( std::flat_multiset& lhs, std::flat_multiset& rhs ) noexcept;
 (C++23 起)

为 std::flat_multiset 特化了 std::swap 算法。交换 lhs 与 rhs 的内容。调用 lhs.swap(rhs)。


参数

lhs, rhs - 要交换内容的容器

返回值

（无）

复杂度

与交换各底层容器相同。

示例

#include <algorithm>
#include <iostream>
#include <flat_set>
 
int main()
{
    std::flat_multiset<int> alice{1, 2, 3};
    std::flat_multiset<int> bob{7, 8, 9, 10};
 
    auto print = [](const int& n) { std::cout << ' ' << n; };
 
    // 打印交换前的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
 
    std::cout << "-- SWAP\n";
    std::swap(alice, bob);
 
    // 打印交换后的状态
    std::cout << "Alice:";
    std::for_each(alice.begin(), alice.end(), print);
    std::cout << "\nBobby:";
    std::for_each(bob.begin(), bob.end(), print);
    std::cout << '\n';
}

输出：

Alice: 1 2 3
Bobby: 7 8 9 10
-- SWAP
Alice: 7 8 9 10
Bobby: 1 2 3
