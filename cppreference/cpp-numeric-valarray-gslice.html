std::gslice

< cpp‎ | numeric‎ | valarray
在标头 <valarray> 定义
class gslice;

std::gslice 是标识 std::valarray 索引子集的选择器，定义为多层的跨度和大小集合。 std::gslice 类型的对象可用作 valarray 的 operator[] 下标，例如，用于选择表示为 valarray 的多维数组的列。

给定起始值 s、跨度列表 ij 及大小列表 dj，从这些值构造的 std::gslice 选择索引集 kj=s+Σj(ijdj)。

例如拥有始于索引 3，跨度为 {19,4,1} 长度为 {2,4,3} 的 gslice 生成下列 24=2*4*3 个索引的集合：

3 + 0*19 + 0*4 + 0*1 = 3,
3 + 0*19 + 0*4 + 1*1 = 4,
3 + 0*19 + 0*4 + 2*1 = 5,
3 + 0*19 + 1*4 + 0*1 = 7,
3 + 0*19 + 1*4 + 1*1 = 8,
3 + 0*19 + 1*4 + 2*1 = 9,
3 + 0*19 + 2*4 + 0*1 = 11,
...
3 + 1*19 + 3*4 + 1*1 = 35,
3 + 1*19 + 3*4 + 2*1 = 36

可以构造选择某些索引多于一次的 std::gslice 对象：如果上述示例使用跨度 {1,1,1}，那么索引将是 {3, 4, 5, 4, 5, 6, ...}。这种 gslice 只可用于 std::valarray::operator[] 的 const 版本的实参，否则行为未定义。

构造新的通用切片。

1) 默认构造函数。等价于 gslice(0, std::valarray<std::size_t>(), std::valarray<std::size_t>())。此构造函数的存在只为允许构造数组切片。
2) 以参数 start、sizes、strides 构造新切片。
3) 构造 other 的副本。

sizes - 定义每个维度元素数量的数组
strides - 定义每个维度中前后元素间的位置差数的数组
other - 要复制的另一切片

示例

演示使用 gslice 定位三维数组的列：

#include <iostream>
#include <valarray>
 
void test_print(std::valarray<int>& v, int planes, int rows, int cols)
{
    for (int r = 0; r < rows; ++r)
    {
        for (int z = 0; z < planes; ++z)
        {
            for (int c = 0; c < cols; ++c)
                std::cout << v[z * rows * cols + r * cols + c] << ' ';
            std::cout << "  ";
        }
        std::cout << '\n';
    }
}
 
int main()
{
    std::valarray<int> v = // 3d 数组：2 x 4 x 3 元素
        {111,112,113 , 121,122,123 , 131,132,133 , 141,142,143,
         211,212,213 , 221,222,223 , 231,232,233 , 241,242,243};
    // int ar3d[2][4][3]
    std::cout << "初始 2x4x3 数组：\n";
    test_print(v, 2, 4, 3);
 
    // 更新两个平面第一列的每个值
    v[std::gslice(0, {2, 4}, {4 * 3, 3})] = 1; // 二层的一个 12 元素跨度
                                               // 然后是四层的二个 3 元素跨度
 
    // 从第一平面的第二列减去第三列的值
    v[std::gslice(1, {1, 4}, {4 * 3, 3})] -= v[std::gslice(2, {1, 4}, {4 * 3, 3})];
 
    std::cout << "\n" "在列操作后：\n";
    test_print(v, 2, 4, 3);
}

输出：

初始 2x4x3 数组：
111 112 113   211 212 213
121 122 123   221 222 223
131 132 133   231 232 233
141 142 143   241 242 243
 
在列操作后：
1 -1 113   1 212 213
1 -1 123   1 222 223
1 -1 133   1 232 233
1 -1 143   1 242 243
