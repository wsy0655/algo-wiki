operator==(std::text_encoding)

< cpp‎ | text‎ | text encoding


friend constexpr bool operator==( const text_encoding& a,
                                  const text_encoding& b ) noexcept;
 (1) (C++26 起)
friend constexpr bool operator==( const text_encoding& a, id i ) noexcept;
 (2) (C++26 起)

对 text_encoding 实施比较操作。

1) 比较两个 text_encoding 对象。当且仅当若 a.mib() 和 b.mib() 都等于 id::other 则 comp-name (a.name(), b.name()) 为 true，否则 a.mib() 等于 b.mib() 时，两对象比较为相等。
2) 比较 text_encoding 对象与 MIBenum 值。当且仅当 a.mib() 等于 i 时两对象比较为相等。

这些函数对常规的无限定或有限定查找不可见，而只能在 std::text_encoding 为实参的关联类时由实参依赖查找找到。

!= 运算符从 operator== 运算符合成。

参数

a, b - 要比较的 text_encoding 对象
i - 要与 a 中包含的 MIBenum 值比较的 id 值

返回值

1) a.mib() == id::other && b.mib() == id::other
    ? comp-name (a.name(), b.name())
    : a.mib() == b.mib()。
2) a.mib() == i。
