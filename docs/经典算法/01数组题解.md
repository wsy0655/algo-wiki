# 题目名称

## 📝 题目描述

给出一个由 $ n(n\le 5000) $ 个不超过 $10^6$ 的正整数组成的序列。请输出这个序列的 **最长上升子序列** 的长度。

## 💡 解题思路

- 状态设计：$f_i$ 代表以 $i$ 结尾的最长上升子序列的长度。
- 状态转移：枚举上一个结尾元素 $j$，如果 $a_i > a_j$，则 $f_i = \max(f_i, f_j + 1)$。
  - 这意味着可以在 $a_j$ 的后面拼接 $a_i$ 使得继续上升。
- 初始化：$f_i = 1$。
- 答案：$\max_{i=1}^n f_i$


$$
f_i = \max(f_i, f_j + 1)
$$


## 🔢 复杂度分析

- 时间复杂度：$O(n^2)$
- 空间复杂度：$O(n)$

## ✅ 代码实现

```cpp
for (int i = 1; i <= n; i++) 
{
    f[i] = 1;
    for (int j = 1; j < i; j++) 
    {
        if (a[i] > a[j]) 
        {
            f[i] = max(f[i], f[j] + 1);
        }
    }
    ans = max(ans, f[i]);
}
```

这是行内公式 $f_i = \max(f_i, f_j + 1)$。
