注意到 $k\leq 3$。即最多只会有三个避难所。

因此本题可以分类讨论完成。

```cpp
else

    // 3 个避难所代码怎么写
}
```

接下来我们一一分析即可：


**一个避难所**

- 首先枚举避难所的编号：`for (int i = 1; i <= n; i++)`。
    - 枚举其余所有村庄 $j$，`for (int j = 1; j <= n; j++)`。
    - 计算 $i,j$ 间距离的最大值记作 $mx$。
    - 对所有的 $mx$ 求一个最小的。即最小的最大距离

其中 $dis(i,j)$ 为自定义函数用于计算 $i,j$ 的曼哈顿距离。


```cpp
int dis(int i, int j)
{
    return abs(x[i] - x[j]) + abs(y[i] - y[j]);
}
if (k == 1)
{
    int mn = 1e9;
    for (int i = 1; i <= n; i++)
    {
        int mx = 0;
        for (int j = 1; j <= n; j++) 
            mx = max(mx, dis(i, j));
        mn = min(mn, mx);
    }
    cout << mn << "\n";
}
```

思路同上，首先用两个循环枚举两个避难所。

然后枚举所有村庄，在最大的距离中找到最小值。
