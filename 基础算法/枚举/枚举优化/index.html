    жһҡдёҫдјҳеҢ–

    жұӮе’Ңзұ»¶

    еҰӮжһңеӨҡдёӘеҸҳйҮҸд№Ӣй—ҙеӯҳеңЁ ж•°еӯҰе…ізі»пјҢеҸҜд»Ҙ еҸӘжһҡдёҫе…¶дёӯйғЁеҲҶеҸҳйҮҸпјҢ жҺЁеҜјеҮәе…¶е®ғеҸҳйҮҸпјҢд»ҺиҖҢ еҮҸе°‘еҫӘзҺҜеұӮж•°гҖӮ

    еёёи§ҒжЁЎејҸпјҡжһҡдёҫдёӨдёӘеҸҳйҮҸеҗҺпјҢз¬¬дёүдёӘеҸҳйҮҸзҡ„еҖје”ҜдёҖзЎ®е®ҡпјҢж— йңҖеҶҚжһҡдёҫпјҒ

    еёёи§ҒдҫӢеӯҗ

    • йёЎе…”еҗҢз¬јпјҡжҖ»ж•°е·ІзҹҘпјҢйёЎ $x$ жһҡдёҫеҗҺпјҢе…” = жҖ»ж•° $-x$гҖӮ
    • дёүж•°жұӮе’ҢпјҡиӢҘ $a_i + a_j = a_k$пјҢжһҡдёҫ $i, j$ еҗҺпјҢ$a_k$ е”ҜдёҖгҖӮ

    дҫӢйўҳдёҖпјҡеҚЎзүҮй—®йўҳ¶

    жңүдёүз§ҚеҚЎзүҮпјҢеҲҶеҲ«дёә AгҖҒBгҖҒCпјҢжҜҸеј еҚЎзүҮеқҮжңү $n$ еј пјҢеҚЎзүҮдёҠеҜ№еә”зҡ„ж•°еӯ—дёә $1,2,3$гҖӮ

    зҺ°еңЁжғіз»„жҲҗдёҖдёӘж•°еӯ— $d$пјҢй—®жңүеӨҡе°‘з§Қж–№жЎҲпјҹ

    жңҙзҙ еҒҡжі•

    • дёүйҮҚеҫӘзҺҜпјҡ$i,j,k$ еҲҶеҲ«иЎЁзӨә AгҖҒBгҖҒC еҚЎзүҮзҡ„дҪҝз”Ёж•°йҮҸгҖӮ
    • жҜҸз§Қеј ж•°д»Һ $0$ еҲ° $n$ жһҡдёҫпјҢеҲӨж–ӯпјҡ
      • if (i + 2 * j + 3 * k == d)
    • ж—¶й—ҙеӨҚжқӮеәҰпјҡ$O(n^3)$

    дјҳеҢ–еҒҡжі•

    жҲ‘д»¬жңүпјҡ$i + 2j + 3k = d$

    жһҡдёҫ $i,j$ еҗҺпјҢ$k$ зҡ„еҖјеҸҜзӣҙжҺҘжұӮеҮәпјҡ

    \[ k = \frac{d - i - 2j}{3} \]

    йңҖиҰҒжіЁж„Ҹпјҡ

    • еҲӨж–ӯ $d - i - 2j$ жҳҜеҗҰдёә $3$ зҡ„еҖҚж•°пјҲеҗҰеҲҷ $k$ дёҚжҳҜж•ҙж•°пјү
    • еҲӨж–ӯ $0 \leq k \leq n$пјҢзЎ®дҝқеҚЎзүҮж•°йҮҸеҗҲжі•

    ж—¶й—ҙеӨҚжқӮеәҰйҷҚдёә $O(n^2)$пјҢжҳҫи‘—жҸҗй«ҳиҝҗиЎҢйҖҹеәҰгҖӮ

    й—®йўҳеҸҳеҪў

    еҪ“й—®йўҳдҝ®ж”№дёәдҪҝз”Ёдёүз§ҚеҚЎзүҮжұӮз»„еҗҲеҮә $\leq d$ зҡ„ж–№жЎҲж•°ж—¶пјҢдёҖж ·еҸҜд»ҘиҝӣиЎҢдјҳеҢ–гҖӮ

    жӯӨж—¶ж–№зЁӢдёәпјҡ$i+2j+3d\leq d$гҖӮ

    еҪ“жһҡдёҫ $i,j$гҖӮи®Ўз®—еҮә $k$:

    \[ k \leq \frac{d - i - 2j}{3} \]

    з”ұдәҺ $k$ еҝ…йЎ»дёәж•ҙж•°пјҢеӣ жӯӨ $k$ еҸҜд»ҘеҸ– $[0,\lfloor \dfrac{d-i-2j}{3}\rfloor]$ д№Ӣй—ҙзҡ„жүҖжңүж•ҙж•°гҖӮ

    int ans = 0;
for (int i = 0; i <= n; i++)
{
    for (int j = 0; j <= n; j++)
    {
        int k = (d - i - 2 * j) / 3;
        ans += k;
    }
}

    дҫӢйўҳдәҢпјҡзҸ еҝғз®—жөӢйӘҢпјҲеҠ ејәзүҲпјү¶

    жңҙзҙ еҒҡжі•

    ж—¶й—ҙеӨҚжқӮеәҰпјҡ$O(n^3)$гҖӮ

    \[ \begin{array}{ll} 1 & \textbf{sort(A)}\\ 2 & \textbf{for } i\gets 1\textbf{ to }n\\ 3 & \qquad \textbf{for } j\gets i+1\textbf{ to }n\\ 4 & \qquad \qquad \textbf{for } k\gets j+1\textbf{ to }n\\ 5 & \qquad \qquad \qquad \textbf{if } A[i]+A[j] = A[k] \textbf{ and } vis[A[k]] = 0\\ 6 & \qquad \qquad \qquad \qquad \textbf{cnt}\gets \textbf{cnt}+1\\ 7 & \qquad \qquad \qquad \qquad \textbf{vis[A[k]]}\gets 1 \end{array}\]

    дјҳеҢ–еҒҡжі•

    еҰӮжһңжҲ‘д»¬еҸӘжһҡдёҫеүҚдёӨдёӘж•° $a_i$ дёҺ $a_j$пјҡ

    йӮЈд№Ҳ $a_k = a_i + a_j$ жҳҜе”ҜдёҖзЎ®е®ҡзҡ„гҖӮ

    • еҸҢйҮҚеҫӘзҺҜжһҡдёҫ $a_i, a_j$пјҢз»ҷ $a_i + a_j$ жү“дёҠж Үи®°гҖӮ
    • жңҖеҗҺйҒҚеҺҶеҺҹеәҸеҲ—пјҢиӢҘ $\texttt{vis[a[i]] == true}$ еҲҷзҙҜи®ЎдёҖж¬Ўзӯ”жЎҲгҖӮ

    ж—¶й—ҙеӨҚжқӮеәҰпјҡ$O(n^2)$гҖӮ

    жіЁж„Ҹ vis ж•°з»„зҡ„з©әй—ҙй—®йўҳгҖӮ

    дёҖдёӘж•° $x$ иў«еӨҡе°‘дёӘеҢәй—ҙеҢ…еҗ«пјҹ

    • е·Ұз«ҜзӮ№ $\leq x$ жңү $x$ з§ҚйҖүжӢ©пјӣ
    • еҸіз«ҜзӮ№ $\geq x$ жңү $n - x + 1$ з§ҚйҖүжӢ©пјӣ
    • жүҖд»Ҙ $x$ зҡ„иҙЎзҢ®жҳҜпјҡ$x \times (n - x + 1)$пјӣ
    • жҜҸдёӘе№іж–№ж•° $x = i^2$пјҢзҙҜеҠ е…¶иҙЎзҢ®еҚіеҸҜгҖӮ
    • и®Ўз®—иҝҮзЁӢдёӯжіЁж„ҸеҸ–жЁЎпјҢйҳІжӯўдёӯй—ҙжәўеҮәгҖӮ
    for (long long i = 1; i * i <= n; i++)
{
    long long x = i * i;
    long long l = x, r = n - x + 1;
    ans += l % mod * r % mod;
    ans %= mod;
}

    дјҳеҢ–дёҠдёӢз•Ң¶

    жһҡдёҫзұ»й—®йўҳдёӯпјҢзј©е°ҸеҫӘзҺҜдёҠз•Ң жҳҜеёёи§Ғдё”жңүж•Ҳзҡ„дјҳеҢ–ж–№ејҸгҖӮ

    дҫӢеҰӮеңЁе®Ңе…Ёе№іж–№ж•°й—®йўҳдёӯпјҢжһҡдёҫ $\leq n$ зҡ„е®Ңе…Ёе№іж–№ж•°ж—¶пјҢеҸӘйңҖйҒҚеҺҶ $\sqrt{n}$ дёӘж•°пјҡ

    • $i \times i \leq n$

    еҶҚеҰӮпјҢиӢҘжұӮйқһиҙҹж•ҙж•°и§Јж»Ўи¶і $x^3 + y^3 \leq n$ дё” $n \leq 10^{18}$гҖӮ

    • еҲҷ $x, y$ зҡ„дёҠз•Ңдёә $n^{1/3} \approx 10^6$
    • жҳҺжҳҫжҜ” $10^{18}$ е°ҸеҫҲеӨҡгҖӮ

    дҫӢйўҳпјҡеӣӣж–№е®ҡзҗҶпјҲеӣӣе№іж–№е’Ңпјү¶

    жңҙзҙ еҒҡжі•

    жһҡдёҫ $a \leq b \leq c \leq d$пјҢеҲӨж–ӯжҳҜеҗҰж»Ўи¶іпјҡ

    \[ a^2 + b^2 + c^2 + d^2 = n \]

    и§ӮеҜҹеҲ° $a + b + c + d$ жҳҜеӣәе®ҡеҖј $\Rightarrow$ иҖғиҷ‘е°Ҷ $d$ з”ЁеүҚйқўдёүдёӘжҺЁеҮәпјҡ

    \[ d^2 = n - a^2 - b^2 - c^2 \]

    йӘҢиҜҒ $d$ жҳҜеҗҰдёәж•ҙж•°е№іж–№пјҢдё”ж»Ўи¶і $d \geq c$

    ж—¶й—ҙеӨҚжқӮеәҰпјҡ$O(n^3)$пјҲд»ҚиҝҮеӨ§пјү

    дјҳеҢ–

    з¬¬дёҖйҮҚеҫӘзҺҜпјҲжһҡдёҫ $a$пјү

    • ж №жҚ® $a \leq b \leq c \leq d$пјҢиӢҘ $a$ еӨӘеӨ§пјҢеҗҺйқўдёүйЎ№ж— жі•еҮ‘еҮә $n$
    • жүҖд»ҘйңҖж»Ўи¶іпјҡ
    \[ 4 \times a^2 \leq n \]
    • еӣ жӯӨз¬¬дёҖеұӮеҫӘзҺҜдёәпјҡ$\texttt{for (int a = 0; 4 * a * a <= n; a++)}$

    з¬¬дәҢеұӮеҫӘзҺҜ $b$пјҲдҝқиҜҒ $b \geq a$пјүпјҡ

    \[ a^2 + 3b^2 \leq n \quad \Rightarrow \quad \texttt{for (int b = a; a * a + 3 * b * b <= n; b++)} \]

    з¬¬дёүеұӮеҫӘзҺҜ $c$пјҲдҝқиҜҒ $c \geq b$пјүпјҡ

    \[ a^2 + b^2 + 2c^2 \leq n \quad \Rightarrow \quad \texttt{for (int c = b; a * a + b * b + 2 * c * c <= n; c++)} \]

    и®Ўж•°зұ»¶

    жӯӨзұ»й—®йўҳзү№зӮ№пјҡж•°жҚ®йҮҸеӨ§пјҢж•°еҖјеҹҹе°ҸгҖӮ

    дҫӢеҰӮеҮәйўҳдәәи®ҫзҪ® $a_i \leq 10^5$пјҲиҖҢйқһ $10^9$пјүпјҢеҚідҫҝ $n \leq 10^5$пјҢдҫқз„¶еҸҜд»ҘиҖғиҷ‘еҖјеҹҹдјҳеҢ–гҖӮ

    д»ҺеҖјеҹҹи§’еәҰиҝӣиЎҢжһҡдёҫ/и®Ўж•°пјҢжҲ–и®ёжҳҜи§Јйўҳе…ій”®гҖӮ

    дҫӢйўҳдёҖпјҡCF1742D - Coprime¶

    жңҙзҙ и§Јжі•

    жҡҙеҠӣжһҡдёҫ $a_i, a_j$

    • еҸҢйҮҚеҫӘзҺҜеҲӨж–ӯжҳҜеҗҰж»Ўи¶і $\gcd(a_i, a_j) = 1$гҖӮ
      • жұӮ $\gcd$ еҸҜд»Ҙз”Ё GCC иҮӘеёҰзҡ„еҮҪж•° __gcd(a, b)гҖӮ
    • иӢҘж»Ўи¶іпјҢжӣҙж–° $i + j$ зҡ„жңҖеӨ§еҖјгҖӮ
    • ж—¶й—ҙеӨҚжқӮеәҰдёәпјҡ$O(n^2 \log a_i)$гҖӮ
      • е…¶дёӯ $\log_{a_i}$ жҳҜжұӮ $\gcd$ зҡ„ж—¶й—ҙеӨҚжқӮеәҰгҖӮ

    дјҳеҢ–и§Јжі•

    жіЁж„ҸеҲ° $a_i \leq 10^3$гҖӮ

    еҸҜд»Ҙд»Һ еҖјеҹҹи§’еәҰжһҡдёҫжүҖжңүж•°еҖјеҜ№ $(i,j)$гҖӮ

    еҰӮжһң $\gcd(i, j) = 1$пјҢе°ұжҹҘжүҫж•°з»„дёӯжҳҜеҗҰеҮәзҺ°иҝҮ $i$ е’Ң $j$гҖӮ

    • еҗҢдёҖж•°еҖјеҸҜиғҪеӨҡж¬ЎеҮәзҺ°пјҢжҲ‘д»¬еҸӘе…іеҝғе…¶ жңҖйқ еҗҺзҡ„зҙўеј•дҪҚзҪ®гҖӮ

    е…·дҪ“е®һзҺ°пјҡ

    • еҲӣе»әж•°з»„ $\texttt{pos[1001]}$пјҢи®°еҪ•жҜҸдёӘж•°еҖјзҡ„жңҖеӨ§дёӢж ҮгҖӮ
    • еҜ№дәҺжҜҸдёӘ $a[i]$ жү§иЎҢпјҡpos[a[i]] = i

    еӨҡж¬ЎиҰҶзӣ–еҗҺпјҢ$\texttt{pos[x]}$ жңҖз»ҲжҳҜ $x$ зҡ„жңҖеҗҺдёҖж¬ЎеҮәзҺ°дҪҚзҪ®гҖӮ

    • жһҡдёҫ $i, j \in [1, 1000]$пјҢиӢҘ $\gcd(i, j) = 1$ дё” $\texttt{pos[i], pos[j]}$ йғҪеӯҳеңЁпјҡ
    \[ \texttt{ans} \leftarrow \max(\texttt{ans}, \texttt{pos[i]} + \texttt{pos[j]}) \]

    ж—¶й—ҙеӨҚжқӮеәҰпјҡ$O(V^2 \log V), \quad V \leq 1000$
