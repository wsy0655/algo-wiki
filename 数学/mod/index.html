    жЁЎиҝҗз®—¶

    жЁЎиҝҗз®—пјҢеҸҲз§° еҸ–жЁЎ жҲ– жЁЎйҷӨпјҢжҳҜж•°еӯҰдёӯдёҖз§ҚеҹәзЎҖзҡ„иҝҗз®—ж–№ејҸгҖӮжЁЎиҝҗз®—зҡ„з¬ҰеҸ·йҖҡеёёдёә $\bmod$пјҢеңЁзј–зЁӢиҜӯиЁҖдёӯеёёз”Ё % иЎЁзӨәгҖӮ

    з»ҷе®ҡдёӨдёӘж•ҙж•° a е’ҢжӯЈж•ҙж•° bпјҢжЁЎиҝҗз®— $a \bmod b$ зҡ„з»“жһңжҳҜ a йҷӨд»Ҙ b жүҖеҫ—зҡ„жңҖе°ҸйқһиҙҹдҪҷж•°пјҢе…¶еҖјж»Ўи¶ід»ҘдёӢе…ізі»пјҡ

    \[ a \equiv r \pmod{b} \]

    е…¶дёӯпјҡ

    • $\equiv$ жҳҜеҗҢдҪҷз¬ҰеҸ·пјҢиЎЁзӨә $a$ дёҺ $r$ еңЁжЁЎ $b$ зҡ„ж„Ҹд№үдёӢзӣёзӯү
    • еӯҳеңЁж•ҙж•° $k$ дҪҝеҫ— $a = k \cdot b + r$
    • дё” $0 \leq r < b$

    дҫӢеҰӮпјҡ

    • $7 \equiv 1 \pmod{3}$пјҢеӣ дёә $7 = 2 \cdot 3 + 1$
    • $10 \equiv 2 \pmod{4}$пјҢеӣ дёә $10 = 2 \cdot 4 + 2$

    C++ дёӯзҡ„ % иҝҗз®—з¬Ұ¶

    еңЁ C++ дёӯпјҢжЁЎиҝҗз®—дҪҝз”Ё % иҝҗз®—з¬Ұе®һзҺ°гҖӮдҪҶйңҖиҰҒжіЁж„ҸпјҢC++ зҡ„ % иҝҗз®—з¬Ұи®Ўз®—зҡ„жҳҜдҪҷж•°пјҲremainderпјүпјҢиҖҢдёҚжҳҜж•°еӯҰдёӯдёҘж је®ҡд№үзҡ„жЁЎпјҲmodulusпјүгҖӮиҝҷеңЁеӨ„зҗҶиҙҹж•°ж—¶е°Өдёәе…ій”®гҖӮ

    C++ дёӯзҡ„ / иҝҗз®—з¬Ұ¶

    C++ дёӯпјҢж•ҙж•°йҷӨжі• / зҡ„иЎҢдёәжҳҜеҗ‘йӣ¶жҲӘж–ӯпјҢиҖҢйқһж•°еӯҰдёӯзҡ„еҗ‘дёӢеҸ–ж•ҙгҖӮ

    • еҗ‘йӣ¶жҲӘж–ӯпјҡз»“жһңйқ иҝ‘ $0$
    • ж•ҙж•°йҷӨжі•зӨәдҫӢпјҡ
      • 5 / 2 = 2 пјҲеҺҹдёә $2.5$пјҢеҗ‘ $0$ жҲӘж–ӯеҗҺдёә $2$пјү
      • -5 / 2 = -2пјҲеҺҹдёә $-2.5$пјҢеҗ‘ $0$ жҲӘж–ӯеҗҺдёә $-2$пјү

    C++ дёҺж•°еӯҰзҡ„ % иҝҗз®—еҜ№жҜ”¶

    C++ зҡ„ % иҝҗз®—е®ҡд№үдёәпјҡa % b = a - (a / b) * b

    жіЁж„ҸпјҡиҝҷйҮҢзҡ„ / жҳҜеҗ‘йӣ¶жҲӘж–ӯгҖӮ

    иҖҢж•°еӯҰдёҠзҡ„жЁЎе®ҡд№үдёәпјҡ

    \[ a \bmod b = a - \left\lfloor \dfrac{a}{b} \right\rfloor \cdot b \]

    е…¶дёӯ $\lfloor \cdot \rfloor$ иЎЁзӨәеҗ‘дёӢеҸ–ж•ҙгҖӮ

    C++ дёӯдҝ®жӯЈжЁЎзҡ„еҶҷжі•¶

    дёәдәҶдҪҝжЁЎиҝҗз®—з»“жһңе§Ӣз»Ҳдёәйқһиҙҹж•°пјҲдёҺж•°еӯҰдёҖиҮҙпјүпјҢеҸҜдҪҝз”ЁеҰӮдёӢж–№ејҸдҝ®жӯЈпјҡ

    int mod(int a, int b) // и®Ўз®— a % b
{
    return (a % b + b) % b; // b еҝ…йЎ»дёәжӯЈ
}

    д№ҹеҸҜд»ҘеҶҷдҪң

    int mod(int a, int b) // и®Ўз®— a % b
{
    int v = a % b;
    if (v < 0) v += b;
    return v;
}

    жӢ“еұ•¶

    дёәд»Җд№Ҳ C++ иҰҒе°Ҷж•ҙж•°йҷӨжі•е®ҡд№үдёәеҗ‘йӣ¶жҲӘж–ӯпјҹ¶

    C++пјҲе’Ң Cпјүдёӯзҡ„ж•ҙж•°йҷӨжі•дҪҝз”ЁвҖңеҗ‘йӣ¶жҲӘж–ӯвҖқпјҲtruncation toward zeroпјүпјҢиҖҢдёҚжҳҜж•°еӯҰдёӯеёёи§Ғзҡ„вҖңеҗ‘дёӢеҸ–ж•ҙвҖқпјҲfloor divisionпјүгҖӮиҝҷжҳҜеҮәдәҺеӨҡж–№йқўзҡ„иҖғиҷ‘пјҡ

    1. еҺҶеҸІе…је®№жҖ§пјҲCиҜӯиЁҖзҡ„йҒ—дә§пјү¶

    C++ жҳҜеңЁ C иҜӯиЁҖеҹәзЎҖдёҠеҸ‘еұ•иҖҢжқҘзҡ„пјҢиҖҢ C жңҖеҲқзҡ„и®ҫи®Ўзӣ®ж Үд№ӢдёҖжҳҜйқўеҗ‘еә•еұӮгҖҒй«ҳж•ҲгҖҒеҸҜз§»жӨҚгҖӮC жңҖеҲқзҡ„йҷӨжі•иЎҢдёәе№¶жңӘе®Ңе…Ёи§„е®ҡз¬ҰеҸ·еӨ„зҗҶж–№ејҸпјҢдҪҶи®ёеӨҡзј–иҜ‘еҷЁе®һзҺ°пјҲе°Өе…¶жҳҜеҹәдәҺ x86 жһ¶жһ„зҡ„пјүйҮҮз”ЁдәҶеҗ‘йӣ¶жҲӘж–ӯзҡ„ж–№ејҸпјҢеҗҺжқҘиҝҷе°ұеҸҳжҲҗдәҶдәӢе®һж ҮеҮҶпјҢC99 жҳҺзЎ®и§„е®ҡдәҶжӯӨиЎҢдёәгҖӮ

    C++ дёәдәҶе…је®№ C е’Ңе·ІжңүеӨ§йҮҸд»Јз ҒпјҢз»§з»ӯйҮҮз”ЁдәҶзӣёеҗҢзҡ„иЎҢдёәгҖӮ

    2. еҗ‘йӣ¶жҲӘж–ӯжӣҙжҳ“дәҺе®һзҺ°пјҲеә•еұӮзЎ¬д»¶еҸӢеҘҪпјү¶

    еӨ§еӨҡж•°зҺ°д»ЈеӨ„зҗҶеҷЁпјҲдҫӢеҰӮ x86/x64пјүзҡ„ж•ҙж•°йҷӨжі•жҢҮд»ӨпјҲеҰӮ IDIVпјүиҝ”еӣһзҡ„з»“жһңе°ұжҳҜпјҡ

    • е•Ҷпјҡеҗ‘йӣ¶жҲӘж–ӯзҡ„з»“жһң
    • дҪҷж•°пјҡж»Ўи¶і a = (a / b) * b + (a % b)

    иҝҷж„Ҹе‘ізқҖзј–иҜ‘еҷЁдёҚйңҖиҰҒйўқеӨ–жҢҮд»ӨжқҘеӨ„зҗҶз¬ҰеҸ·гҖҒи°ғж•ҙе•ҶеҖјгҖҒйҮҚз®—дҪҷж•°пјҢж•ҲзҺҮжӣҙй«ҳгҖӮ

    дҫӢеҰӮпјҡ

    int q = a / b;
int r = a % b;
// е§Ӣз»Ҳжңүпјҡa == b * q + r

    3. дҪҷж•°е®ҡд№үеҝ…йЎ»е’ҢйҷӨжі•дҝқжҢҒдёҖиҮҙ¶

    еңЁ C++ дёӯпјҢa % b зҡ„е®ҡд№үеҝ…йЎ»е’Ң a / b й…ҚеҗҲпјҢж»Ўи¶ідёӢеҲ—жҒ’зӯүејҸпјҡ

    a == (a / b) * b + (a % b)

    иҝҷжҳҜ C/C++ иҜӯиЁҖи§„иҢғдёӯеҜ№ % зҡ„ж №жң¬е®ҡд№үгҖӮеҰӮжһңе°Ҷ / ж”№дёәеҗ‘дёӢеҸ–ж•ҙпјҲfloorпјүпјҢе°ұеҝ…йЎ»еҗҢж—¶и°ғж•ҙ % зҡ„и®Ўз®—йҖ»иҫ‘гҖӮ

    еҗҰеҲҷдёҠиҝ°жҒ’зӯүејҸе°ҶдёҚеҶҚжҲҗз«ӢпјҢеҸҜиғҪеҜјиҮҙзЁӢеәҸйҖ»иҫ‘еҮәй”ҷгҖӮ

    4. еҗ‘йӣ¶жҲӘж–ӯе…·еӨҮвҖңеҜ№з§°жҖ§вҖқ¶

    еҗ‘йӣ¶жҲӘж–ӯеңЁжӯЈиҙҹеҜ№з§°еӨ„зҗҶдёҠжӣҙзӣҙи§ӮгҖӮдҫӢеҰӮпјҡ

    • 5 / 2 = 2 => 5 % 2 = 1
    • -5 / 2 = -2 => -5 % 2 = -1

    иҝҷж„Ҹе‘ізқҖеҪ“ж“ҚдҪңж•°еҸҳеҸ·ж—¶пјҢз»“жһңд№ҹдјҡеҜ№з§°еҸҳеҸ·пјҲйҷӨжі•е’ҢеҸ–дҪҷдёҖиө·еҸҳеҢ–пјүпјҢдҫҝдәҺеӨ„зҗҶй•ңеғҸгҖҒеқҗж ҮеҸҚеҗ‘зӯүеңәжҷҜгҖӮ

    дёәд»Җд№ҲдёҚжҢүж•°еӯҰжқҘи®ҫи®Ўпјҹ¶

    ж•°еӯҰдёӯзҡ„жЁЎиҝҗз®—йҖҡеёёиҰҒжұӮз»“жһңйқһиҙҹпјҲ0 <= r < bпјүпјҢиҝҷеңЁж•°и®әгҖҒеҠ жі•зҫӨгҖҒеҗҢдҪҷзӯүеңәжҷҜдёӯжӣҙжңүж„Ҹд№үгҖӮдҪҶеңЁе®һйҷ…зј–зЁӢдёӯпјҢжЁЎиҝҗз®—еёёз”ЁдәҺпјҡ

    • ж•°з»„зҙўеј•зҺҜз»•пјҲеҰӮ i % nпјү
    • еҲӨж–ӯеҘҮеҒ¶жҖ§пјҲx % 2 == 1пјү
    • зҺҜзҠ¶и®Ўж•°еҷЁгҖҒдҪҚз§»еҒҸз§»
    • еҲҶеёғејҸе“ҲеёҢпјҲеҰӮдёҖиҮҙжҖ§е“ҲеёҢдёӯзҡ„ж§ҪдҪҚжҳ е°„пјү

    иҝҷдәӣеә”з”ЁжӣҙжіЁйҮҚж•ҲзҺҮе’ҢдёҖиҮҙжҖ§пјҢиҖҢйқһз¬ҰеҸ·дёҘж јжҖ§гҖӮ

    C++ жүҖйҮҮз”Ёзҡ„еҗ‘йӣ¶жҲӘж–ӯзӯ–з•ҘдёҺеӨ„зҗҶеҷЁжң¬иә«жҢҮд»ӨдёҖиҮҙпјҢйҒҝе…ҚдәҶз¬ҰеҸ·еҲӨж–ӯе’Ңдҝ®жӯЈи®Ўз®—пјҢд»ҺиҖҢжҸҗеҚҮжҖ§иғҪпјҢд№ҹдҫҝдәҺз§»жӨҚе’Ңе…је®№ж—©жңҹд»Јз ҒгҖӮ
